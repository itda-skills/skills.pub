# pixel-art 사용 가이드

이미지를 **도트(픽셀) 그림**으로 바꿔주는 스킬입니다. 캐릭터·로고·사진을 레트로 픽셀 아트로 만들고, 배경을 투명하게 해서 발표자료·문서에 스티커처럼 넣을 수 있습니다.

## 이렇게 말하면 됩니다

- "이 이미지 픽셀아트로 만들어줘"
- "로고를 도트 그림으로 바꿔줘"
- "8비트 스타일로, 배경은 투명하게"
- "이 캐릭터를 픽셀로 만들어서 슬라이드에 넣어줘"

## 무엇을 준비하나요

이미지 소스는 세 가지이고, 어떤 경우든 **후보를 먼저 보여드리고 확인받은 뒤** 픽셀화합니다.

1. **검색(기본)** — "○○ 이미지 찾아서 픽셀아트로"처럼 말하면 **라이선스-프리 이미지**(Creative Commons·퍼블릭도메인)를 Openverse에서 찾아 후보 몇 장을 보여드립니다. 그중 하나를 고르면 됩니다.
   - 저작자 표시가 필요 없는 것만 원하면 "표시 필요 없는 걸로(cc0)"라고 하세요.
2. **새로 생성** — 마땅한 게 없으면 "새로 그려줘"로 이미지를 만들어(imagegen) 픽셀화합니다. (특정 게임/애니 캐릭터는 저작권상 "○○ 풍 오리지널"로 나옵니다.)
3. **가진 파일** — 이미지 경로를 주면 그대로 씁니다.

## 결과 조절 (원하는 느낌대로)

| 원하는 느낌 | 이렇게 말하세요 |
|---|---|
| 굵고 각진 레트로 도트 | "더 굵게 / 색 적게" (격자 40, 색 12) |
| 형태가 살아있는 디테일 | "디테일 살려서" (격자 64, 색 20) |
| 문서·슬라이드에 얹을 스티커 | "배경 투명하게" |

## 시나리오 예

1. **가진 사진을 도트로** — "downloads/cat.jpg 를 픽셀아트로 만들어줘" → `cat-pixel.png` 생성.
2. **투명 스티커로 슬라이드에** — "이 로고 픽셀아트로, 배경 투명하게 해서 슬라이드에 넣어줘" → 투명 PNG 생성 후 발표자료에 삽입.
3. **새 캐릭터부터** — "귀여운 공룡 마스코트 만들어서 도트로 바꿔줘" → 이미지 생성 → 픽셀화.

## 알아두면 좋은 점

- **같은 입력이면 항상 같은 결과**입니다(결정론). 마음에 안 들면 격자 크기·색 수만 바꿔 다시 요청하세요.
- 출력은 **PNG**로 나옵니다(투명 지원).
- 지금은 이미지(PNG)와 문서 삽입까지 지원합니다. "엑셀 셀 하나하나를 색칠하는" 네이티브 셀 모자이크는 후속으로 준비 중입니다.
