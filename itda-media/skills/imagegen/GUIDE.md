# imagegen 사용 가이드

발표자료·블로그·문서에 넣을 이미지를 만듭니다. **Claude에게 말로 부탁하면 됩니다** — "블로그 대표 이미지 만들어줘", "쇼츠용 세로 삽화 만들어줘", "이 주제로 포스터 만들어줘"처럼요.

케이스별로 **실측 검증된 프롬프트 템플릿**을 갖춰 두어, 같은 요청이라도 품질 하한이 보장됩니다.

> ⚠️ 동작하려면 hyve의 이미지 생성 기능(`image.generate`)이 연결돼 있고, **codex가 ChatGPT 계정으로 로그인**돼 있어야 합니다(별도 API 키는 필요 없습니다). 로그인이 안 돼 있으면 Claude가 "먼저 codex 로그인이 필요하다"고 안내합니다.

---

## 무엇을 할 수 있나요?

| 케이스 | 이렇게 말하세요 |
|------|-----------------|
| 블로그·문서 히어로/삽화 | "이 글에 어울리는 대표 이미지 만들어줘" |
| 발표·슬라이드 비주얼 | "발표자료에 넣을 배경 이미지 만들어줘" |
| 아이콘·로고·UI 시안 | "이런 의미의 앱 아이콘 시안 몇 개 만들어줘" |
| 캐릭터·일러스트 | "마스코트 캐릭터 만들어줘" |
| 쇼츠/유튜브 영상 삽화 | "쇼츠용 세로 장면 이미지 만들어줘" |
| 상품화 피규어 | "이 캐릭터를 피규어 사진으로 만들어줘" |
| 실사 인물 | "실사 느낌 프로필 사진 만들어줘" |
| 제품 카탈로그 | "이 제품 화이트 배경 상품샷 만들어줘" |
| 포스터 | "이 주제로 포스터 만들어줘" |

여러 장이 필요하면 "5장 만들어줘"처럼 장수를 말하면 됩니다(한 장씩 차례로 만듭니다).

## 자주 쓰는 시나리오

### 블로그 대표 이미지 만들기

> "AI 에이전트가 사무 업무를 자동화하는 내용의 블로그 글 대표 이미지 만들어줘"

Claude가 실사 사진풍으로 만들고, 제목을 얹을 여백도 한쪽에 비워 둡니다. 일러스트풍을 원하면 "일러스트로 만들어줘"라고 하면 됩니다.

### 쇼츠용 세로 이미지 만들기

> "주식이 오르는 걸 보고 기뻐하는 장면, 쇼츠용 세로로 만들어줘"

9:16 세로 비율로 만들고, 위·아래는 자막 얹을 공간을 비워 둡니다.

### 포스터 만들기 (스타일은 알아서 추천)

> "바다 플라스틱 줄이자는 캠페인 포스터 만들어줘"

스타일을 말하지 않아도 Claude가 내용을 보고 어울리는 스타일을 골라 제안합니다. 제목 글자도 이미지에 직접 넣을 수 있습니다(영문·한글 모두). 정확한 긴 문구나 특정 폰트가 필요하면 글자 자리를 비워 두고 따로 얹는 방식을 권합니다.

### 제품 사진 만들기

> "매트 블랙 텀블러를 화이트 배경 상품샷으로 만들어줘"

쇼핑몰 상세페이지에 바로 쓸 만한 팩샷을 만듭니다.

### 여러 장 만들기

> "이 캐릭터로 표정만 다른 이미지 5장 만들어줘"

요청한 만큼 한 장씩 차례로 만듭니다.

## 안 될 때

- **"codex 로그인이 필요합니다" 안내가 나옴** → Claude에게 codex 로그인을 마쳤다고 알리고 다시 요청하세요(ChatGPT 계정 로그인 한 번이면 됩니다).
- **"codex가 설치돼 있지 않습니다" 안내가 나옴** → codex를 설치한 뒤 다시 요청하세요. Claude가 설치 위치를 안내합니다.
- **이미지에 글자가 깨져 나옴** → 긴 문구·정확한 브랜드명은 글자 자리를 비우고 따로 얹는 방식을 쓰세요("글자 없이 만들고 자리만 비워줘").
- **정사각형인데 크기가 1024가 아님** → 비율은 맞으나 픽셀이 다를 수 있습니다. 정확한 크기가 필요하면 리사이즈를 요청하세요.
- **품질이 아쉬움** → 어느 부분이 아쉬운지 말해 주면 그 피드백을 반영해 다시 만듭니다.

## 알아두세요

- 생성 품질은 케이스 카드(`cases/`)에 **실측 기록**으로 쌓여 있어, 같은 케이스를 반복할수록 품질이 안정적입니다.
- 콘텐츠용 이미지(블로그·슬라이드·영상)는 **실사 사진풍이 기본**입니다. 일러스트·3D 렌더는 요청할 때만 씁니다.
- 정밀한 마스킹 편집·초고해상도(2K+)·멀티 레퍼런스 합성은 이 스킬의 범위가 아닙니다.
- 한 장 만드는 데 보통 1~2분 걸립니다(생성 모델 응답 시간).
