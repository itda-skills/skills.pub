## 빠른 시작

HWP/HWPX 문서를 읽거나 Markdown으로 변환할 때 가장 자주 쓰는 한 줄 요청입니다.

```
이 HWP 파일 읽어줘
```

```
한글 문서를 마크다운으로 변환해줘
```

```
서식 그대로 HTML로 변환해줘
```

스킬이 자동으로 `hwpx` 바이너리를 찾아 변환하고, 이미지가 있으면 캡션까지 달아 돌려줍니다.

## 활용 시나리오

### LLM이 바로 읽을 수 있는 Markdown 만들기

공공기관·기업 HWP 문서를 LLM이 이해하기 쉬운 Markdown으로 정리합니다. 레이아웃용으로 쓰인 표는 자동으로 불릿·들여쓰기 리스트로 평탄화됩니다.

```
이 보고서를 마크다운으로 변환해줘
```

### 서식을 보존한 HTML로 전달

표·정렬·색상 등 원본 서식을 유지하면서 브라우저에서 바로 열 수 있는 단일 HTML 파일이 필요할 때 사용합니다. 이미지는 Base64로 임베디드되어 파일 하나로 완결됩니다.

```
HWP를 HTML로 변환해줘. 서식 그대로
```

### HWP5 변환 실패 시 HTML 우회

HWP5(.hwp) 포맷은 heading 자동 감지·하이퍼링크 추출이 지원되지 않습니다. MD 변환 품질이 떨어지면 HTML로 변환한 뒤 Claude가 다시 Markdown으로 재가공하는 경로를 쓸 수 있습니다.

```
이 HWP가 MD로 이상하게 변환됐어. HTML로 우회해줘
```

## 출력 옵션

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| `--format md` | Markdown 변환 + Table 평탄화 + 이미지 캡션 | LLM이 본문을 이해해야 할 때 (기본) |
| `--format html` | 서식 보존 HTML + Base64 이미지 임베디드 | 브라우저 조회·보고서 전달용 |
| `-o <경로>` | 출력 파일 경로 지정 | 원본 HWP 옆에 결과물을 두고 싶을 때 권장 |

출력 위치는 `-o` 플래그로 원본 문서와 같은 폴더에 두는 것을 권장합니다. 예: 입력이 `docs/report.hwp`면 `-o docs/report.md`로 지정하면 결과물을 찾기 쉽습니다. 원본 폴더가 read-only인 경우에는 자동으로 `.itda-skills/`로 저장됩니다.

## 팁

- **이미지 캡션 건너뛰기**: 요청에 `"이미지 설명 없이"`, `"캡션 없이"`, `"이미지 그대로"`를 포함하면 Sonnet 서브에이전트 캡션 생성을 스킵합니다. 이미지가 많은 문서에서 응답 속도·토큰 소비를 줄일 수 있습니다.
- **표 형식 유지**: 비교표·교차분석표처럼 행-열 교차가 핵심 정보인 경우 `"원본 그대로"`, `"테이블 유지"`, `"표 형식으로 보여줘"`를 덧붙이면 Table 평탄화를 건너뛰고 Markdown 테이블을 그대로 유지합니다.
- **HTML fallback**: HWP5 변환 결과가 `??`로 깨지거나 제목이 유실되면 `--format html`로 변환 후 내용을 재가공하세요. HWPX(.hwpx) 포맷이 가능하다면 그쪽이 훨씬 안정적입니다.
- **버전 확인**: 변환 결과가 의심스러우면 `hwpx version`으로 최소 요구 버전(v0.9.7 이상)을 먼저 확인하세요.

## 제한사항

| 항목 | 내용 |
|------|------|
| 쓰기 미지원 | Markdown → HWPX, HWP/HWPX 수정은 지원하지 않습니다. 읽기·변환 단방향 전용입니다. |
| HWP5 포맷 제약 | Heading 자동 감지·하이퍼링크 추출·Wingdings 특수문자가 지원되지 않습니다. 가능하면 HWPX(.hwpx)를 사용하세요. |
| 수식·차트 OLE 객체 | 수식(수학식)·차트 등 OLE 기반 객체는 추출되지 않습니다. 학술 문서·계산 리포트는 내용 일부가 유실될 수 있습니다. |
| 기타 포맷 미지원 | PDF, DOCX, XLSX 등 HWP/HWPX가 아닌 포맷은 처리하지 않습니다. 다른 스킬(pdf-context-refinery 등)을 사용하세요. |
| 암호 보호 파일 | 비밀번호로 보호된 HWP/HWPX는 변환할 수 없습니다. 먼저 암호를 해제한 뒤 요청해야 합니다. |
| 바이너리 요구 | `hwpx` CLI v0.9.7 이상이 필요합니다. Linux(Cowork)는 자동 추출되지만 macOS/Windows는 PATH에 설치되어 있어야 합니다. |
