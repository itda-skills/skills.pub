# hwp-report 사용 가이드

작성하려는 보고서 내용을 말하면, **대한민국 정부 범용 한글 서식(.hwpx)** 으로 만들어 줍니다.
**Claude에게 말로 부탁하면 됩니다** — "올해 청년정책 추진현황을 정부 보고서로 만들어줘"처럼요. 터미널을 직접 다룰 필요는 없습니다.

내용은 Claude가 작성하고, 이 스킬이 제목 박스·머리글·□/❍ 개조식 본문·표가 들어간 한글 문서로 바꿔 줍니다. 다 쓴 한글 문서(읽기·요약)를 마크다운으로 바꾸는 일은 반대 방향인 `hwpx` 스킬을 쓰세요.

---

## 빠른 시작

```
이 내용으로 정부 한글 보고서 만들어줘
분기 실적을 개조식 보고서(.hwpx)로 정리해줘
아래 표까지 넣어서 한글 보고서로 뽑아줘
```

내용만 주면 Claude가 개조식(□·❍)으로 구조를 잡고, 제목·보고일·부서가 든 한글 문서를 만들어 경로를 알려줍니다.

---

## 무엇을 만들 수 있나 — 이렇게 말해 보세요

내용은 Claude가 채웁니다. 주제·핵심 사실만 주면 됩니다.

| 만들고 싶은 문서 | 이렇게 말하세요 |
|---|---|
| 정책·사업 추진 현황 | "청년 주거지원 정책 추진현황을 정부 보고서로 만들어줘" |
| 결과·실적 보고 | "상반기 도시재생 사업 결과를 보고서로 정리해줘" |
| 통계·수치 보고(표) | "민원 처리 통계를 표로 넣어서 한글 보고서로 만들어줘" |
| 예산 집행 점검 | "2분기 예산 집행 점검 결과를 개조식으로 작성해줘" |
| 회의 결과 정리 | "오늘 추진단 회의 결과를 정부 서식 보고서로 만들어줘" |
| 연구·검토 보고 | "친환경 소재 연구 중간보고를 한글 문서로 뽑아줘" |
| 점검·감사 결과 | "노후시설 안전점검 결과를 보고서로 정리해줘" |
| 행사·교육 계획 | "신규 임용자 교육 계획을 정부 보고서로 만들어줘" |

여러 건을 한 번에 부탁해도 됩니다 — "이 세 가지 사업을 각각 보고서로 만들어줘".

---

## 표 넣기

수치·현황은 표로 부탁하면 헤더와 데이터가 그대로 칸에 맞춰 들어갑니다.

```
부서별 인력 현황을 표로 정리해서 보고서에 넣어줘
권역별 방문객 수를 표로 만들어서 같이 넣어줘
```

- 표는 **헤더(맨 윗줄) + 데이터 행**으로 구성되고, 칸마다 값이 그대로 채워집니다.
- 열 개수·행 개수는 자유입니다(2열이든 6열이든). 숫자·기호(`·`, `「」`, `%`, 소수점)도 그대로 들어갑니다.
- 표는 해당 항목(섹션) 안에 함께 배치됩니다.

> ⚠️ 지금은 **데이터가 칸에 정확히 들어가는 것**까지 지원합니다. 칸별 정렬(왼쪽/오른쪽), 셀 안 굵게·기울임, 셀 병합은 아직 반영되지 않습니다(향후 개선). 숫자 오른쪽 정렬 같은 세부 서식이 꼭 필요하면 미리 말해 주세요 — 현재 한계를 안내해 드립니다.

---

## 보고일·부서 넣기 (머리글)

머리글 오른쪽 위에 "'26. 6. 9., ○○과)"처럼 보고일과 부서가 들어갑니다.

```
보고일은 6월 9일, 부서는 재정기획담당관으로 해줘
```

- **함께 말해 주는 것을 권장**합니다. 보고일을 안 주면 만든 날짜로 자동으로 채우고, 부서를 안 주면 머리글에 "부서명"이라는 빈 자리표시가 남습니다 — 이때는 안내해 드립니다.

---

## 잘 나오게 하는 팁

- **개조식이 기본**입니다 — 한 항목당 한 줄(□), 그 아래 세부는 한 단계 들여쓰기(❍)로 정리됩니다. "개조식으로", "핵심만 짧게"라고 하면 더 깔끔합니다.
- **번호 항목(1. 2. 3.)** 은 보고서의 큰 절 제목으로, 그 안의 세부는 □·❍로 들어갑니다.
- **소제목이 너무 깊으면**(3단계 이상) 한 단계로 정리되고, 그 사실을 알려줍니다.
- 길게 쓴 문단도 자동으로 □ 항목으로 정리됩니다.

---

## 결과 받기

- 만들어진 한글 문서(`.hwpx`)의 **저장 위치를 알려드립니다.** 더블클릭하면 한글에서 열립니다.
- 빠르게 모양만 보고 싶으면 "PDF로도 같이 보여줘"라고 하면 미리보기를 함께 만들어 줍니다.
- 다시 고치고 싶으면 "두 번째 항목 표현 바꿔서 다시 만들어줘"처럼 말하면 됩니다.
