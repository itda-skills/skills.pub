# 회의록

## 개요

- **스타일명**: 회의록 (공식 회의록, 간담회 기록, 협의 결과)
- **톤**: 객관적·사실 기록체. 발언 내용을 정확히 기록하되 요약 정리.
- **대상 독자**: 회의 참석자, 미참석 관계자, 후속 조치 담당자
- **적정 분량**: A4 1-3매

## 인터뷰 질문

- 회의 유형은? (정례회의/임시회의/간담회/워크숍)
- 회의 주제/안건은 무엇인가요?
- 참석자와 소속은?
- 주요 논의 내용과 결정 사항은?
- 후속 조치(Action Item)가 있나요?
- 다음 회의 일정이 정해졌나요?

## 필수 구성 요소

1. **회의 정보**: 회의명, 일시, 장소, 참석자.
2. **안건**: 논의할 안건 목록 (번호 매김).
3. **논의 내용**: 안건별 주요 발언 요약. 발언자 표시.
4. **결정 사항**: 합의된 결론 명확히 기록.
5. **Action Items**: 후속 조치, 담당자, 기한을 표로 정리.
6. **다음 회의**: 일시, 장소, 예정 안건.

## 문체 규칙

- **어미**: "~하였음", "~하기로 함", "~예정임" (기록체)
- **발언 기록**: "○○○ 과장: ~라고 언급" 또는 "○○○ 위원이 ~를 제안"
- **문장 길이**: 한 문장 50자 이내
- **객관성**: 기록자의 주관 배제. 사실만 기록.
- **약칭**: 첫 등장 시 정식명칭, 이후 약칭 가능.

### 발언 기록 원칙

- 발언 요지를 간결하게 정리 (축어록이 아님)
- 핵심 키워드와 수치를 정확히 기록
- 질의-응답은 짝으로 기록
- 민감한 발언은 "비공개" 표시 가능

### 금지 표현

| 금지 | 대체 |
|------|------|
| 열띤 토론이 있었음 | 구체적 논의 내용 기록 |
| 대체로 동의하였음 | 결정 사항을 명시 |
| 좋은 의견이 많았음 | 주요 의견을 항목별로 기록 |

## Frontmatter 템플릿

```yaml
---
title: "{회의명}"
description: "{1문장 회의 요약}"
created_at: {YYYY-MM-DD}
type: {정례회의|임시회의|간담회|워크숍|협의}
datetime: "{YYYY-MM-DD HH:mm~HH:mm}"
location: "{장소 또는 화상회의 URL}"
chair: "{주재자 직급·성명}"
recorder: "{기록자 직급·성명}"
attendees:
  - "{참석자1 소속·직급·성명}"
  - "{참석자2 소속·직급·성명}"
---
```

## 예시 패턴

### 회의 정보 예시

```markdown
# 제3차 AI 서비스 품질인증 TF 회의

| 항목 | 내용 |
|------|------|
| 일시 | 2026. 3. 28.(목) 14:00~15:30 |
| 장소 | 세종청사 6동 3층 대회의실 |
| 주재 | ○○○ 정책관 |
| 참석 | (별첨 참석자 명단 참조, 총 12명) |
| 기록 | ○○과 ○○○ 사무관 |
```

### 논의 내용 예시

```markdown
## 안건 1. AI 인증제 시행 일정 조정

### 주요 논의

○ ○○○ 과장: 52개 서비스 중 23개가 미준비 상태로, 4.1. 시행은 현실적으로 어려움
○ ○○○ 사무관: 보안 점검 외주 업체 일정이 4월 중순까지 확보됨
○ ○○○ 위원: 유예기간을 두는 방안을 검토해 볼 필요가 있음

### 결정 사항

○ 4. 1. 시행하되, 6개월 유예기간을 두기로 함
○ 미준비 기관에 대한 지원 방안을 ○○과에서 마련하기로 함
```

### Action Items 예시

```markdown
## Action Items

| 번호 | 조치 사항 | 담당 | 기한 |
|------|----------|------|------|
| 1 | 유예기간 운영 방안 마련 | ○○과 ○○○ | 4. 5.(토) |
| 2 | 미준비 기관 현황 업데이트 | ○○과 ○○○ | 4. 3.(목) |
| 3 | 국회 보고 자료 수정 | ○○과 ○○○ | 4. 4.(금) |

## 다음 회의

○ 일시: 2026. 4. 11.(금) 14:00
○ 장소: 세종청사 6동 3층 대회의실
○ 안건: 유예기간 운영 방안 확정, 지원 대책 보고
```
