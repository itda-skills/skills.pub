# 사내 공지

## 개요

- **스타일명**: 사내 공지 (공지사항, 안내문)
- **톤**: 공손하고 명확한 경어체. 격식 있되 딱딱하지 않게.
- **대상 독자**: 임직원 전체 또는 특정 부서
- **적정 분량**: A4 0.5-1매 (핵심만 간결하게)

## 인터뷰 질문

- 공지 유형은? (인사발령/제도변경/행사안내/시설공지/IT공지)
- 대상은 전 직원인가요, 특정 부서인가요?
- 시행일/적용일은 언제인가요?
- 문의처(담당 부서/담당자)는?
- 첨부 자료가 있나요?

## 필수 구성 요소

1. **제목**: "[공지]" 또는 "[안내]" 접두어 + 핵심 내용.
2. **배경/목적**: 왜 이 공지를 하는지 1-2문장.
3. **핵심 내용**: 일시, 장소, 대상, 방법 등을 구조화하여 제시.
4. **협조/유의사항**: 임직원에게 요청하는 행동.
5. **문의처**: 담당 부서, 담당자명, 내선번호/이메일.
6. **첨부**: 관련 양식이나 상세 자료 (선택).

## 문체 규칙

- **어미**: "~합니다", "~바랍니다", "~드립니다"
- **문장 길이**: 한 문장 40자 이내
- **구조**: 핵심 정보를 표나 글머리로 한눈에 파악 가능하게
- **경어**: "~하오니 협조 부탁드립니다", "~참고하시기 바랍니다"

### 금지 표현

| 금지 | 대체 |
|------|------|
| ~해주세요 | ~하여 주시기 바랍니다 |
| 꼭, 반드시 (남용) | 필수 사항을 별도 표기 |
| 장문 배경 설명 | 핵심만 간결히 |

## Frontmatter 템플릿

```yaml
---
title: "{공지 제목}"
description: "{1문장 요약}"
created_at: {YYYY-MM-DD}
type: {인사발령|제도변경|행사안내|시설공지|IT공지|일반공지}
effective_date: {YYYY-MM-DD}
target: "{전 직원|특정 부서명}"
contact:
  department: "{담당 부서}"
  name: "{담당자명}"
  phone: "{내선번호}"
---
```

## 예시 패턴

### 제도 변경 공지 예시

```markdown
# [공지] 2026년 4월 재택근무 제도 변경 안내

안녕하십니까. 인사팀입니다.

2026년 4월 1일부터 재택근무 제도가 아래와 같이 변경되오니 참고하시기 바랍니다.

## 변경 내용

| 구분 | 현행 | 변경 |
|------|------|------|
| 재택 가능일 | 주 2일 | 주 3일 |
| 신청 방법 | 주간 사전 신청 | 월간 사전 신청 |
| 코어타임 | 10:00-16:00 | 10:00-15:00 |

## 유의사항

○ 4월 근무 계획은 3. 25.(화)까지 팀장 승인을 완료하여 주시기 바랍니다.
○ 상세 가이드는 [첨부] 재택근무 운영지침(개정)을 참고하시기 바랍니다.

## 문의

인사팀 근무제도파트 ○○○ (내선 1234, hr@company.co.kr)
```
