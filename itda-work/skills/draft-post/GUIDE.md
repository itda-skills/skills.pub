블로그·보고서·기획서·보도자료·뉴스레터를 인터뷰 기반으로 초안 작성하는 스킬입니다. **Claude에게 주제와 스타일을 말하면** 몇 가지 질문을 거쳐 완성도 높은 마크다운 초안을 만들어 줍니다.

---

## 빠른 시작

```
"Claude Skills로 반복 업무 자동화하기" 블로그 글 써줘
```

스킬이 자동으로 **인터뷰 → 초안 → AI 문체 점검 → 수정 반복 → 저장** 흐름으로 진행합니다.

---

## 사용 가능한 스타일

글쓰기 스타일을 말해주지 않으면 블로그(기본값)로 진행합니다. 다른 스타일이 필요하면 함께 말하세요.

| 스타일 | 설명 |
|--------|------|
| `blog` (기본) | 블로그 포스팅 |
| `official-report` | 공무원 업무보고·현황보고·기획보고 |
| `press-release` | 정부·공공기관 보도자료 |
| `proposal` | 기획서·제안서 (정책 기획, 사업 제안) |
| `briefing` | 간부 보고용 브리핑 자료 (A4 1매) |
| `official-letter` | 공문·시행문·업무연락 |
| `policy-brief` | 정책 브리프 (이슈 분석·제언) |
| `business-plan` | 사업계획서 (IR 덱, 정부지원사업) |
| `announcement` | 사내 공지·안내문 |
| `newsletter` | 이메일 뉴스레터 |
| `meeting-minutes` | 회의록·간담회 기록 |
| `product-brief` | 제품·서비스 소개서 |

사용 가능한 전체 스타일 목록을 보고 싶으면 "스타일 목록 알려줘"라고 말하세요.

---

## 사례 — 블로그 글쓰기

### 사용자 첫 발화

```
"Claude Skills로 반복 업무 자동화하기" 블로그 글 써줘
```

### Round 1 — 핵심 인터뷰

스킬이 2~3개 질문을 던집니다. 첫 선택지가 항상 권장 값입니다.

**Q1. 대상 독자는 누구인가요?**

- (권장) 개발자 — 기술적 배경 있음, 코드 예시 OK
- 일반 직장인 — 비개발자, 도구 활용 관점
- 팀 리더 — 도입 의사결정 관점

**Q2. 독자가 이 글을 읽고 얻어갈 한 가지는?**

- (권장) "내 업무에 바로 적용할 수 있다"는 확신
- 스킬의 기술적 동작 원리 이해
- 타 자동화 도구와의 차이

### Round 2 — 구체화 인터뷰

Round 1 답변을 반영해 범위·톤을 좁힙니다.

**Q3. 본문 분량은?**

- (권장) 1,500~2,500자 (3~5분 분량)
- 800자 내외 (리드 중심)
- 3,000자 이상 (심층)

### AI 문체 자가 점검

초안 직후 스킬이 AI 문체 자가 점검 결과를 노출합니다.

```
🔍 AI 문체 자가 점검
  ✅ AI 습관어 미포함
  ⚠️ 2문단에서 "~할 수 있습니다" 3회 연속 → 어미 변화 권장
  ✅ 번역투 패턴 미포함
  ✅ 문장 길이 변화 충분
```

### 수정 루프

초안 이후 스킬은 다음 옵션을 반복 제시합니다.

- AI 문체 점검에서 발견된 항목 자동 수정
- 특정 섹션 다시 쓰기
- 톤 조절
- 내용 추가/삭제
- 완료 (현재 상태로 확정)

---

## 나만의 스타일 만들기

기존 문서(보고서·메일·PPT·PDF 등)를 주면 그 문체를 분석해 새 스타일을 자동으로 만들어 줍니다.

```
이 보고서 파일 분석해서 우리 팀 스타일 만들어줘
```

만들어진 스타일은 이름을 지정해 나중에도 "우리팀-보고서 스타일로 써줘"처럼 불러쓸 수 있습니다.

---

## 팁

- **인터뷰 단축**: 첫 발화에 대상·메시지·분량을 함께 넣으면 질문이 1~2개로 줄어듭니다.
  예: `개발자 대상, 자동화 확신 주기, 2000자로 블로그 글 써줘`
- **저장 위치 지정**: "초안을 drafts 폴더에 저장해줘"처럼 저장 경로를 말하면 바로 파일로 만들어 줍니다. 말하지 않으면 대화창에 출력 후 저장 여부를 다시 물어봅니다.
- **Cowork 마운트 없이 사용 시**: 작성한 스타일이 세션이 끝나면 사라집니다. 영구 보관하려면 Cowork에 호스트 폴더를 마운트하세요. 스킬이 저장 시 이 상황을 안내합니다.
