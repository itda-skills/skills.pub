> `work/draft-post`가 실제 대화를 어떻게 이끌고, 어떤 결과물을 만드는지 한 가지 사례로 보여드립니다.

## 빠른 시작

Claude Code / Cowork에 이렇게 말해보세요.

```
draft-post로 "Claude Skills로 반복 업무 자동화하기" 블로그 글 써줘
```

스킬이 자동으로 **인터뷰 → 초안 → AI 문체 점검 → 수정 반복 → 저장** 까지 진행합니다.

## 기본 인자

- `<topic>` (필수) — 글 주제
- `--style <style>` (선택) — 기본값 `blog`. 다른 값: `press-release`, `business-plan`, `newsletter`, `meeting-minutes` 등 (총 12종)
- `--save <path>` (선택) — 저장 경로. 생략 시 채팅창에만 출력하고 저장 여부를 다시 질문
- `--list-styles` (선택) — 사용 가능한 스타일만 출력

전체 스타일 목록과 환경별 저장 경로는 스킬 저장소의 `SKILL.md`를 참고하세요.

## 사례 — 블로그 글쓰기

### 사용자 첫 발화

```
draft-post로 "Claude Skills로 반복 업무 자동화하기" 블로그 글 써줘
```

### Round 1 — 핵심 인터뷰

스킬이 AskUserQuestion으로 2~3개 질문을 한 번에 묶어 던집니다. 첫 선택지가 항상 "권장" 값입니다.

**Q1. 대상 독자는 누구인가요?**

- (권장) 개발자 — 기술적 배경 있음, 코드 예시 OK
- 일반 직장인 — 비개발자, 도구 활용 관점
- 팀 리더 — 도입 의사결정 관점

**Q2. 독자가 이 글을 읽고 얻어갈 한 가지는?**

- (권장) "내 업무에 바로 적용할 수 있다"는 확신
- 스킬의 기술적 동작 원리 이해
- 타 자동화 도구와의 차이

### Round 2 — 구체화 인터뷰

Round 1 답변을 반영해 범위·톤을 좁힙니다.

**Q3. 본문 분량은?**

- (권장) 1,500~2,500자 (3~5분 분량)
- 800자 내외 (리드 중심)
- 3,000자 이상 (심층)

**Q4. 코드 예시 포함 여부?**

- (권장) 스킬 호출 예시 2~3개
- 전체 SKILL.md 구조까지 포함
- 코드 없이 개념 위주

### AI 문체 자가 점검

초안 직후 스킬이 `_anti-ai-korean.md` 규칙으로 자가 점검 결과를 먼저 노출합니다.

```
🔍 AI 문체 자가 점검
  ✅ AI 습관어 미포함
  ⚠️ 2문단에서 "~할 수 있습니다" 3회 연속 → 어미 변화 권장
  ✅ 번역투 패턴 미포함
  ✅ 문장 길이 변화 충분
```

### 결과물 (발췌)

```markdown
---
title: "Claude Skills로 반복 업무 자동화하기"
description: "매주 반복되는 주간보고, 이메일 회신을 스킬로 자동화한 3주간의 기록"
tags: [claude, skills, automation, productivity]
---

## 반복 작업이 주 5시간을 잡아먹고 있었다

지난달 내 시간표를 돌아보니 매주 월요일 주간보고 작성에 90분,
금요일 보고서 초안에 2시간, 메일 회신 템플릿 정리에 1시간이 들어갔다.
합치면 주당 4시간 반. 한 달이면 20시간, 연간 240시간이다.

(후략)
```

### 수정 루프

초안 이후 스킬은 다음 옵션을 반복해서 제시합니다.

- AI 문체 점검에서 발견된 항목 자동 수정
- 특정 섹션 다시 쓰기
- 톤 조절
- 내용 추가/삭제
- 완료 (현재 상태로 확정)

사용자가 "완료"를 선택할 때까지 반복합니다.

## 팁

- **인터뷰 단축**: 첫 발화에 대상·메시지·분량을 함께 넣으면 Round 1이 1~2문항으로 줄어듭니다.
  예: `draft-post로 "개발자 대상, 자동화 확신 주기, 2000자" 블로그 글 써줘`
- **스타일 직접 만들기**: `--analyze <기존문서>` 로 pptx/docx/md를 분석해 새 스타일을 자동 생성할 수 있습니다.
- **저장 경로**: `--save ./drafts/2026-04-blog.md` 처럼 지정하면 바로 파일로 떨어집니다.
