# Warm Editorial — 에디토리얼 리포트 문법

## Overview

크림 `#FAF9F5` 가 지배하고 잉크 `#141413`(순흑 대신 따뜻한 흑)이 본문을 맡는다.
코랄 `#CC785C` 가 주 액센트(키워드·빅넘버·차트 주 시리즈), 틸 `#5DB8A6` 과 앰버 `#E8A55A` 류가
보조를 선다. 표지와 결론만 다크 `#181715` 로 뒤집는 샌드위치가 서사의 무게중심을 만든다.
라틴 헤드라인에 세리프(Georgia)를 쓰면 에디토리얼 감각이 살아나지만, 한글은 고딕으로
폴백된다는 한계를 받아들이고 색·여백으로 정체성을 보강한다.

## 슬라이드 문법

- **표지(다크)**: 다크 배경 + 크림 타이틀 + 코랄 킥커 + 부제 한 문장.
- **본문**: 크림 캔버스 + 헤어라인으로 섹션 구획. 2단(텍스트 좌/시각 우) 기본.
- **스탯 콜아웃**: surface 카드 + 코랄 빅넘버 + 작은 라벨.
- **결론(다크)**: 핵심 메시지 한 문장 + 후속 액션.

## 차트 권장

matplotlib 차트는 크림 배경에 맞춰 `transparent=True` + 코랄/틸 시리즈.
네이티브 차트 palette 에 `["CC785C", "5DB8A6", "E8A55A"]` 순으로 지정.
