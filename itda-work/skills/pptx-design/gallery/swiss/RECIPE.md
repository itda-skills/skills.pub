# 덱 #11 — 스위스 모듈러 그리드 (세계 인구·고령화)

> 갤러리 #401 · 페이퍼 화이트 + 잉크 + 단일 스위스 레드 · 6 슬라이드 · 결함 0

**한 줄 콘셉트** — ★레이아웃 아키타입: **스위스(국제 타이포그래피) 모듈러 그리드**. 엄격한 12단 그리드, 번호 섹션(01~), 플러시 좌측, 넉넉한 여백, 가는 룰, 절제된 흑백 + 단일 레드. 객관적·합리적 데이터 미학. 6번째 구성.

## Backend
- **COM** (라이브 PowerPoint via `hyve-office.exe`).

## 이렇게 말하면 이 덱이 나온다 (자연어 요청 프롬프트)
> "세계 인구·고령화 데이터로 **스위스 모듈러 그리드** 6장 — 엄격한 12단 그리드, **번호 섹션(01~)**, 플러시 좌측, 넉넉한 여백, 가는 룰, **흑백 + 단일 레드 액센트**, 객관적 톤. 표지(좌측 대형 타이틀+우측 메타셀)·정점(라인)·저출생(막대)·역전(가로막대)·고령화(모듈 블록)·코다. 차트는 그리드 셀에 정렬, 핵심 데이터만 레드 강조. 2출처 교차+전망 플래그."

## 디자인 토큰
| 역할 | 값 |
|---|---|
| 배경 | `#FCFCFB` 페이퍼 화이트 |
| 잉크 | `#16181D` / 그레이 `#8A8F98` / 라이트 `#C9CDD4` |
| 단일 액센트 | `#E2231A` 스위스 레드 (핵심 데이터만) |
| 룰/그리드 | `#E3E5E9` |
| 폰트 | 맑은 고딕(헤드/본문) · **Consolas**(섹션번호·러닝라벨·수치) |
| 그리드 | 12단 (GW=균등, GAP=14) — 모든 요소를 컬럼 경계에 정렬 |

## ★레이아웃 아키타입: 스위스 모듈러 그리드 (전용 헬퍼)
- `gx(i)`/`gw(n)` — 12단 그리드 좌표/폭. 모든 블록을 컬럼 경계에 스냅.
- `secnum`(라이트그레이 대형 번호 01~), `head`(러닝라벨+가는 룰+페이지), `kicker_title`(그리드 정렬 타이틀), `rule`(가는 선). **rail/dashboard/editorial/timeline/poster 헬퍼 미재사용.**
- 절제: 색은 흑백 기본 + 레드는 "한 슬라이드에 핵심 1~2개"만. 여백이 디자인의 일부.

## 레이아웃 레시피 (6 슬라이드)
1. 표지 — 좌측 대형 타이틀(레드 룰) + 우측 메타 셀(현재/정점/2100).
2. §01 정점 — 라인 차트(2024→2084 정점→2100) + 우측 키 수치 셀.
3. §02 저출생 — 막대(중국 레드 최저/대체수준 그레이 참조) + 설명 셀.
4. §03 역전 — 가로 막대(중위연령, 인도 레드) + 인도/중국 비교 행.
5. §04 고령화 — 3 모듈 블록(8.2억/2.25/2084) + 결론.
6. 코다 — 플러시 좌측 스테이트먼트 + 콜로폰.

## 실증한 PPT 요소
- 12단 그리드 정렬 조판(번호 섹션·가는 룰·모듈 블록).
- 네이티브 line·column·bar + point_colors(핵심 데이터 레드 강조) + 축/데이터라벨(라이트 배경, #404).

## 데이터 (2+ 출처 교차 · 전망 플래그)
- 세계 인구 **8.2억(80억, 2024)** → **정점 10.3억(103억)·2084** → 10.2억(2100, 감소 전환).
- 합계출산율: 중국 **1.01**·인도 1.96·세계 2.25 (대체수준 2.10, 세계 평균 2036년경 도달).
- 중위연령: 인도 28.4·미국 38.3·중국 39.6세. 인도가 2023년 중국 제치고 최다 인구국.
- ⚠️ 정점·2100은 UN 중위 시나리오(전망).
- 출처: UN World Population Prospects 2024·Our World in Data·INED.
