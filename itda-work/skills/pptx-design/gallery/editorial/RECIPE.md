## ★레이아웃 아키타입: 에디토리얼 스프레드 (전용 헬퍼)
- `runhead`(마스트헤드+이슈+룰+폴리오), `section_label`(FEATURE 01…), `headline`(거대 세리프), `dropcap`(코랄 첫글자 + '. ' 경계 분할 본문 — 숫자 중간 분절 방지), `column`(다단 본문, line_spacing), `pull_quote`(룰+이탤릭 세리프), `rule`(가로/세로 선), 듀오톤 `img`. **rail/dashboard 헬퍼 재사용 안 함.**
- 레시피 주의: 드롭캡 본문과 옆 칼럼은 **내용 중복 금지**(별개 텍스트). 드롭캡 분할은 첫 문장(`. `) 경계로 — 숫자(6.5% 등) 분절 방지.

## 레이아웃 레시피 (6 슬라이드)
1. 표지 스프레드 — 마스트헤드+거대 세리프 타이틀+스탠드퍼스트+듀오톤 히어로(우측 풀하이트).
2. 리드피처(게임) — 드롭캡 본문 2단 + 빅스탯 `$8.39B` + 듀오톤 블록 + 풀쿼트.
3. The Numbers — 장르 막대(게임 압도, 코랄 강조 point_colors) + 성장률 사이드바(−6.5%/+73.9%/+159%/+31.3%).
4. 음악 — 풀쿼트(+73.9%) + 2단 본문 + 빅스탯 + 듀오톤 블록.
5. 방송&웹툰 — 2단 분할(방송 +159% 코랄 / 웹툰 +31.3% 골드) + 세로 룰.
6. 코다 — 에세이 단락 + 콜로폰(시장 $79.1B 세계7위·무역흑자 $11.25B·출처·마스트헤드).

## 실증한 PPT 요소

- 네이티브 막대 + point_colors(게임 코랄 강조) + 축/데이터라벨.
