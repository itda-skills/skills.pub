> 갤러리 #401 · 클리니컬 웜(페이퍼 + 메디컬 틸 + 어트리션 코랄) · 6 슬라이드 · 결함 0

**한 줄 콘셉트** — ★레이아웃 아키타입: **스토리보드**. 번호 패널(프레임) + 거터 + 커넥터 화살표(`rightArrow`) + 필름 스트립 스프로킷 모티프로 **시퀀스 서사(콘티/만화 컷)**를 조판한다. 타임라인(#9)의 수직 스파인과 달리 프레임 패널을 격자로 늘어놓고 화살표로 흐름을 잇는다. 12번째 구성, 전용 헬퍼 신작.

## Backend

> "신약 개발 과정을 **스토리보드(만화 컷/콘티)**로 — 단계마다 **프레임 패널 + 번호 배지 + 화살표**로 잇고, 위아래에 **필름 스트립 스프로킷**. 클리니컬 톤(페이퍼 배경, 메디컬 틸 액센트, 탈락은 코랄). 표지(헤드라인 스탯 3) · 여섯 컷 스토리보드 시트(발견→전임상→1/2/3상→승인, 각 컷에 기간·후보수·통과율) · 감소 깔때기 · 비용·시간 · 인사이트 · 클로징. Tufts·BIO·FDA·RAND 2+ 출처 교차, 비용·성공률 편차 플래그."

## ★레이아웃 아키타입: 스토리보드 (전용 헬퍼)
- `panel(...)` — **프레임 패널**(테두리 있는 rectangle). 모든 컷의 기본 프레임.
- `story_panel(...)` — 번호 배지(틸 원) + 단계명 + 기간(틸) + 통과율 칩(코랄, top-right) + 후보수(scene 스탯) + 캡션.
- `badge(...)` — 번호 배지(oval + 흰 숫자). `arrow(...)` — 컷 사이 커넥터(`rightArrow`).
- `sprockets(si, y)` — **필름 스트립**(작은 사각형 행) 위·아래 모티프(스토리보드 아이덴티티).
- 깔때기(funnel): 중앙 정렬 감소 막대(틸→코랄) + ▼ + 탈락률 + "도식(scale 아님)" 명시. **timeline 헬퍼 미재사용.**

## 레이아웃 레시피 (6 슬라이드)
1. **표지** — 스프로킷 + 타이틀 + 헤드라인 스탯 패널 3(~10년·~$2.6B·~10%).
2. **여섯 컷 시트** — 6 프레임 패널(3×2) + 번호 배지 + 통과율 칩 + → 화살표. (★시그니처)
3. **감소의 해부** — 깔때기 감소 막대(5,000–10,000→250→5→1) + 단계별 탈락률.
4. **비용·시간** — 임상 단계별 기간 막대 + 비용($2.6B↔$708M 정의편차).
5. **인사이트** — 프레임 패널 3(번호 배지 + 헤드 + 본문).
6. **클로징** — 다크 틸 패널에 스테이트먼트 + 스프로킷.

## 실증한 PPT 요소
- 프레임 패널(테두리 rectangle) + 번호 배지 + `rightArrow` 커넥터 — 스토리보드 헬퍼 세트 신작.
- 필름 스트립 스프로킷 모티프(작은 사각 행).
- 중앙 정렬 깔때기(감소 막대 + ▼ + 탈락률) — "도식, 실제 비율 아님" 명시.
- 단계별 기간 막대 + 정의편차 비용 대비.

- 평균 **~10년**, **~$2.6B**(Tufts/DiMasi, 실패·자본비용 포함) ⚠️ RAND(2025) 중앙값 **$708M** — 방법론 편차.
