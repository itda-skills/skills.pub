**한 줄 콘셉트** — ★레이아웃 아키타입: **신문 브로드시트**. 1면을 **마스트헤드(네임플레이트·에디션·데이트라인) + 다단 컬럼 + 컬럼 룰(세로 헤어라인) + 헤드라인 deck/kicker + byline**으로 조판하고, 내지는 러닝 헤드 + folio(지면 푸터)로 신문 지면을 흉내 낸다. 매거진 에디토리얼(#8)과 다른 **인쇄 전통**(잡지 스프레드 ≠ 신문 1면). 8번째 구성, 전용 헬퍼 신작.

## ★레이아웃 아키타입: 신문 브로드시트 (전용 헬퍼)
- `masthead(si)` — 에디션(라틴 세리프 캡스) + 네임플레이트(헤비 고딕 중앙) + **더블룰**(2.6px+0.8px) + 데이트라인 행(좌 발행정보·우 기준일) + 얇은 룰. 1면 전용.
- `running_head(si, section)` — 내지 상단: 얇은 룰 + 좌(제호)·우(섹션·면번호).
- `folio(si, page, section)` — 내지 하단: 룰 + 좌(N면·제호)·우(섹션 라틴 캡스).
- `kicker(si, x, y, text)` — 작은 사각 + 라틴 세리프 캡스(자간) 레드 킥커.
- `hr/vr` — 가로/세로 헤어라인(컬럼 룰). **rail/dashboard/editorial/timeline/poster/swiss/split 헬퍼 미재사용.**
- **이미지 라우팅**: split과 동일 — verb 없는 `type=image` 명령은 빌드엔진이 분리해 standalone add(텍스트 아래에 깔림).

## 레이아웃 레시피 (6 슬라이드)
1. **1면 톱** — 마스트헤드 + 2줄 대형 헤드라인 + 3단 그리드(좌: 바이라인+리드 본문 / 중: 리드 아트+캡션+deck / 우: `2.5%` 스탯 박스+플래그 + 안쪽 면 티저 4).
2. **리드(2면)** — 3개년 궤적 빅스탯(2.9/**2.5**/2.8%) + 2단 본문(둔화의 해부 · 같은 세계 다른 숫자).
3. **지역 데스크(3면)** — EMDE 지역별 막대(남아시아 6.3 강조 잉크블루 · 중동 1.6 강조 레드) + 데스크 노트 + 주요국 사이드바.
4. **마켓·에너지(4면)** — 빅스탯 3단(물가 `4.0%` · 유가 `$94` · 하방 `1.3%/4.4%`) + 본문.
5. **데이터(5면)** — 네이티브 표(기관별 2026/2027 세계성장·가중방식) + 해설 + 2027 회복 전망.
6. **사설(6면)** — 다크(잉크) 클로징: 스테이트먼트 + 2027 전망 트리오(IMF/WB/OECD) + 콜로폰.

## 실증한 PPT 요소
- 마스트헤드 더블룰 + 다단 컬럼 룰(세로 헤어라인) + folio/러닝 헤드 — 신문 조판 헬퍼 세트 신작.
- 라틴 세리프(Georgia) ↔ 한글 고딕 **하이브리드 타이포**(제호·수치는 세리프, 한글 본문은 고딕 — 한글 세리프 폴백 위험 회피).
- 네이티브 가로 막대 + point_colors(2색 강조) + 축/데이터라벨.
- 네이티브 표(헤더 잉크블루 + 제브라 + 컬럼 정렬).
- PIL 듀오톤 리드 아트(저작권 무관) standalone 배치.
- 다크/라이트 샌드위치(1면 라이트 ↔ 사설 다크).

## 데이터 (2+ 출처 교차)
- 세계 성장률 **2026: 2.5%(세계은행 GEP, 시장환율) · 3.1%(IMF WEO, PPP) · 2.9%(OECD)** — ⚠️ 가중방식별 편차(플래그). 2025 2.9% → 2026 둔화 → 2027 회복(WB 2.8 / IMF 3.2 / OECD 3.0).
- 지역별 EMDE(2026, 세계은행): 남아시아 6.3 · 동아시아·태평양 4.2 · 사하라이남 4.0 · 라틴 2.2 · 유럽·중앙아 2.1 · 중동·북아프리카 1.6.
- 물가 **2026 4.0%**(2025 3.3%에서 반등) · 브렌트유 평균 **$94**(+36%) · 하방 시나리오 성장 1.3%/물가 4.4%.
- 주요국 2026: 미국 2.0–2.4%(OECD·IMF) · 중국 4.4%(OECD) · 개도국 3.6%(팬데믹 이후 최저).
- ⚠️ 모두 **전망치**이며 중동 분쟁 전개에 종속(IMF "분쟁이 제한적이라는 가정"). 출처: IMF WEO(2026.4) · World Bank GEP(2026.6) · OECD Economic Outlook 잠정(2026.3).
