# 덱 #10 — 풀블리드 데이터 포스터 (K-라면/K-푸드 수출)

> 갤러리 #401 · 토마토/차콜/크림/머스터드 볼드 색면 · 7 슬라이드 · 결함 0

**한 줄 콘셉트** — ★레이아웃 아키타입: **풀블리드 데이터 포스터**. 슬라이드당 1개 거대 요소(초대형 모노 숫자/스테이트먼트 또는 풀폭 차트), 엣지투엣지 볼드 색면(슬라이드마다 색 교체), 미니멀 크롬(코너 마이크로 라벨만). 5번째 구성 — rail/dashboard/editorial/timeline과 다름.

## Backend
- **COM** (라이브 PowerPoint via `hyve-office.exe`).

## 이렇게 말하면 이 덱이 나온다 (자연어 요청 프롬프트)
> "K-라면 수출 데이터로 **풀블리드 데이터 포스터** 7장 — 슬라이드마다 **엣지투엣지 볼드 색면(토마토/차콜/크림/머스터드)** + **초대형 모노 숫자 하나**(150~200pt). 크롬은 코너 마이크로 라벨만. 표지(타이틀)·$1.25B(라면)·$13B(K-Food+)·시장(풀폭 막대)·+31%·2×(멕시코)·클로징. 한 슬라이드=한 메시지. 2출처 교차."

## 디자인 토큰
| 역할 | 값 |
|---|---|
| 색면(풀블리드) | 토마토 `#E23A2E` · 차콜 `#1C1714` · 크림 `#F4EEE1` · 머스터드 `#E8A93C` |
| 초대형 숫자 | **Consolas**(모노) 150~200pt |
| 헤드/본문 | 맑은 고딕 Bold |
| 크롬 | 코너 태그(좌상) + 페이지(우하)만 — 푸터·레일·킥커블록 없음 |

## ★레이아웃 아키타입: 풀블리드 포스터 (전용 헬퍼)
- `hero_number(value, label, num_color, lbl_color, num_size)` — 캔버스를 채우는 초대형 숫자 + 라벨. `corner_tag` — 미니멀 크롬. 슬라이드 bg(set_slide_background)가 곧 포스터 색면. **rail/dashboard/editorial/timeline 헬퍼 미재사용.**
- 한 슬라이드 = 한 거대 요소(숫자 or 풀폭 차트 or 스테이트먼트). 색면이 슬라이드마다 바뀌어 리듬을 만든다.

## 레이아웃 레시피 (7 포스터)
1. 표지 — 토마토 색면 + 거대 타이틀.
2. `$1.25B` — 차콜 색면 + 머스터드 초대형 숫자(라면 수출).
3. `$13.0B` — 크림 색면 + 토마토 초대형 숫자(K-Food+).
4. 시장 — 크림 색면 + 풀폭 가로 막대(미국·중국 토마토, 멕시코 머스터드).
5. `+31%` — 토마토 색면 + 크림 초대형 숫자.
6. `2×` — 머스터드 색면 + 차콜 초대형 숫자(멕시코).
7. 클로징 — 차콜 색면 + 스테이트먼트 + 머스터드 키스탯 블록 + 콜로폰.

## 실증한 PPT 요소
- 슬라이드별 풀블리드 색면(set_slide_background) + 초대형 모노 타이포 포스터 조판.
- 풀폭 가로 막대 + point_colors(멕시코 머스터드 강조) + 축/데이터라벨.

## 데이터 (2+ 출처 교차)
- 라면 수출 2024 **$1.25B(12.5억 달러, 약 1.8조원)**, +31%, **사상 최대**. 9월 월 1억 달러 첫 돌파.
- **K-Food+ 전체 $13B(130억 달러)**, 역대 최고, 14개 품목 최고.
- 상위 시장: 미국 $0.22B·중국 $0.21B·네덜란드·일본·멕시코. **멕시코 전년比 2배**(중남미 급성장).
- 출처: 관세청·농식품부(MAFRA) K-Food+·뉴스1·MBC·한국경제.
