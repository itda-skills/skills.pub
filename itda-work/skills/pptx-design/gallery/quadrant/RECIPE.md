**한 줄 콘셉트** — ★레이아웃 아키타입: **쿼드런트 매트릭스(2×2 포지셔닝 맵)**. 두 축(가격·규모)으로 사분면을 나누고, 각 항목을 **값→픽셀로 직접 계산해 버블(oval)로 수동 플롯**한다. fintech(#7)의 네이티브 scatter와 달리, 사분면 프레임워크(틴트 영역 + 사분면 라벨 + 축·그리드 + 라벨 버블)를 손좌표로 구성하는 **컨설팅 포지셔닝 맵**이다. 10번째 구성, 전용 헬퍼 신작.

## ★레이아웃 아키타입: 쿼드런트 매트릭스 (전용 헬퍼)
- `plot(price, subs) → (px, py)` — **값→픽셀 선형 매핑**(축 min/max, 플롯 사각형). 세로는 반전(큰 값=위).
- `quad_of(price, subs)` — 두 디바이더(`div`)로 사분면(tr/tl/br/bl) 자동 판정 → 버블 색·멤버십 결정.
- `oval(...)` — **버블(oval 도형)** + 흰 외곽선. 리더(Netflix)는 반경 확대.
- 축 프레임: 사분면 틴트 rect 4개(배경) → 그리드/눈금 → 축 크로스 강조 → 축 타이틀 → 사분면 코너 라벨 → **버블+라벨 순서로 z-order**.
- 라벨 충돌 회피: 동일 좌표(Disney+·Max 둘 다 $16.99)는 `jx` 지터 + `lpos`(좌/우) 라벨 앵커로 분리.
- `native_table`로 데이터 표. **rail/dashboard/scatter 차트 헬퍼 미재사용.**

## 레이아웃 레시피 (6 슬라이드)
1. **표지** — 타이틀 + 두 축·**사분면 4색 키**(색 점 + 라벨 + 고가/저가·대형/소형).
2. **포지셔닝 맵** — 틴트 사분면 + 축·그리드 + **7 버블 수동 플롯** + 라벨 + 우측 사분면 읽기 레일. (★시그니처)
3. **사분면 해설** — 2×2 카드(맵과 같은 자리 배치) · 전략 노트 + 멤버.
4. **데이터** — 네이티브 표(구독자 순위·요금·사분면) + 데이터 주의(플래그) 레일.
5. **인사이트** — 3줄 요약(번호) + `2.5×` 하이라이트 밴드.
6. **클로징(다크)** — 스테이트먼트 + 콜로폰.

## 실증한 PPT 요소
- **수동 좌표 플롯**(값→픽셀) — 네이티브 scatter 대신 oval 버블을 직접 배치(신규 기법, fintech #7과 차별).
- oval(타원) 도형 + 흰 외곽선, 사분면 자동 색 판정.
- 사분면 틴트 영역 + 축 크로스 + 그리드 + 눈금 라벨 손조판.
- 라벨 충돌 회피(지터 + 좌/우 앵커).
- 네이티브 표 + 다크/라이트 샌드위치.
