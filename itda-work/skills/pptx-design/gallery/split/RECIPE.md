# 덱 #12 — 스플릿스크린 (글로벌 커피 시장)

> 갤러리 #401 · 에스프레소/크림/카라멜/커피그린 · 6 슬라이드 · 결함 0

**한 줄 콘셉트** — ★레이아웃 아키타입: **스플릿스크린**. 슬라이드를 50/50 두 패널로 나눠 한쪽은 색면/PIL 듀오톤 이미지, 다른 쪽은 콘텐츠+차트. **슬라이드마다 좌우가 교대**해 리듬을 만든다. 7번째 구성.

## Backend
- **COM** (라이브 PowerPoint via `hyve-office.exe`).

## 이렇게 말하면 이 덱이 나온다 (자연어 요청 프롬프트)
> "글로벌 커피 시장으로 **스플릿스크린** 6장 — 화면을 **50/50 두 패널**로 나눠 한쪽은 커피톤 이미지/색면, 다른 쪽은 제목·데이터·차트. **슬라이드마다 좌우 교대**. 에스프레소/크림/카라멜/커피그린 따뜻한 톤. 표지·시장규모·생산집중(막대)·가격충격(빅스탯)·브라질수출·클로징. 2출처 교차+시장규모 정의편차 플래그."

## ★레이아웃 아키타입: 스플릿스크린 (전용 헬퍼)
- `panel_cmds(side, img, bg)` — 50/50 분할: 한쪽 색면 rect + 다른 쪽 이미지. `content_x(side)` — 콘텐츠 패널 좌표. `side`를 슬라이드마다 L/R 교대.
- **이미지 라우팅**: 이미지 명령은 verb 없음(type=image) → 빌드엔진이 cmds에서 분리해 standalone add 경로로(텍스트 아래에 깔림). 색면 rect는 batch verb. **rail/dashboard/editorial/timeline/poster/swiss 헬퍼 미재사용.**

## 레이아웃 레시피 (6 슬라이드, 좌우 교대)
1. 표지(L콘텐츠) — 에스프레소 패널 타이틀 + 커피 이미지.
2. 시장규모(R) — `$137.97B` + CAGR.
3. 생산집중(L) — 막대(브라질 38% 강조) + 이미지.
4. 가격충격(R) — `+80%`/`+120%` 빅스탯(에스프레소).
5. 브라질수출(L) — `47.3M` + `+32.7%`.
6. 클로징(R) — 스테이트먼트 + 콜로폰.

## 실증한 PPT 요소
- 50/50 스플릿 패널(색면 rect + PIL 이미지) 좌우 교대 조판.
- 네이티브 bar + point_colors(브라질 강조) + 축/데이터라벨.
- ★불필요 개행 점검: 클로징 스테이트먼트가 "두 나라"/"의 날씨가"로 어색하게 끊겨 **수동 개행(값은\n두 나라…)으로 교정** — SKILL.md 관문4 신규 지침 실적용.

## 데이터 (2+ 출처 교차)
- 시장 **$137.97B(2024)**, +6.2% CAGR → $223.25B(2032). ⚠️ 시장 규모는 정의별 편차 — 플래그.
- 생산: **브라질 38%**(브라질+베트남 절반 이상). 베트남·콜롬비아·인니 비중은 근사(플래그).
- 가격: 아라비카 **+80%**($264.88/60kg)·로부스타 **+120%**($251.88), 세계 +37% **13년 최고(2024-12)**.
- 브라질 수출 **47.3M 자루(2023/24, +32.7%, 사상 최대)**.
