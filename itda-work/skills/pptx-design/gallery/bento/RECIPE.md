## ★레이아웃 아키타입: 벤토 그리드 (전용 헬퍼)
- `rrect(...)` — **라운드 타일**(`roundedRectangle` 도형). 모든 타일의 기본.
- `num_tile(si, x, y, w, h, num, label, sub, accent, num_size, flag)` — 액센트 점 + 거대 숫자(Arial Black, 액센트색) + 라벨 + 출처 마이크로카피 + (옵션)⚠ 플래그. 히어로는 `num_size`↑·타일 크기↑로 생성.
- `platform_tile(...)` — 타일 안에 **stacked bar**(매출 점유 3분할: 모바일/PC/콘솔)를 수동 조판.
- `dot(...)` — 타일 액센트 점(oval). **dashboard/rail 헬퍼 미재사용.**
- 비대칭 배치: 히어로(410×380) + 우상 2 소형(216·222×182) + 우하 와이드(454×182) 식으로 **사이즈를 의도적으로 섞는다**.

## 실증한 PPT 요소
- `roundedRectangle` 라운드 타일 모자이크 + 가변 사이즈 비대칭 배치(벤토).
- 타일 안 stacked bar(매출 점유 3분할) 수동 조판.
- 타일별 멀티 액센트(바이올렛 주 + 시안/라임/앰버/핑크) + 액센트 점.
- 다크 일관 테마(아케이드) + Arial Black 거대 숫자.
