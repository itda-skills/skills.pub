## 빠른 시작

Claude Code에게 이렇게 말해보세요:

```
카카오 이모티콘 만들어줘
내 사진으로 ghibli 스타일 이모티콘 만들고 싶어
귀여운 강아지 캐릭터 병맛 스타일로 만들어줘
```

사진 한 장 또는 텍스트 설명만 있으면 베이스 캐릭터 생성부터 32개 감정 세트 + 아이콘, 심사 제출용 ZIP까지 자동으로 만들어드립니다.

## 카카오 이모티콘 심사 규격

> **멈춰있는 이모티콘** 유형만 지원합니다. (움직이는·큰·미니 이모티콘 미지원)

| 구성 | 개수 | 형식 | 사이즈 | 최대 용량 |
|------|------|------|--------|----------|
| **이모티콘 이미지** | 32개 | PNG | 360 × 360 px | 150 KB |
| **아이콘 이미지** | 1개 | PNG | 78 × 78 px | 16 KB |

전체 규격은 [카카오 이모티콘 이미지 가이드](https://kakaoemoticonstudio.notion.site/image-guide?v=23ae52082d858072aaca000cbbc4588e)를 참고하세요.

## 사전 준비: Gemini API 키 등록

이 스킬은 이미지를 생성하기 위해 Google Gemini API를 사용합니다.

**Step 1** — [Google AI Studio](https://aistudio.google.com/)에서 API 키를 발급받으세요.

> **무료 티어 주의**: 무료 티어는 분당 요청 수 제한이 있어, 32개 이모티콘 세트를 생성하다가 중간에 실패할 수 있습니다. **유료 결제(Pay-as-you-go)를 활성화**하는 것을 권장합니다. 이미지 1장에 6개씩 배치해 생성하므로 32개 세트 기준 약 6장 = **약 600원** 수준입니다.

**Step 2** — Claude Cowork **지침(설정)**에 API 키를 등록합니다:

```
GEMINI_API_KEY=your-api-key-here
```

지침 등록 방법: Claude Cowork 우측 상단 설정 → 지침 → 위 내용 추가

## 스타일 프리셋

8가지 스타일 중에서 분위기에 맞는 것을 선택하세요.

| 스타일 | 키 | 분위기 | 추천 상황 |
|--------|-----|--------|----------|
| 병맛 스케치 | `byungmat` | 의도적으로 엉성한 낙서풍 | B급 감성, 친구 사이, 분위기 풀기 |
| 지브리풍 | `ghibli` | 수채화풍 따뜻한 파스텔 | 감성적, 선물용 |
| 미니멀 라인 | `minimal` | 깔끔한 선화 | 직장용, 슬랙·팀즈 |
| 귀여운 3D | `clay3d` | 말랑말랑 클레이 질감 | 대중적, 범용 (가장 인기) |
| 애니메이션 | `anime` | 치비 스타일 | 역동적, 애니 팬 |
| 실사화 틸트시프트 | `realistic-tiltshift` | 미니어처 사진처럼 리얼 | 실물 닮은꼴, 인스타 감성 |
| 실사화 3D 피규어 | `realistic-3d` | 하이엔드 비닐 피규어 느낌 | 고급스러운 굿즈 이미지 |
| 커스텀 | `custom` | 직접 스타일 프롬프트 입력 | 원하는 스타일 자유롭게 |

**상황별 추천:**
- "귀엽게 만들어줘" → `clay3d` (대중적), `ghibli` (감성적), `anime` (활발한)
- "회사에서 쓸 거야" → `minimal` (깔끔), `byungmat` (분위기 풀기)
- "실제 사진처럼" → `realistic-tiltshift` (진짜 사진 느낌) vs `realistic-3d` (피규어 느낌)
- "웃기게 만들어줘" → `byungmat` 강력 추천

## 제작 과정

### Phase 1: 베이스 캐릭터 생성

- **사진 입력**: 실제 인물·동물·캐릭터 사진을 기반으로 생성 (최대 3장)
- **텍스트 입력**: "안경 쓴 30대 남성, 짧은 머리" 등 설명만으로도 생성 가능
- 생성된 이미지가 마음에 들지 않으면 같은 설명으로 재생성하거나, 새 설명으로 다시 시작할 수 있습니다.
- 이미 만들어둔 베이스 이미지가 있다면 바로 Phase 2로 건너뛸 수 있습니다.

### Phase 2: 감정/동작 세트 선택

베이스 캐릭터를 확인한 후 감정 세트를 고릅니다:

| 세트 | 감정 예시 |
|------|---------|
| **기본 감정 세트** (32종) | 기쁨, 슬픔, 화남, 놀람, 부끄러움, 사랑, 졸림, 화이팅… |
| **직장인 세트** (32종) | 파이팅, 퇴근, 야근, 감사합니다, 죄송합니다, 확인했습니다… |
| **일상 대화 세트** (32종) | 안녕, 밥먹자, 잘자, 사랑해, 힘내, 고마워, 미안해… |
| **직접 입력** | 쉼표로 구분해 원하는 감정/동작 목록 직접 지정 |

말풍선(감정 텍스트) 포함 여부도 선택할 수 있습니다.

### Phase 3: 이모티콘 세트 + 아이콘 생성

32개 이모티콘과 아이콘(78×78px)을 한 번에 생성합니다.

**출력 결과:**

```
.itda-skills/emoticon/set-{timestamp}/
  01.png ~ 32.png   (이모티콘 이미지, 360×360px)
  icon.png          (아이콘 이미지, 78×78px)
  kakao-set.zip     (심사 제출용 ZIP)
```

ZIP 파일을 [카카오 이모티콘 스튜디오](https://emoticonstudio.kakao.com/)에 업로드하면 심사 신청이 완료됩니다.

## 트리거 키워드

아래와 같이 말하면 이 스킬이 자동으로 활성화됩니다:

- 카카오 이모티콘 만들어줘, 이모티콘 만들어줘
- 내 사진으로 이모티콘 만들고 싶어
- 귀여운 캐릭터 스티커 만들어줘
- 카카오 이모티콘 심사 신청하려고
- 카카오 스티커 만들어줘, 이모티콘 제작해줘
- create Kakao emoticon, make emoticon set, generate sticker

## 제한사항

- **멈춰있는 이모티콘 전용**: 움직이는·큰·미니 이모티콘은 지원하지 않습니다
- **API 비용**: Gemini API 유료 사용량에 포함됩니다 (32개 세트 생성 시 약 6회 API 호출)
- **SAFETY 필터**: 인물 사진 입력 시 간헐적으로 안전 필터에 걸릴 수 있습니다 → 다른 사진으로 재시도하세요
- **아이콘 품질**: 베이스 캐릭터를 78×78px로 단순 리사이즈합니다. 별도 디자인이 필요하면 `--no-icon` 옵션 후 수동 제작을 권장합니다
- **생성 품질**: 입력 사진 품질이 높을수록 결과물도 좋아집니다
