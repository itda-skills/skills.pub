네이버 블로그의 글 목록·본문·댓글·블로그 내 검색을 로그인 없이 읽는 스킬입니다. **Claude에게 자연어로 말하면 됩니다** — 터미널 명령을 직접 입력할 필요는 없습니다.

---

## 빠른 시작

```
네이버 블로그 글 가져와줘
```

```
이 블로그 본문이랑 댓글 같이 보여줘
```

```
이 블로그 최근 7일 글 보여줘
```

---

## 활용 시나리오

### 최근 글 목록 조회

특정 블로그의 최근 포스트를 날짜 내림차순으로 가져옵니다.

```
아스트로유지 블로그 최근 7일 포스팅 가져와줘
```

최대 몇 개까지 가져올지, 특정 카테고리만 볼지도 함께 말할 수 있습니다.

```
아스트로유지 블로그 여행 카테고리 글 10개만 보여줘
```

### 본문 + 댓글 한 번에 보기

URL 하나로 본문과 댓글 트리를 동시에 가져오고 싶을 때 유용합니다.

```
이 블로그 포스팅 본문이랑 댓글 같이 보여줘
```

본문만 보고 싶으면 "본문만", 댓글만 보고 싶으면 "댓글만"이라고 말하세요.

### 블로그 내 키워드 검색

특정 블로그 안에서 키워드로 포스트를 찾습니다.

```
이 블로그에서 '인공지능' 검색해줘
```

```
아스트로유지 블로그에서 파이썬 관련 글 5개만 찾아줘
```

### 특정 작성자 댓글 모으기

```
이 글에 홍길동이 단 댓글만 모아줘
```

---

## 팁

- **이미지 URL 제거**: 이미지가 많은 블로그(여행·맛집·리뷰 등)를 읽을 때 "이미지는 빼고", "본문 텍스트만"처럼 말하면 이미지 URL이 제거되어 응답이 훨씬 가벼워집니다. 실측으로 이미지 26장 여행 포스트가 18,723 B → 6,522 B(-65%)로 줄었습니다. 단순히 "요약해줘" 같은 말에는 적용되지 않습니다.
- **자가차단 방지**: 동일 블로그에 대한 반복 조회는 줄이고, 필요한 정보를 한 번에 가져온 뒤 사용하세요. 과도한 연속 요청은 네이버 봇 감지에 걸릴 수 있습니다.
- **댓글 깊이 제한**: 댓글이 매우 많을 때는 "댓글 2단계까지만", "최대 50개만"처럼 범위를 말해주면 됩니다.

## 제한 사항

| 증상 | 원인 / 해결 |
|------|-------------|
| "차단됐다"는 안내 (`4`) | 네이버 봇 차단(403·429). 잠시 후 다시 시도하세요. anti-bot 우회는 하지 않습니다 |
| "결과가 없다"는 안내 (`3`) | 해당 날짜·카테고리에 게시물이 없거나 검색어에 맞는 글이 없음 |
| "비공개 블로그"라는 안내 (`6`) | 로그인이 필요한 비공개 포스트 — 지원하지 않습니다 |
| "지원하지 않는 플랫폼"이라는 안내 (`5`) | 네이버 외 도메인(Tistory·Velog·Brunch)은 아직 미지원입니다 |
| 이미지 위치가 `[이미지]`로만 표시됨 | "이미지는 빼고" 요청 시 URL은 제거되지만 이미지 위치와 설명은 남습니다 — 정상 동작입니다 |

- **조회 전용**: 포스트 작성·수정·댓글 달기는 지원하지 않습니다.
- **네이버 블로그만**: Tistory·Velog·Brunch는 향후 별도 지원 예정입니다.
- **캐시 없음**: 매번 신선하게 조회합니다.
