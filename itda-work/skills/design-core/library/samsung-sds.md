# Samsung SDS — 기업 IT·IR 발표 톤

## Overview

Samsung Blue `#1428A0` 가 시각 비중 60~70%를 지배한다(표지/클로징 배경·헤드라인·차트 주 시리즈). 라이트 블루 `#5B8DEF` 가 보조(보조 막대·패널·강조 칩), 화이트가 여백을 맡는다. 장식이 아니라 **절제와 규율**이 코퍼레이트 신뢰를 만든다: 같은 위치의 킥커, 같은 위치의 푸터, 일관된 헤어라인. 멀티 시리즈 차트의 위계는 채도로 만든다(주인공=Samsung Blue, 비교군=중간 블루 `#6E84CF`·그레이 `#A9B2C3`).

## 슬라이드 문법

- **표지(블루)**: Samsung Blue 풀블리드(또는 상단 블루 밴드) 위 대문자 영문 킥커 + 보고서 제목 + 주장형 부제 + 기준일 칩. 하단에 핵심 KPI 스트립 3~4종(라벨 작게·숫자 크게, 헤어라인 구분).
- **Executive Summary**: 번호 칩(01/02/03) + 소제목 + 1줄 근거 + 우측 정렬 빅넘버. 3행으로 압축.
- **차트 슬라이드**: 좌 차트(폭 2/3) + 우 KPI 레일(헤어라인 구분 스탯 3~4개), 또는 풀폭 차트 + 하단 스탯 스트립. 차트가 주인공이면 표를 겹치지 않는다.
- **그리드 슬라이드(솔루션·아키텍처)**: 2×3 카드(surface 배경 + 번호 칩 + 볼드 소제목 + 1~2줄). 카드 높이는 본문에 맞춰 줄인다.
- **클로징(블루)**: 핵심 권고 한 문장 + 기준일·출처·면책 3블록.
- **푸터(전 슬라이드)**: 헤어라인 위 좌측 `자료: …`, 우측 `NN / NN`.

## 차트 권장

- 분기 실적·KPI 비교 → 네이티브 편집 차트(`add_native_chart`), 시리즈 색은 본 팔레트 hex 1:1.
- 기본 파랑/주황 금지. 주 시리즈 Samsung Blue, 비교군은 채도 낮춘 블루·그레이.

> 출처 주: 색은 Samsung 공식 CI 의 공개 Blue(#1428A0)를 참조한 **스타일 프리셋**이며, 삼성SDS 의 사내 브랜드 가이드 원본을 재현하지 않는다. 정확한 브랜드 적용이 필요하면 조직 브랜드 가이드의 hex 로 `colors` 를 교체하라.
