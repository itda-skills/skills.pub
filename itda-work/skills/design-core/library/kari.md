# KARI — 항공우주·연구 성과 발표 톤 (딥스페이스 다크)

## Overview

딥스페이스 네이비 `#0A1430` 가 캔버스를 덮고, Royal Blue `#3C2DA0` 가 구조(밴드·헤드라인·차트 주 시리즈)를 잡는다. 시안 틸 `#2BC0BD` 는 데이터·미션 강조에만 희소하게 쓰여 우주 톤에 생기를 준다. 정체성은 **어둠과 글로우의 대비** — 풀블리드 다크 위 Royal Blue→시안 그라디언트 글로우(Pillow 베이크)가 우주의 깊이를 만든다. 멀티 시리즈는 채도/밝기로 위계화(주인공=시안, 비교군=중간 블루 `#5566B8`·그레이 `#93A0C0`).

## 슬라이드 문법

- **표지(다크)**: 딥네이비 풀블리드 + Royal Blue→시안 라디얼 글로우 위 대문자 영문 킥커 + 미션/보고서 제목 + 주장형 부제 + 기준일 칩. 하단 핵심 성과 KPI 스트립 3~4종.
- **미션 요약**: 번호 칩(01/02/03) + 소제목 + 1줄 근거 + 우측 정렬 빅넘버(발사·궤도·관측 수치).
- **차트 슬라이드**: 좌 차트(폭 2/3, 다크 투명 배경·밝은 축) + 우 KPI 레일. 시안 시리즈가 주인공, 헤어라인으로 스탯 구분.
- **성과 그리드(미션·기술)**: 2×3 카드(surface `#14224A` + 번호 칩 + 볼드 소제목 + 1~2줄). 시안 마커로 상태 표시.
- **클로징(다크)**: 핵심 비전 한 문장 + 기준일·출처·면책 3블록.
- **푸터(전 슬라이드)**: 시안 헤어라인 위 좌측 `자료: …`, 우측 `NN / NN`.

## 차트 권장

- 발사·궤도·관측 시계열 → matplotlib PNG(`transparent=True`, 밝은 축/라벨) 또는 네이티브 편집 차트.
- 기본 파랑/주황 금지. 주 시리즈 시안 `#2BC0BD`, 비교군 Royal Blue·그레이. 다크 배경 위 대비 우선.

> 출처 주: 색은 KARI 공개 CI 페이지의 Main Royal Blue(RGB 60,45,160)·Sub Green Blue(teal)를 참조한 **스타일 프리셋**이다. 공개 페이지의 RGB↔HEX 표기가 상충(RGB 60,45,160 ↔ HEX #2a5caa)해 **RGB 를 신뢰원으로 변환**(#3C2DA0)했고, 다크 캔버스·글로우는 우주 톤으로 파생했다(로고 정밀 재현 아님). 정확한 적용이 필요하면 기관 브랜드 가이드 hex 로 `colors` 를 교체하라.
