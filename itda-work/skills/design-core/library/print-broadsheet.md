# Print Broadsheet — 인쇄 분석 리포트 문법

## Overview

페이퍼 `#F4F1EA` 위 잉크 `#111111` — 종이와 잉크의 대비가 전부다. 잉크블루 `#0047AB` 는
유일한 액센트로 절제해서 쓴다. 정체성은 구조에서 온다: 모든 슬라이드 상단의 마스트헤드
(좌 발행물명/섹션, 우 날짜·페이지, 아래 굵은 헤어라인)와 2~3컬럼 본문 그리드.

## 슬라이드 문법

- **표지**: 마스트헤드 + 1면 헤드라인(대형 세리프 라틴 + 한글 고딕) + 리드 문단 + 키 스탯 박스.
- **본문**: 마스트헤드 반복 + 2~3컬럼(컬럼 사이 가는 헤어라인). 표 대신 컬럼 안 미니 리스트.
- **차트**: 화이트 surface 박스에 차트를 '도판'처럼 얹고 캡션(도판 번호 + 출처)을 단다.
- **클로징**: 사설 스타일 — 결론 한 단락 + 굵은 헤어라인 마감.

## 차트 권장

차트는 surface(화이트) 박스 위에 잉크·잉크블루 2색 위주로. 그리드라인은 hairline 색으로 가늘게.
