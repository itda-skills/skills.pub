이미지 정보 조회, 리사이즈, 가장자리 크롭, DPI 변경, 포맷 변환, 회전·반전을 처리합니다.
**Claude에게 말로 부탁하면 됩니다** — "이미지 크기 줄여줘", "PNG를 JPG로 바꿔줘"처럼요.

---

## 빠른 시작

```
이 사진 정보 알려줘
이미지 크기 줄여줘 (1920px로)
PNG를 JPG로 바꿔줘
```

파일 경로만 알려주면 스킬이 어떤 작업인지 판단해서 실행합니다.

---

## 활용 시나리오

### 웹 업로드용 리사이즈·변환

블로그·SNS에 올리기 전 용량을 줄입니다.

```
photo.jpg를 가로 1920px로 줄여줘
스크린샷을 JPG로 바꿔서 용량 줄여줘
```

### 인쇄 준비 (DPI 300)

온라인 인쇄 업체가 요구하는 300 DPI로 맞춥니다.

```
이 사진 DPI 300으로 바꿔줘
```

### 파일 관리 (여백 정리·회전)

스캔본의 흰 여백 제거나 사진 방향 보정에 사용합니다.

```
스캔본 여백 자동으로 잘라줘
사진 90도 회전해줘
```

## 출력 파일 기본 이름 규칙

별도로 저장 경로를 지정하지 않으면 입력 파일과 같은 폴더에 아래 접미사로 저장합니다.

| 입력 파일 | 작업 | 기본 출력 파일 |
|-----------|------|----------------|
| `photo.jpg` | 리사이즈 | `photo-resized.jpg` |
| `scan.jpg` | DPI 변경 | `scan-300dpi.jpg` |
| `logo.png` | 포맷 변환 | `logo.jpg` |
| `image.png` | 여백 크롭 | `image-cropped.png` |
| `photo.jpg` | 회전 | `photo-rotated.jpg` |

## 리사이즈 모드

크기를 지정할 때 비율 처리 방식을 말해주면 됩니다.

| 모드 | 동작 | 사용 케이스 |
|------|------|-------------|
| 비율 유지 (기본) | 지정 크기 안에 맞춤 | 일반 리사이즈 |
| 채우기 | 지정 크기를 가득 채움 | 썸네일, 배경 이미지 |
| 정확히 | 비율 무시, 정확한 크기로 | 특정 규격이 필요한 경우 |

```
1920×1080으로 딱 맞춰줘 (비율 무시)
800px 정사각 썸네일로 만들어줘 (채우기)
```

## 팁

- **자동 여백 크롭**: "여백 자동으로 잘라줘"라고 말하면 균일한 가장자리를 자동 감지해 제거합니다. 감지 민감도 조정이 필요하면 "좀 더 공격적으로 크롭해줘" / "덜 잘라줘"라고 말하세요.
- **저장 전 미리보기**: "실제로 저장하지 말고 결과만 보여줘"라고 하면 파일을 생성하지 않고 예상 수치만 확인할 수 있습니다. 큰 이미지를 다루기 전에 먼저 확인하세요.
- **DPI 변경은 고품질 기본값**: DPI 변경 작업은 화질 손실을 최소화하기 위해 JPEG 품질을 높게 유지합니다.
- **덮어쓰기 보호**: 출력 파일이 이미 있으면 실수로 덮어쓰지 않도록 에러(`OUTPUT_EXISTS`)를 반환합니다. 덮어쓰기를 원하면 "기존 파일 덮어써도 돼"라고 말하세요.

## 제한사항 및 에러 안내

| 증상 | 원인 / 해결 |
|------|-------------|
| "지원하지 않는 포맷" 안내 (`UNSUPPORTED_FORMAT`) | JPEG·PNG 외 포맷(WebP·HEIC·TIFF 등)은 지원하지 않습니다 |
| "파일을 찾을 수 없음" 안내 (`FILE_NOT_FOUND`) | 파일 경로를 다시 확인하세요 |
| "출력 파일이 이미 있음" 안내 (`OUTPUT_EXISTS`) | 다른 경로를 지정하거나 덮어쓰기를 명시적으로 허용하세요 |
| "같은 포맷으로 변환 불가" 안내 (`SAME_FORMAT`) | 변환 전후의 확장자가 같으면 의미가 없습니다. 다른 포맷을 지정하세요 |
| 10MB 이상 이미지 | 처리 시간이 더 걸릴 수 있습니다 |
