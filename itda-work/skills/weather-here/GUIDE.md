---
title: "weather-here (현재 위치 날씨) 활용 가이드"
---

## 빠른 시작

현재 대략적 위치 또는 지정한 지역의 날씨를 한국어로 빠르게 알려줍니다.
위치를 말하지 않아도 되묻지 않고 바로 답합니다.

```
지금 날씨 어때
```

```
부산 날씨 알려줘
```

```
수원 날씨, 자세히
```

위치를 말하지 않으면 인터넷 접속 위치로 대략의 지역을 자동 추정하고,
지역명(한국어 또는 주요 도시 영문)을 말하면 그 지역을 우선합니다.

## 사전 준비

**없습니다.** 인증키·회원가입·신청이 전혀 필요 없습니다. 인터넷
접속만 되면 설치 직후 바로 사용할 수 있습니다.

## 활용 시나리오

### 위치 안 말하고 바로 묻기

가장 흔한 사용법입니다. 현관 나서기 전 "비 와?" 같은 순간.

```
지금 날씨
```

위치를 되묻지 않고 접속 위치 기준으로 바로 답합니다.

### 특정 지역 지정 (한국어·영문)

```
대전 날씨 알려줘
```

```
Busan 날씨
```

시·도(서울·부산·경기 등)와 시군구(해운대구 등), 그리고 일반구가 있는
시(수원·성남·창원 등)를 지원합니다. 주요 도시는 영문 이름도 됩니다.
지역명은 기상청 공식 좌표표를 기준으로 해석해 엉뚱한 곳의 날씨가
나오지 않습니다.

### 상세 수치까지 보기

```
서울 날씨 자세히
```

기본은 한 줄 요약(지역·상태·강수확률)이고, "자세히/상세"를 붙이면
기온·습도·강수량·바람까지 보여줍니다.

## 출력 옵션

| 모드 | 보여주는 것 | 사용 시점 | 부르는 법 |
|------|-------------|-----------|-----------|
| 기본(요약) | 지역, 날씨 상태, 강수확률 한 줄 | 외출 전 빠른 확인 | "지금 날씨" |
| 상세 | 기온·습도·강수량·바람까지 | 수치가 필요할 때 | "자세히" / "상세" 덧붙이기 |
| 지역 지정 | 다른 지역의 날씨 | 현재 위치 말고 다른 곳 | "부산 날씨" 처럼 지역명 명시 |

### 결과 화면 예시

기본:

```
서울 · 흐림, 강수확률 30% — 우산 챙기는 게 안전합니다.
```

자세히:

```
서울 · 흐림 (강수확률 30%)
기온 12℃ · 습도 68% · 강수량 0mm · 풍속 2.1m/s
```

## 팁

- **되묻지 않음** — 위치를 안 말해도 자동 추정해 바로 답합니다. 추정이
  빗나가면 지역명을 직접 말해 주세요(그 지역이 우선됩니다).
- **키 발급 불필요** — 별도 신청이 없어 설치 직후 바로 동작합니다.
- **해외 위치는 대략값만** — 한국 밖이면 "(해외·대략·미검증)" 라벨과 함께
  대략값만 제공합니다. 정확도를 보장하지 않습니다.
- ⚠️ **읍·면·동 단위는 지원 안 함** — 시·도와 시군구(일반구 포함)까지만
  지원합니다. 같은 시 안의 동네별 차이는 구분하지 않습니다.
- ⚠️ **항상 최신 조회** — 호출할 때마다 새로 가져오며 이전 결과를 재사용하지 않습니다.
- ⚠️ **조회 실패 시** — "잠시 후 다시 시도" 안내가 나오면 일시적 네트워크
  문제이니 잠깐 뒤 다시 요청하세요.
