# domain-electronics.md — 전기·전자·반도체

> 도메인: 전기·전자·반도체 | 매칭 키워드: 회로·임피던스·트랜지스터·datasheet·MOSFET·옴

## 1. 수식·기호·단위 처리

### 수식 패턴

이공계 PDF의 수식은 LaTeX 표기로 복원한다.

- **인라인 수식**: `$V = IR$`, `$P = IV$`, `$Z = R + jX$`
- **LaTeX 명령**: `\alpha`, `\beta`, `\omega`, `\mu` 등 그리스 문자
- **블록 수식**: 독립 행에 위치하는 수식은 별도 블록으로

verify_quality.py 검증 패턴: `\$[^$]+\$|\\[a-zA-Z]+` (수식 존재 ≥1)

### 전기 단위

| 단위 기호 | 표기 | 예시 |
|----------|------|------|
| 옴 (저항) | Ω | `10Ω`, `4.7kΩ` |
| 패럿 (정전용량) | F | `100μF`, `10pF` |
| 헨리 (인덕턴스) | H | `1mH`, `10μH` |
| 헤르츠 (주파수) | Hz | `50Hz`, `1MHz` |
| 암페어 (전류) | A | `100mA`, `2.5A` |
| 볼트 (전압) | V | `3.3V`, `5V`, `12V` |
| 와트 (전력) | W | `100mW`, `10W` |

단위 접두사: `p(피코)`, `n(나노)`, `μ(마이크로)`, `m(밀리)`, `k(킬로)`, `M(메가)`, `G(기가)`

## 2. 다중헤더 표·각주 처리

### Datasheet 전기 특성 표

datasheet의 전기적 특성(Electrical Characteristics) 표는 복잡한 병합 셀 구조를 가진다.

```markdown
| Parameter | Symbol | Min | Typ | Max | Unit |
|-----------|--------|-----|-----|-----|------|
| Input Voltage | Vin | 3.0 | 3.3 | 3.6 | V |
| Output Current | Iout | — | — | 500 | mA |
```

- 조건 열(`Conditions`)이 있으면 그대로 보존
- `Min/Typ/Max` 구조 유지, 값 없는 셀은 `—`

### 핀맵 표

IC 핀 배치 표:

```markdown
| Pin | Name | Type | Description |
|-----|------|------|-------------|
| 1 | VDD | Power | Supply voltage (3.3V) |
| 2 | GND | Ground | Ground reference |
```

## 3. 참조 번호 형식 보존

- **부품 번호**: `BC547`, `IRF540N`, `LM358`, `ATmega328P` — 대소문자 정확 보존
- **회로도 참조**: `R1`, `C2`, `Q3`, `U4` (부품 지정자)
- **문서 참조**: `Figure 3-2`, `Table 4.1`, `Appendix A`
- **스펙 참조**: `IEC 60950-1`, `IEEE 802.11`, `USB 3.1 Gen 2`

## 4. 이미지·다이어그램 감지 신호

다음 페이지는 이미지 모드로 처리한다:

- **회로도**: 저항, 콘덴서, 트랜지스터 기호가 그려진 페이지
- **핀 다이어그램**: IC 패키지 핀 배치 그림
- **파형 그래프**: 오실로스코프 파형, 주파수 응답 그래프
- **블록 다이어그램**: 시스템 아키텍처 블록

감지 기준: 텍스트 밀도 < 30% 이고 규칙적인 심볼 패턴

## 5. 코드 블록·특수 콘텐츠

펌웨어·임베디드 코드 보존:

```c
// 레지스터 설정 코드
PORTB |= (1 << PB0);   // PB0 HIGH
DDRB = 0xFF;           // 전체 출력 설정
```

어셈블리, C, Python 코드는 언어별 코드 펜스로 감싼다.

## 6. 도메인 OCR 아티팩트 패턴

### 영문 매뉴얼 페이지 헤더

전자 부품 datasheet·매뉴얼 전형적인 OCR 아티팩트 — 제거 대상:

```
3 MOSFET Gate Driver 28
12 Application Note AN-1234
```

패턴: `^\d+\s+[A-Za-z][A-Za-z\s]+\d+$` (페이지번호 + 섹션명 + 페이지번호)

### 단위 붙여쓰기 누락

- `10 Ω` → `10Ω` 또는 `10 Ω` (공백 통일)
- `3 . 3 V` → `3.3V` (소수점 공백 제거)
- `1 0 0 m A` → `100mA` (숫자 사이 공백 제거)

### 기호 OCR 오인식

| 원문 기호 | OCR 오인식 | 정정 |
|---------|-----------|------|
| Ω (옴) | S2, Q | Ω |
| μ (마이크로) | u, M | μ |
| ≥ (이상) | >, 2 | ≥ |
| ± (플러스마이너스) | +-, t | ± |
