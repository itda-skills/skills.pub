# domain-tax-accounting.md — 세무·회계

> 도메인: 세무·회계 | 매칭 키워드: 법인세·조특법·법인령·매출원가·손익계산서·세액감면

## 1. 수식·기호·단위 처리

한국 세무 문서의 수치 표현 패턴:

- 금액: 쉼표 단위 구분 (`1,000,000원`, `₩1,000,000`)
- 세율: 퍼센트 표기 (`9%`, `20%`, `과세표준의 20/100`)
- 법정 계산식: 들여쓰기 단계별 표시

  ```
  산출세액 = 과세표준 × 세율
  납부세액 = 산출세액 - 세액공제 - 세액감면
  ```

- 법령 조항 인용 시 분수 형태 보존: `소득금액의 1/2`, `72④`

## 2. 다중헤더 표·각주 처리

세무 교재·서식의 복합 표 구조:

- **병합 셀**: 수직 병합이 있는 경우 첫 행에 내용 기입, 하위 셀은 `↑` 또는 `(위와 같음)` 표시
- **각주**: 표 하단 `※`, `주)`, `1)` 형태 각주는 표 바로 아래 blockquote(`>`)로 이전
- **서식 번호**: `별표 1`, `별지 제3호 서식` 형태는 정확히 보존

## 3. 참조 번호 형식 보존

법령 참조 정확한 인용 형식:

| 원문 표현 | 마크다운 보존 형식 |
|-----------|-------------------|
| 조특법 제7조 | `조특법 제7조` |
| 법인령 72④ | `법인령 72④` |
| 소득령 제25조 제1항 | `소득령 제25조 제1항` |
| 법인세법 제60조 | `법인세법 제60조` |

- `조·항·호·목` 계층 구조 보존 (`제7조 제1항 제2호 가목`)
- 별표·서식 번호 형식 정확 보존 (`별표 1`, `별지 제3호 서식`)

## 4. 이미지·다이어그램 감지 신호

세무 문서에서 이미지 모드로 처리할 페이지:

- **서식 페이지**: 빈 칸이 많은 그리드, `□`, `○`, 날인란
- **흐름도**: 세액 계산 흐름도, 신고 절차 다이어그램
- **저밀도 페이지**: 텍스트 밀도가 낮고 구조화된 칸 위주

```bash
# 서식 페이지 감지 후 이미지 처리
pdftoppm -f <page> -l <page> -png -r 200 "<pdf_path>" .itda-skills/page
```

## 5. 코드 블록·특수 콘텐츠

- **법령 본문 인용**: `>` 블록으로 감싸기
  ```markdown
  > **조세특례제한법 제7조 (중소기업 세액감면)**
  > 중소기업이 … 이익에 대한 소득세 또는 법인세의 100분의 [X]에 상당하는 세액을 감면한다.
  ```
- **계산 예시**: 들여쓰기 코드 블록으로 단계별 표시
- **참고사항·예외**: `> **참고**` 형태로 강조

## 6. 도메인 OCR 아티팩트 패턴

### 페이지 머리글·꼬리글 혼입

세무 교재 전형적인 OCR 아티팩트 — 제거 대상:

```
84  2026 법인세 신고 핵심실무
Ⅱ. 법인세 결산과 신고  95
```

패턴: `^\d{1,3}\s{2,}\d{4}\s` (줄 앞 페이지 번호 + 2칸 이상 공백 + 4자리 연도)

### 한국어 띄어쓰기 누락

세무 OCR의 가장 흔한 아티팩트. 재구성 예시:

- 원시: `재고자산이라함은손익계산서의매출원가`
- 정리: `재고자산이라 함은 손익계산서의 매출원가`

조사(은/는/이/가/을/를/에/의/로/와/과/도/만/까지)는 앞 단어에 붙이고 뒤에 공백.

### 괄호·기호 간격

- `( 4 )` → `(4)`
- `( 조특법 제32조 )` → `(조특법 제32조)`
- `½` → `1/2` (분수 통일)

## 7. 평가 케이스

### 케이스 1: 법인세 결산·신고 (16페이지, p.85~100)

- **프롬프트**: "이 PDF 교재의 p.85~100을 마크다운으로 변환해줘. 표와 구조를 잘 살려서."
- **기대 결과**: 법인세 결산 섹션이 깨끗한 마크다운으로 변환. 비교표가 markdown table로 재구성.
- **검증 기준**:
  - 마크다운 헤더 3개 이상 (`##`)
  - 테이블 구문 존재 (`|` 행)
  - 한국어 띄어쓰기 자연스러움
  - 최소 100줄 출력
  - OCR 아티팩트 없음

### 케이스 2: 조특법 세액감면 (21페이지, p.325~345)

- **프롬프트**: "이 PDF의 p.325~345를 마크다운으로 변환해줘. 업종 목록, 감면율 표 정리."
- **기대 결과**: 조특법 세액감면 섹션 구조화. 업종 분류 목록이 깔끔한 리스트, 감면율 비교가 표.
- **검증 기준**:
  - 마크다운 헤더 3개 이상
  - 테이블 구문 존재
  - 목록 형식 존재 (`-` 또는 숫자)
  - 최소 120줄 출력
  - `조특법`, `법인령` 등 법령 약칭 패턴 ≥1건 존재
