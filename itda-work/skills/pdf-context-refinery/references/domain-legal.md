# domain-legal.md — 법률·판례

> 도메인: 법률·판례 | 매칭 키워드: 판결·원고·피고·법령·민법·형법·소송

## 1. 수식·기호·단위 처리

법률 문서의 수치·비율 표현:

- **손해배상액**: `금 5,000,000원`, `손해액의 1/3`
- **기간**: `30일 이내`, `6개월`, `3년`
- **이율**: `연 5%`, `지연손해금 연 12%`
- **형량**: `징역 3년`, `벌금 1,000만 원`, `집행유예 2년`

분수 표현은 원문 그대로 보존: `2/3`, `1/2`.

verify_quality.py 검증 패턴: `제\d+조` 카운트 ≥3

## 2. 다중헤더 표·각주 처리

### 법령 조문 구조

조문 계층을 들여쓰기로 표현:

```markdown
**제7조 (중소기업 세액감면)**

① 중소기업이 다음 각 호의 사업을 경영하는 경우에는 …
1. 제조업
2. 광업
③ 제1항을 적용받으려는 자는 …
```

## 4. 이미지·다이어그램 감지 신호

다음 페이지는 이미지 모드로 처리한다:

## 5. 코드 블록·특수 콘텐츠

- **조문 본문**: `>` blockquote로 감싸기

  ```markdown
  > **민법 제750조 (불법행위의 내용)**
  > 고의 또는 과실로 인한 위법행위로 타인에게 손해를 가한 자는 그 손해를 배상할 책임이 있다.
  ```

- **판결 주문**: 볼드 + 들여쓰기

  ```markdown

  1. 피고는 원고에게 금 5,000,000원 및 이에 대하여 … 지급하라.
  2. 소송비용은 피고가 부담한다.
  ```

## 6. 도메인 OCR 아티팩트 패턴

### 판결문 페이지 헤더

판결문·법령집 전형적인 OCR 아티팩트 — 제거 대상:

```
제3조                    민법  47
판례 2023다12345         대법원  95
```

패턴: `^제\d+조` 또는 `^\d+\s+판례\s+\d+`

주의: `제\d+조` 패턴은 본문 조문 인용과 동일하므로, 줄 전체가 짧고 (`< 20자`) 페이지 번호가 뒤따르는 경우만 아티팩트로 처리한다.

### 괄호·기호 OCR 오인식

- `①②③` → 숫자 동그라미 보존 (제거 금지, 법령 항 번호)
- `가.나.다.` → 목록 마커로 처리
- `○○○` (이름 가림) → `○○○` 그대로 보존

## 7. 평가 케이스

### 케이스: 민법 조문 + 판례 (법률·판례 도메인)

- **PDF**: 민법 주석서 또는 판례집 (20페이지 내외)
- **프롬프트**: "이 PDF를 마크다운으로 변환해줘. 조문 계층과 판례 인용 형식을 살려서."
- **검증 기준**:
  - 마크다운 헤더 3개 이상
  - 조문 패턴 `제\d+조` ≥3건
  - 판례 인용 형식 보존
  - 조문 계층 (`제N조 제N항 제N호`) 정확 보존
  - OCR 아티팩트 제거

**verify 명령** (AC-11):
```bash
python3 scripts/verify_quality.py output/ --pages 20 --domain legal
```
