# domain-engineering.md — 일반 공학 (기계·토목)

> 도메인: 일반 공학 | 매칭 키워드: 응력·변형률·하중·열역학·유체·재료·KS

## 1. 수식·기호·단위 처리

### 공학 단위

| 단위 기호 | 물리량 | 예시 |
|----------|--------|------|
| MPa | 응력 (메가파스칼) | `250 MPa`, `0.5 MPa` |
| kN | 힘 (킬로뉴턴) | `150 kN`, `50 kN/m` |
| Pa | 압력 (파스칼) | `101325 Pa` |
| N/m² | 압력 (파스칼 동의) | `1000 N/m²` |
| m³ | 체적 | `2.5 m³` |
| m/s² | 가속도 | `9.81 m/s²` |
| kJ | 에너지 (킬로줄) | `100 kJ` |
| °C, K | 온도 | `25°C`, `298 K` |

verify_quality.py 검증 패턴: `\d+\s*(MPa|kN|Pa|N)\b|KS\s+[A-Z]\s+\d+`

### 공학 수식 표현

- 응력-변형률: `σ = E × ε` (σ: 응력, E: 탄성계수, ε: 변형률)
- 안전율: `SF = σ_ult / σ_allow`
- 단위 일관성 강제: `F [N]`, `A [m²]` 형태로 단위 명시

## 2. 다중헤더 표·각주 처리

### 재료 물성표

```markdown
| 재료 | 항복강도 (MPa) | 인장강도 (MPa) | 탄성계수 (GPa) |
|------|---------------|---------------|---------------|
| SS400 | 245 | 400~510 | 206 |
| SUS304 | 205 | 520 | 193 |
```

- 단위는 열 헤더에 괄호로 명시
- 범위 값은 `~` 또는 `to`로 표시
- 각주 `※ KS B 0801 기준`은 표 하단 blockquote로

### 시험 결과 표

반복 측정 결과는 평균±표준편차 형태: `125.3 ± 2.1 MPa`

## 3. 참조 번호 형식 보존

### KS 규격 번호

| 원문 | 마크다운 보존 |
|------|--------------|
| KS B 0801 | `KS B 0801` |
| KS F 2405 | `KS F 2405` |
| KS D 3503 | `KS D 3503` |

verify 패턴: `KS\s+[A-Z]\s+\d+`

### 도면 번호·참조

- `Figure 3-2`, `Fig. 3.2` → `Figure 3-2` 통일
- `Table 4.1`, `표 4.1` → 원문 그대로 보존
- `Eq. (3.5)`, `식 (3.5)` → 원문 그대로 보존
- ISO·ASTM 규격: `ASTM A36`, `ISO 6892-1` 정확 보존

## 4. 이미지·다이어그램 감지 신호

다음 페이지는 이미지 모드로 처리한다:

- **구조도·도면**: CAD 도면, 단면도, 평면도
- **그래프**: 응력-변형률 곡선, 하중-처짐 그래프, P-V 선도
- **장치 사진**: 시험 장치, 실험 셋업 사진
- **순서도**: 제조 공정, 시험 절차 흐름도

감지 기준: 텍스트 밀도 < 25%, 그리드·좌표계 패턴

## 5. 코드 블록·특수 콘텐츠

- **수치 계산 예시**: 들여쓰기 단계별 표시

  ```
  하중: P = 150 kN
  단면적: A = 0.06 m²
  응력: σ = P/A = 150,000 / 0.06 = 2,500,000 Pa = 2.5 MPa
  ```

- **규격 조항 인용**: blockquote

  ```markdown
  > **KS B 0801 (금속재료 인장시험)**
  > 시험편의 평행부 길이는 표점거리의 1.5배 이상으로 한다.
  ```

## 6. 도메인 OCR 아티팩트 패턴

### 영문·한국어 혼용 매뉴얼 페이지 헤더

공학 교재·매뉴얼 전형적인 OCR 아티팩트 — 제거 대상:

```
5 Chapter 3 42
12 응력해석 이론 95
```

패턴:
- 영문: `^\d+\s+Chapter\s+\d+`
- 한국어: `^\d+\s+[가-힣]+\s+\d+$`

### 단위·수식 OCR 오인식

| 원문 | OCR 오인식 | 정정 |
|------|----------|------|
| MPa | MPa, Mpa, МРа | MPa |
| kN/m² | kN/m2, kN/m^ | kN/m² |
| ε (엡실론) | e, E, ε | ε |
| σ (시그마) | o, 0, σ | σ |
| ≤ (이하) | <, =, <- | ≤ |

### 소수점·천단위 구분 오인식

- `2,500.0` (영문 표기) ↔ `2.500,0` (유럽 표기) — 문서 원문 기준 통일
- `3 . 14` → `3.14` (공백 제거)
