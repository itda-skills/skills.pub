## Sans-Serif

### Inter
- **License**: OFL 1.1
- **Weights**: Variable (100~900)
- **Download**: `https://fonts.google.com/download?family=Inter`
- **Features**: UI 최적화. Pretendard의 라틴 기반 폰트.
- **Best pairing**: Pretendard, SUIT
- **Best for**: UI, 보고서, PPT

### Roboto
- **License**: Apache 2.0
- **Weights**: 6 (Thin ~ Black) + Italic
- **Download**: `https://fonts.google.com/download?family=Roboto`
- **Features**: Google 디자인 시스템 표준.
- **Best pairing**: Noto Sans KR, 나눔고딕
- **Best for**: 보고서, 웹, UI

### Source Sans 3
- **License**: OFL 1.1
- **Weights**: Variable (200~900)
- **Download**: `https://fonts.google.com/download?family=Source+Sans+3`
- **Features**: Adobe 오픈소스. 가독성 우수.
- **Best pairing**: Noto Sans KR
- **Best for**: 기술 문서, 보고서

### Montserrat
- **License**: OFL 1.1
- **Weights**: Variable (100~900)
- **Download**: `https://fonts.google.com/download?family=Montserrat`
- **Features**: 기하학적 산세리프. 제목에 강력.
- **Best pairing**: Pretendard, 나눔스퀘어
- **Best for**: PPT 제목, 마케팅, 포스터

### Poppins
- **License**: OFL 1.1
- **Weights**: 9 (Thin ~ Black)
- **Download**: `https://fonts.google.com/download?family=Poppins`
- **Features**: 기하학적, 트렌디. 둥근 형태.
- **Best pairing**: Pretendard, Wanted Sans
- **Best for**: 마케팅, 스타트업, PPT

### Open Sans
- **License**: OFL 1.1
- **Weights**: Variable (300~800)
- **Download**: `https://fonts.google.com/download?family=Open+Sans`
- **Features**: 범용 가독성. 긴 본문에 적합.
- **Best pairing**: Noto Sans KR, 나눔고딕
- **Best for**: 보고서, 이메일, 웹

### Lato
- **License**: OFL 1.1
- **Weights**: 5 (Thin, Light, Regular, Bold, Black)
- **Download**: `https://fonts.google.com/download?family=Lato`
- **Features**: 따뜻하면서도 안정적. 본문/제목 겸용.
- **Best pairing**: Noto Sans KR, Spoqa Han Sans Neo
- **Best for**: 보고서, 웹, 프레젠테이션

## Serif

### Noto Serif
- **License**: OFL 1.1
- **Weights**: Variable (100~900)
- **Download**: `https://fonts.google.com/download?family=Noto+Serif`
- **Features**: Noto Serif KR과 같은 패밀리. 완벽 매칭.
- **Best pairing**: Noto Serif KR
- **Best for**: 학술, 출판, 공문서

### Source Serif 4
- **License**: OFL 1.1
- **Weights**: Variable (200~900)
- **Download**: `https://fonts.google.com/download?family=Source+Serif+4`
- **Features**: Adobe 오픈소스 세리프.
- **Best pairing**: Noto Serif KR, KoPubWorld 바탕체, 마루 부리
- **Best for**: 학술 보고서, 출판

### Lora
- **License**: OFL 1.1
- **Weights**: Variable (400~700)
- **Download**: `https://fonts.google.com/download?family=Lora`
- **Features**: 부드럽고 우아한 세리프. 본문 가독성 우수.
- **Best pairing**: 마루 부리, 나눔명조
- **Best for**: 문학, 에세이, 초대장

### Merriweather
- **License**: OFL 1.1
- **Weights**: 4 (Light, Regular, Bold, Black)
- **Download**: `https://fonts.google.com/download?family=Merriweather`
- **Features**: 화면 가독성에 최적화된 세리프.
- **Best pairing**: Noto Serif KR, 나눔명조
- **Best for**: 학술, 블로그, 긴 본문

### Playfair Display
- **License**: OFL 1.1
- **Weights**: Variable (400~900)
- **Download**: `https://fonts.google.com/download?family=Playfair+Display`
- **Features**: 고대비 디스플레이 세리프. 고급스러운 느낌.
- **Best pairing**: 마루 부리
- **Best for**: 고급 브랜딩, 초대장, 제목

## Monospace

### JetBrains Mono
- **License**: OFL 1.1
- **Weights**: 8 (Thin ~ ExtraBold) + Variable
- **Download**: `https://fonts.google.com/download?family=JetBrains+Mono`
- **Features**: 코드 가독성 최적화. 리가처 지원.
- **Best pairing**: D2Coding (한영 코드 혼용)
- **Best for**: 코드, 기술 문서

### Fira Code
- **License**: OFL 1.1
- **Weights**: 5 (Light ~ Bold)
- **Download**: `https://fonts.google.com/download?family=Fira+Code`
- **Features**: 프로그래밍 리가처 전문. Mozilla 기반.
- **Best pairing**: D2Coding
- **Best for**: 코드, 터미널

### IBM Plex Mono
- **License**: OFL 1.1
- **Weights**: 7 (Thin ~ Bold)
- **Download**: `https://fonts.google.com/download?family=IBM+Plex+Mono`
- **Features**: IBM 패밀리. IBM Plex Sans KR과 완벽 매칭.
- **Best pairing**: IBM Plex Sans KR
- **Best for**: 기술 문서 코드 블록

### Source Code Pro
- **License**: OFL 1.1
- **Weights**: Variable (200~900)
- **Download**: `https://fonts.google.com/download?family=Source+Code+Pro`
- **Features**: Adobe 오픈소스 모노스페이스.
- **Best pairing**: Source Sans 3 + Noto Sans KR
- **Best for**: 코드, 기술 문서
