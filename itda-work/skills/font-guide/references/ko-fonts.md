## Gothic (Sans-Serif)

### Pretendard
- **License**: OFL 1.1
- **Weights**: 9 (Thin 100 ~ Black 900) + Variable
- **Source**: https://github.com/orioncactus/pretendard/releases/latest
- **Install**: `brew install font-pretendard`
- **Features**: Inter 기반 라틴 + Source Han Sans 한글. 한영 조화 최고. 현재 한국에서 가장 인기 있는 무료 고딕.
- **Best for**: PPT, UI/UX, 보고서, 마케팅, 웹
- **English built-in**: Yes (Inter 기반)

### Noto Sans KR (본고딕)
- **License**: OFL 1.1
- **Weights**: 9 (Thin 100 ~ Black 900) + Variable
- **Source**: https://fonts.google.com/noto/specimen/Noto+Sans+KR
- **Download**: `https://fonts.google.com/download?family=Noto+Sans+KR`
- **Features**: Google/Adobe 공동 개발. 현대한글 11,172자 완전 지원. 글로벌 표준.
- **Best for**: 공문서, 학술논문, 보고서, 모든 범용
- **English built-in**: Yes

### 나눔고딕 (NanumGothic)
- **License**: OFL 1.1
- **Weights**: 3 (Regular, Bold, ExtraBold)
- **Source**: https://hangeul.naver.com/font
- **Download**: `https://fonts.google.com/download?family=Nanum+Gothic`
- **Features**: 네이버 제작. 2008년 공개. 한국에서 가장 오래된 무료 한글 폰트 중 하나.
- **Best for**: 일반 문서, 웹, 이메일
- **English built-in**: Yes

### 나눔바른고딕 (NanumBarunGothic)
- **License**: OFL 1.1
- **Weights**: 4 (UltraLight, Light, Regular, Bold)
- **Source**: https://hangeul.naver.com/font
- **Features**: 나눔고딕 대비 더 깔끔하고 현대적. 폰트릭스 제작.
- **Best for**: UI, 모바일, 웹, 깔끔한 보고서
- **English built-in**: Yes

### 나눔스퀘어 (NanumSquare)
- **License**: OFL 1.1
- **Weights**: 4 (Light, Regular, Bold, ExtraBold)
- **Source**: https://hangeul.naver.com/font
- **Features**: 네모난 형태의 제목용 고딕. PPT 제목에 매우 인기.
- **Best for**: PPT 제목, 포스터, 배너, 마케팅
- **English built-in**: Yes

### 나눔스퀘어 네오 (NanumSquare Neo)
- **License**: OFL 1.1
- **Weights**: 6 (Light ~ Heavy)
- **Source**: https://hangeul.naver.com/font
- **Features**: 나눔스퀘어 업그레이드. 더 많은 굵기, 개선된 글리프.
- **Best for**: PPT, 보고서, 마케팅, UI
- **English built-in**: Yes

### Spoqa Han Sans Neo
- **License**: OFL 1.1
- **Weights**: 4 (Thin, Regular, Medium, Bold)
- **Source**: https://github.com/spoqa/spoqa-han-sans/releases
- **Features**: 한/영/일 조화 우수. 디지털 환경 최적화.
- **Best for**: UI/UX, 웹서비스, 보고서
- **English built-in**: Yes

### IBM Plex Sans KR
- **License**: OFL 1.1
- **Weights**: 7 (Thin 100 ~ Bold 700)
- **Source**: https://github.com/IBM/plex/releases
- **Download**: `https://fonts.google.com/download?family=IBM+Plex+Sans+KR`
- **Features**: IBM 브랜드 한국어 확장. 기업용 느낌.
- **Best for**: 기업 보고서, UI, 기술 문서
- **English built-in**: Yes (IBM Plex Sans 매칭)

### Wanted Sans
- **License**: OFL 1.1
- **Weights**: 7 + Variable
- **Source**: https://github.com/wanteddev/wanted-sans/releases/latest
- **Features**: 기하학적 산세리프. 직선적 인상과 가독성 균형. OpenType 대체 기능.
- **Best for**: PPT, 마케팅, 스타트업 브랜딩, UI
- **English built-in**: Yes

### SUIT
- **License**: OFL 1.1
- **Weights**: 9 (Thin ~ Black) + Variable
- **Source**: https://github.com/sunn-us/SUIT/releases
- **Features**: 기하학적 산세리프 UI 폰트. 본문용 최적화.
- **Best for**: UI/UX, 본문, 웹
- **English built-in**: Yes

### LINE Seed Sans KR
- **License**: OFL 1.1
- **Weights**: 3 (Thin, Regular, Bold)
- **Source**: https://github.com/line/seed
- **Features**: LINE 브랜드 서체. 기하학 기반. 친근한 느낌.
- **Best for**: UI, 마케팅, 친근한 느낌의 문서
- **English built-in**: Yes

## Serif (명조)

### Noto Serif KR (본명조)
- **License**: OFL 1.1
- **Weights**: 7 (ExtraLight ~ Black)
- **Source**: https://fonts.google.com/specimen/Noto+Serif+KR
- **Download**: `https://fonts.google.com/download?family=Noto+Serif+KR`
- **Features**: Google/Adobe 공동 개발. 가장 완성도 높은 무료 한글 명조.
- **Best for**: 계약서, 학술논문, 공문서, 문학, 격식 문서
- **English built-in**: Yes

### 마루 부리 (MaruBuri)
- **License**: OFL 1.1
- **Weights**: 5 (ExtraLight ~ Bold)
- **Source**: https://hangeul.naver.com/font
- **Features**: 2021 한글날 공개. 디지털 매체 최적화 명조. 따뜻한 느낌.
- **Best for**: 브랜딩, 문학/출판, 격식 PPT, 초대장
- **English built-in**: Yes

### 나눔명조 (NanumMyeongjo)
- **License**: OFL 1.1
- **Weights**: 3 (Regular, Bold, ExtraBold)
- **Source**: https://hangeul.naver.com/font
- **Download**: `https://fonts.google.com/download?family=Nanum+Myeongjo`
- **Features**: 네이버 제작. 전통적 명조 스타일.
- **Best for**: 공문서, 계약서, 학술논문, 격식 문서
- **English built-in**: Yes

### KoPubWorld 바탕체/돋움체
- **License**: 무료 (수정/변형/유료판매 불가, 웹 임베딩 별도 승인)
- **Weights**: 3 (Light, Medium, Bold) — 바탕/돋움 각각
- **Source**: https://www.kopus.org/biz-electronic-font2/
- **Features**: 한국출판인회의/문화체육관광부 제작. 현대한글 11,172자 + 옛한글 9,170자.
- **Best for**: 출판, 전자책, 공문서, 학술, 교과서
- **English built-in**: Yes

## Coding / Monospace

### D2Coding
- **License**: OFL 1.1
- **Weights**: 2 (Regular, Bold) + Ligature 변형
- **Source**: https://github.com/naver/d2codingfont
- **Features**: 나눔바른고딕 기반 코딩 전용. 한글 코딩에 최적.
- **Best for**: 코드 에디터, 터미널, 기술문서 코드 블록
- **English built-in**: Yes

### 나눔고딕코딩 (NanumGothicCoding)
- **License**: OFL 1.1
- **Weights**: 2 (Regular, Bold)
- **Source**: https://github.com/naver/nanumfont
- **Download**: `https://fonts.google.com/download?family=Nanum+Gothic+Coding`
- **Features**: 고정폭 코딩 폰트. 네이버 제작.
- **Best for**: 코드 에디터, 터미널
- **English built-in**: Yes

## System Fonts (참고용)

OS 번들 폰트는 해당 OS에서 별도 설치 없이 사용 가능. 재배포 불가.

| Font | OS | Weights | Type | Notes |
|---|---|---|---|---|
| 맑은 고딕 (Malgun Gothic) | Windows | Regular, Bold, Semilight | Gothic | 한국 공문서/기업 사실상 표준 |
| Apple SD Gothic Neo | macOS/iOS | 9 weights | Gothic | macOS 시스템 한글 폰트 |
| 바탕 / 바탕체 | Windows | Regular | Serif | 전통 명조 |
| 굴림 / 굴림체 | Windows | Regular | Gothic (rounded) | 레거시 호환 |
| 돋움 / 돋움체 | Windows | Regular | Gothic | 레거시 호환 |
