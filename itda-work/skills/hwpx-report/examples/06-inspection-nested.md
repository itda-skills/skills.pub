# 노후 사회기반시설 안전점검 결과

## 1. 점검 개요

- 점검 대상: 준공 30년 경과 교량·터널·옹벽
  - 1종 시설물 전수 점검
    - 외관 조사 및 비파괴 시험 병행
      - 균열·누수·철근 노출 중점 확인
- 점검 기간: 상반기 정기점검 기간 내 완료

## 2. 점검 결과

- 전체 시설 중 보수·보강 필요 등급 일부 발생
  - 교량 신축이음 손상이 다수

## 3. 조치 계획

- 긴급 보수 필요 시설 즉시 사용 제한 검토
- 중장기 성능개선 계획 반영
