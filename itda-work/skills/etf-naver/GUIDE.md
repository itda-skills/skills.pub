title: "etf-naver 활용 가이드"

네이버 금융 공개 데이터로 국내 ETF 시세·기술적 분석·섹터 비교를 제공하는 스킬입니다. **Claude에게 자연어로 말하면 됩니다** — 터미널 명령을 직접 입력할 필요는 없습니다.

---

## 빠른 시작

```
069500 기술적 분석해줘
```

```
KODEX 200 RSI랑 MACD 보여줘
```

```
해외 ETF 상위 10개 괴리율 보여줘
```

---

## 활용 시나리오

### 후보 스크리닝 → 개별 상세 분석

분류별 상위 종목을 먼저 추린 뒤, 관심 종목만 자세히 들여다보는 2단계 흐름이 토큰 절약에 유리합니다.

```
해외 주식 ETF 시가총액 상위 10개 괴리율 포함해서 보여줘
```

```
TIGER 미국나스닥100 (360750) 전체 지표랑 판단 신호 분석해줘
```

### 섹터 로테이션 → 개별 종목 검증

분류별 평균 등락률·자금 흐름(유입/유출)을 한 표로 요약합니다. 유입 섹터를 확인한 다음 그 섹터의 대표 종목을 상세 분석으로 넘어가는 것이 일반적입니다.

```
국내 주식·해외 주식·국내 채권·해외 채권 섹터별 상위 3개씩 비교해줘
```

```
KODEX 200 (069500) 이동평균선이랑 MACD 분석해줘
```

### 포트폴리오 리밸런싱 시뮬레이션

보유 중인 ETF의 목표 비중을 자연어로 알려주면 목표 비중 대비 편차와 매수/매도 액션을 계산합니다.

```
KODEX 200 30%, TIGER 미국나스닥100 40%, KODEX 인버스 30% 목표 비중으로 리밸런싱 계산해줘
```

## 팁

- **조회 기간은 기본값을 유지하세요**: 365일 미만으로 줄이면 52주 고저 정확도가 떨어집니다. 실측 기준 200일로 조회하면 KODEX 200의 52주 최저가가 47,468원으로 나오지만, 기본값 365에서는 32,658원이 나옵니다. 단기 추세만 볼 때도 1년치 데이터를 기준으로 하세요.
- **지표 지정은 결정적 근거가 필요할 때**: 선택하지 않은 지표는 분석 결과에서 비워지지 않고 그대로 나옵니다. "RSI와 MACD만", "이동평균선과 볼린저밴드"처럼 지표를 조합으로 요청하면 집중해서 판단하기 좋습니다.
- **스크리닝 → 상세 분석 2단계로 토큰을 아낍니다**: 전체 목록에서 상위 10개만 추린 뒤 관심 종목만 상세 분석하는 흐름이 대화 누적 토큰을 크게 줄입니다.
- **리밸런싱 결과의 "현재비중"은 추정값**: 네이버 API의 시가총액 기준으로 계산된 값이라 실제 보유 수량·금액 비중과 다를 수 있습니다. 편차 부호와 상대 크기를 참고 지표로만 사용하세요.

## 제한사항

- ⚠️ **투자 조언이 아닙니다.** 이 스킬의 데이터와 신호는 참고 자료이며, 실제 투자 결정은 본인 판단과 책임으로 내려야 합니다.
- ⚠️ **해외 직상장 ETF는 지원하지 않습니다.** SPY·QQQ 같은 미국 직상장 ETF는 국내 6자리 종목코드가 없으므로 조회 대상이 아닙니다. 국내 상장된 복제형 ETF(해외 주식 분류)는 조회할 수 있습니다.
- ⚠️ **네이버 금융 비공식 API에 의존합니다.** `finance.naver.com` 응답 포맷이 바뀌면 일부 기능이 동작하지 않을 수 있습니다.
- ⚠️ **실시간 tick 단위 데이터가 아닙니다.** 응답은 장 마감가/최신 캐시 기준으로, 장중 호출 시 실시간가 대비 지연이 있을 수 있습니다.
- ⚠️ **지원 범위 밖 요청.** 분봉/틱 데이터, 경제 지표, 펀드 운용보수·순자산·구성종목, 월간 리밸런싱 자동화, 증권사 주문 실행은 제공하지 않습니다.
