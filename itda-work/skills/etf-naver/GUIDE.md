## 빠른 시작

네이버 금융 공개 데이터로 국내 ETF 시세·기술적 분석·섹터 비교를 수행합니다. 대화창에서 자연어로 바로 호출할 수 있습니다.

```
069500 기술적 분석해줘
```

```
KODEX 200 RSI랑 MACD 보여줘
```

```
해외 ETF 상위 10개 괴리율 보여줘
```

## 활용 시나리오

### 후보 스크리닝 → 개별 상세 분석

`fetch_etf.py`로 분류별 상위 종목을 먼저 추린 뒤, 관심 종목만 `fetch_etf_detail.py`로 자세히 들여다보는 2단계 흐름이 토큰 절약에 유리합니다.

```bash
# 1) 해외 주식 ETF 시가총액 상위 10개 + 괴리율
python3 scripts/fetch_etf.py --type 4 --top 10 --premium

# 2) 관심 종목 하나만 골라 전체 지표 + 판단 신호
python3 scripts/fetch_etf_detail.py --code 360750 --indicators all
```

### 섹터 로테이션 → 개별 종목 검증

`compare_etf.py --sectors`는 분류 코드별 평균 등락률·자금 흐름(유입/유출)을 한 표로 요약합니다. 유입 섹터를 확인한 다음 그 섹터의 대표 종목을 상세 분석으로 넘어가는 것이 일반적입니다.

```bash
python3 scripts/compare_etf.py --sectors 1,2,4,5 --top 3
python3 scripts/fetch_etf_detail.py --code 069500 --indicators ma,macd
```

### 포트폴리오 리밸런싱 시뮬레이션

보유 중인 ETF의 목표 비중을 `CODE:TARGET%` 형식으로 넘기면 목표 비중 대비 편차와 매수/매도 액션을 계산합니다.

```bash
python3 scripts/compare_etf.py --portfolio "069500:30,360750:40,148070:30"
```

## 출력 옵션

`--format` 플래그로 형식을 전환할 수 있습니다. 세 형식은 사용 맥락이 다릅니다.

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| `table` (기본) | Claude 대화창에 마크다운 표로 바로 출력 | 대화 내 즉시 해석·판단 요청 |
| `json` | 구조화된 필드 배열 | 외부 대시보드·노트북으로 데이터 적재 |
| `csv` | 쉼표 구분 레코드 | 엑셀·구글 시트에 붙여넣기, 주간 보고 정리 |

## 팁

- **`--days`는 기본값 365를 유지하세요.** 실측 기준 `--days 200`으로 호출하면 KODEX 200의 52주 최저가가 47,468원으로 나오지만, 기본값 365에서는 32,658원이 나와 기간 설정이 52주 고저 정확도에 직접 영향을 줍니다. 단기 추세만 볼 때도 365 이상을 권장합니다.
- **`--indicators`는 출력 길이를 줄이지 않습니다.** 선택하지 않은 지표는 `N/A`로 채워진 채 표가 그대로 나옵니다. 가볍게 보고 싶으면 `all`로 받아 필요한 행만 읽는 쪽이 해석 비용이 더 낮습니다. 반대로 결정적 근거를 명확히 하고 싶을 때 `ma,rsi` / `macd,bb`처럼 조합으로 호출합니다.
- **스크리닝 → 상세 분석 2단계로 토큰을 아낍니다.** 30개 목록을 바로 자세히 보기보다는 `fetch_etf.py --top 10`으로 후보를 좁힌 뒤 `fetch_etf_detail.py`로 넘어가는 흐름이 대화 누적 토큰을 크게 줄입니다.
- **리밸런싱 결과의 "현재비중"은 추정값입니다.** `--portfolio` 출력 상단에 고정 경고가 붙어 있듯이, 이 값은 네이버 API의 시가총액 기준으로 계산된 값이라 실제 보유 수량·금액 비중과 다릅니다. 편차 부호와 상대 크기를 참고 지표로만 사용하세요.

## 제한사항

- ⚠️ **투자 조언이 아닙니다.** 이 스킬의 데이터와 신호는 참고 자료이며, 실제 투자 결정은 본인 판단과 책임으로 내려야 합니다.
- ⚠️ **해외 직상장 ETF는 지원하지 않습니다.** SPY·QQQ 같은 미국 직상장 ETF는 국내 6자리 종목코드가 없으므로 호출 대상이 아닙니다. 국내 상장된 복제형 ETF는 `--type 4` 해외 주식 분류로 조회합니다.
- ⚠️ **네이버 금융 비공식 API에 의존합니다.** `finance.naver.com` 응답 포맷이 바뀌면 스크립트가 깨질 수 있습니다.
- ⚠️ **실시간 tick 단위 데이터가 아닙니다.** 응답은 장 마감가/최신 캐시 기준으로, 장중 호출 시 실시간가 대비 지연이 있을 수 있습니다.
- ⚠️ **지원 범위 밖 요청.** 분봉/틱 데이터, 경제 지표, 펀드 운용보수·순자산·구성종목, 월간 리밸런싱 자동화, 증권사 주문 실행은 제공하지 않습니다.
