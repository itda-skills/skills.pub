# ETF 분석 판단 가이드라인

Claude가 ETF 데이터를 받은 후 해석/판단하는 기준입니다.

## 괴리율 (Premium/Discount Rate)

괴리율 = ((시장가 - NAV) / NAV) × 100

| 괴리율 범위 | 신호 | 해석 |
|------------|------|------|
| \|괴리율\| < 0.5% | 정상 | 시장가와 NAV가 정상 범위 내 |
| 0.5% ≤ \|괴리율\| < 1.0% | 주의 | 모니터링 필요, 일시적 수급 불균형 가능 |
| \|괴리율\| ≥ 1.0% | 경고 | 차익거래 기회 가능, 유동성·시장 상황 확인 |

참고:
- 양수 괴리율: 시장가가 NAV보다 높음 (프리미엄)
- 음수 괴리율: 시장가가 NAV보다 낮음 (디스카운트)
- 해외 ETF, 원자재 ETF는 시차로 인해 괴리율이 높을 수 있음

## RSI (Relative Strength Index, 14일)

| RSI 범위 | 신호 | 해석 |
|----------|------|------|
| RSI > 70 | 과매수 | 단기 조정 가능, 신규 매수 주의 |
| 50 < RSI ≤ 70 | 강세 | 상승 모멘텀 유지 중 |
| 30 ≤ RSI ≤ 50 | 약세 | 하락 모멘텀, 관망 |
| RSI < 30 | 과매도 | 반등 가능, 분할매수 고려 |

## 이동평균선 (SMA)

| 조건 | 신호 | 해석 |
|------|------|------|
| 현재가 > SMA20 > SMA60 | 상승추세 | 추세 추종 매수 유효 |
| 현재가 < SMA20 < SMA60 | 하락추세 | 매수 보류, 손절 고려 |
| SMA20이 SMA60 상향 돌파 | 골든크로스 | 중기 매수 신호 |
| SMA20이 SMA60 하향 돌파 | 데드크로스 | 중기 매도 신호 |
| 현재가가 SMA20 근처에서 지지 | 지지 확인 | 추세 지속 가능성 |

## MACD (12, 26, 9)

| 조건 | 신호 | 해석 |
|------|------|------|
| MACD > Signal | 매수 모멘텀 | 상승 추세 강화 중 |
| MACD < Signal | 매도 모멘텀 | 하락 추세 강화 중 |
| MACD 0선 상향 돌파 | 강한 매수 | 추세 전환 확인 |
| MACD 0선 하향 돌파 | 강한 매도 | 추세 전환 확인 |
| MACD 히스토그램 축소 | 모멘텀 약화 | 추세 전환 임박 가능 |

## 볼린저밴드 (20일, 2σ)

| 조건 | 신호 | 해석 |
|------|------|------|
| 현재가 > 상단밴드 | 과매수 | 강한 상승 모멘텀이나 되돌림 주의 |
| 현재가 < 하단밴드 | 과매도 | 강한 하락 모멘텀이나 반등 가능 |
| 밴드폭 수축 (스퀴즈) | 변동성 감소 | 큰 움직임 임박, 방향 주시 |
| 밴드폭 확대 | 변동성 증가 | 추세 진행 중 |

## ATR (Average True Range, 14일)

| 활용 | 기준 |
|------|------|
| 손절폭 설정 | 현재가 - (ATR × 2) |
| 변동성 비교 | ATR / 현재가 × 100 = 변동성(%) |
| 포지션 크기 | 리스크금액 / ATR = 적정 수량 |

## 52주 고저 대비 현재 위치

| 위치 | 해석 |
|------|------|
| 현재가 ≥ 52주 고점 95% | 고점 근접, 돌파 or 저항 |
| 현재가 ≤ 52주 저점 105% | 저점 근접, 지지 or 추가 하락 |
| 중간 (30-70% 구간) | 추세 방향에 따라 판단 |

## 섹터 로테이션 판단

| 지표 | 유입 신호 | 유출 신호 |
|------|----------|----------|
| 평균등락률 | 양수, 특히 시장 평균 초과 시 | 음수, 특히 시장 평균 미달 시 |
| 총거래량 | 최근 평균 대비 증가 | 최근 평균 대비 감소 |
| 시가총액 변화 | 증가 추세 | 감소 추세 |

## 포트폴리오 리밸런싱 기준

| 편차 | 액션 |
|------|------|
| \|편차\| < 3% | 유지 (거래비용 > 리밸런싱 효과) |
| 3% ≤ \|편차\| < 5% | 모니터링 (다음 매수 시 조정) |
| \|편차\| ≥ 5% | 리밸런싱 필요 (적극적 비중 조정) |

## 종합 판단 프레임워크

Claude는 데이터를 받은 후 다음 순서로 종합 판단을 제시합니다:

1. **추세 판단**: 이평선 배열 + MACD 방향
2. **모멘텀 판단**: RSI 구간 + MACD 히스토그램
3. **변동성 판단**: 볼린저밴드 위치 + ATR 수준
4. **밸류에이션**: 괴리율 + 52주 고저 위치
5. **액션 제안**: 위 4가지 종합 → 매수/매도/관망 + 구체적 조건
