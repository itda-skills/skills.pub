---
title: "Exchange Rate (매매기준율) 활용 가이드"
---

원화 기준 매매기준율을 일별·월평균 단위로 조회하는 스킬입니다. **Claude에게 자연어로 말하면 됩니다** — 날짜와 통화를 자동으로 파악하고, 휴일에는 직전 영업일 환율로 자동 폴백합니다.

---

## 빠른 시작

```
오늘 달러 환율 알려줘
```

```
이번 달 엔화 평균 환율 보여줘
```

```
이번 달과 지난달 달러 환율 비교해줘
```

---

## 활용 시나리오

### 특정 일자 환율 조회

```
2025년 1월 5일 달러 환율 알려줘
```

날짜는 `2025-01-05`, `2025.01.05` 형태 모두 그대로 붙여넣어도 됩니다.

### 한 달 일별 환율 전체 조회

월 단위로 일별 매매기준율을 한 번에 조회할 때 가장 자주 쓰는 패턴입니다.

```
2025년 1월 한 달간 달러 환율을 일별로 보여줘
```

### 월간 비교 분석

서로 다른 월·통화의 평균을 비교해 환율 추이를 파악합니다.

```
이번 달 엔화 평균과 지난달 엔화 평균을 비교해줘
```

### 휴일 자동 폴백

주말·공휴일 날짜로 조회해도 자동으로 직전 영업일 매매기준율이 반환됩니다.

```
이번 주 토요일 달러 환율 알려줘
```

내부적으로 토요일 → 금요일(직전 영업일) 값으로 폴백합니다.

---

## 팁

- **휴일 폴백 트릭**: 주말·공휴일 날짜로 조회하면 직전 영업일 값이 돌아옵니다. "가장 최근 영업일 환율"을 빠르게 확인할 때 유용합니다.
- **날짜 포맷 유연성**: 점 표기(`2025.01.05`)와 하이픈 표기(`2025-01-05`) 둘 다 허용됩니다. 한국 문서에서 흔한 점 표기를 그대로 붙여넣어도 됩니다.
- **한국어 통화 별칭**: "달러", "엔", "유로" 같은 한국어 이름도 자동으로 ISO 코드(USD, JPY, EUR)로 매핑됩니다.
- ⚠️ **조회 범위 제한**: 데이터 소스가 지원하는 기간·통화 안에서만 조회됩니다. 수십 년 전 장기 과거 데이터나 마이너 통화는 누락될 수 있습니다.
