title: "Exchange Rate (매매기준율) 활용 가이드"
---

## 빠른 시작

원화 기준 매매기준율을 일별·월평균 단위로 조회하고 비교할 수 있습니다.

```
이번 달과 지난달 달러 환율 비교해줘
```

```
2025년 1월 엔화 일별 환율 보여줘
```

```
오늘 유로 환율 알려줘
```

자연어 요청만으로 스킬이 날짜·통화를 자동 파싱하고, 휴일에는 직전 영업일 값으로 폴백합니다.

## 활용 시나리오

### 특정 월의 일별 환율 + 통화 지정

월 단위로 일별 매매기준율을 한 번에 조회할 때 가장 자주 쓰는 패턴입니다.

```
2025년 1월 한 달간 달러 환율을 일별로 보여줘
```

CLI 예시:

```bash
python3 scripts/exchange_rate.py --month 2025-01 --currency USD
```

### 월간 비교 분석

서로 다른 월·통화의 평균을 비교해 환율 추이를 파악합니다.

```
이번 달 엔화 평균과 지난달 엔화 평균을 비교해줘
```

CLI 예시:

```bash
python3 scripts/exchange_rate.py --month 2025-03 --currency JPY
python3 scripts/exchange_rate.py --month 2025-02 --currency JPY
```

### 휴일 자동 폴백

주말·공휴일 날짜로 조회해도 자동으로 직전 영업일 매매기준율이 반환됩니다.

```
이번 주 토요일 달러 환율 알려줘
```

내부적으로는 토요일 → 금요일(직전 영업일) 값으로 폴백합니다.

## 출력 옵션

| 모드 | 설명 | 사용 시점 |
|------|------|-----------|
| 일별 (`--date`) | 특정 일자 매매기준율 1건 | 결산·송금 등 단일 시점 조회 |
| 월평균 (`--month`) | 한 달 일별 환율 + 평균값 | 회계 마감, 추이 분석, 보고서 |
| 통화 지정 (`--currency`) | `USD`/`JPY`/`EUR` 등 ISO 코드 또는 한국어 별칭(`달러`, `엔`, `유로`) | 특정 통화만 필요한 경우 |
| JSON 출력 | 다른 스크립트·도구가 파싱하기 좋은 구조화 데이터 | 파이프라인·자동화 연동 |

## 팁

- **휴일 폴백 트릭**: 주말·공휴일 날짜로 조회하면 직전 영업일 값이 돌아옵니다. "가장 최근 영업일 환율"을 빠르게 확인할 때 유용합니다.
- **날짜 포맷 유연성**: `YYYY-MM-DD`(`2025-01-05`)와 `YYYY.MM.DD`(`2025.01.05`) 둘 다 허용됩니다. 한국 문서에서 흔한 점 표기를 그대로 붙여넣어도 됩니다.
- **한국어 통화 별칭**: `--currency 달러`, `--currency 엔`, `--currency 유로` 같은 입력도 자동으로 ISO 코드로 매핑됩니다.
- **JSON 파이프라인**: JSON 출력을 `jq` 등으로 파이프해 다른 스크립트의 입력으로 넘길 수 있습니다.
- ⚠️ **조회 범위 제한**: 데이터 소스가 지원하는 기간·통화 안에서만 조회됩니다. 수십 년 전 장기 과거 데이터나 마이너 통화는 누락될 수 있습니다.
