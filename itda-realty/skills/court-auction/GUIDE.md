# court-auction 사용 가이드

대법원 **법원경매정보**(courtauction.go.kr)에 올라온 부동산 경매 매각공고와 사건·물건을
대화로 조회해 줍니다. "오늘 어디서 경매가 열리는지", "특정 사건이 어떻게 진행 중인지",
"조건에 맞는 경매 물건이 있는지"를 사이트를 직접 열지 않고 물어볼 수 있어요.

## 어떻게 말하면 되나요

Claude에게 자연스럽게 말하면 됩니다.

- "오늘 서울중앙지방법원에서 열리는 경매 공고 보여줘"
- "이번 달 인천 경매 매각공고 알려줘"
- "사건번호 2024타경100001 진행 상황 알려줘"
- "강남구 아파트 중에 최저가 5억 이하, 한 번 이상 유찰된 물건 찾아줘"
- "법원 코드표 보여줘" (어느 법원인지 코드를 모를 때)

어느 법원인지 모르면 먼저 법원 목록을 보여주거나, 어느 지역 법원인지 되물어요.

## 무엇을 찾을 수 있나요

- **매각공고 목록** — 특정 월·법원·입찰구분(기일입찰/기간입찰)으로 그날 나온 공고를 정리합니다.
- **공고 펼치기** — 고른 공고 안의 사건번호·용도·주소·감정평가액·최저매각가를 보여줍니다.
- **사건 직접 조회** — 사건번호로 사건명·접수일·청구액·재판부·진행상태와 기일별 결과를 봅니다.
- **물건 자유검색** — 지역·용도(토지·건물·아파트 등)·가격·면적·유찰횟수 조건으로 물건을 찾습니다.

## 조건으로 좁히기

물건을 찾을 때 이런 조건을 덧붙이면 그에 맞는 것만 추려 줍니다.

- 지역 — "서울 강남구", "부산"
- 용도 — "아파트", "건물", "토지"
- 가격 — "최저가 5억 이하", "감정가 3억에서 8억 사이"
- 유찰 — "한 번 이상 유찰된 것"(가격이 떨어진 물건)
- 면적 — "84㎡ 이상"

## 결과에 담기는 정보

매각공고는 법원·매각기일·장소·입찰구분을, 물건은 사건번호·용도·주소·감정평가액·
최저매각가·유찰횟수를 함께 줍니다. 금액은 천단위와 억/만 단위로 풀어서 보여드려요.

## 알아두세요

- 이 도구는 **법원경매정보 사이트의 공개 정보를 그대로 옮긴 참고용**입니다. 입찰을
  생각한다면 **반드시 법원의 원문 매각공고를 다시 확인**하세요.
- 가격·매각기일·장소는 **공고 시점 기준**이며, 정정·취하·연기로 바뀔 수 있습니다.
- 사이트가 공식 제공 방식이 아닌 내부 검색을 쓰기 때문에, 사이트 정책 변경으로 동작이
  멈출 수 있습니다. 또 **짧은 시간에 여러 번 조회하면 사이트가 약 1시간 동안 접속을
  막을 수 있어**, 천천히·꼭 필요한 만큼만 조회합니다.
- **읽기 전용**입니다. 입찰서 작성·제출이나 결제는 하지 않습니다(입찰은 법원에서 직접 하세요).
- 동산(자동차·중기) 경매, 매각물건명세서·감정평가서 같은 첨부 문서, 물건 사진은 아직 다루지 않습니다.
