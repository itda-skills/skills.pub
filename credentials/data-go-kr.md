# 공공데이터포털 (data.go.kr) — 가입·키 발급 가이드

- 공식 사이트: <https://www.data.go.kr>
- 발급 키: `KO_DATA_API_KEY` (계정당 1개의 공용 인증키)
- Last Verified: 2026-06-10 (itda-gov 스킬 GUIDE들에서 통합 — 실화면 재검증 시 갱신)

## 1. 가입 조건

- **일반 회원가입**으로 충분합니다 (비용 없음).
- 인증키는 계정당 1개가 발급되며, **하나의 키로 모든 데이터셋을 호출**합니다.
- 단, **데이터셋(오픈API)마다 별도 "활용신청"이 필요**합니다 — 대부분 자동승인이라 신청 즉시 5~30분 내 사용 가능합니다.

## 2. 발급 절차

1. [www.data.go.kr](https://www.data.go.kr) 접속 → 회원가입 후 로그인
2. 쓰려는 스킬이 요구하는 **데이터셋을 각각 활용신청** — 데이터셋별 신청 링크는 **각 스킬의 GUIDE**에 안내돼 있습니다 (예: 부동산 실거래는 아파트 매매·전월세 등 4건)
3. **마이페이지 → 오픈API → 활용신청 현황**에서 발급 키 확인
4. **Decoding 키(일반 인증키)**를 복사합니다

## 3. 키 ↔ 환경변수 매핑

| 발급 화면 명칭 | 환경변수 |
|---|---|
| 일반 인증키 (**Decoding**) | `KO_DATA_API_KEY` |

```dotenv
KO_DATA_API_KEY=발급받은_Decoding_키
```

## 4. 한도·주의사항

- **Encoding 키가 아니라 Decoding 키**를 쓰세요. 마이페이지에 두 가지가 함께 표시되는데, `%`가 섞여 있는 쪽이 Encoding 키입니다 — 그쪽을 넣으면 인증 오류가 납니다.
- **활용신청을 안 한 데이터셋을 조회하면 권한 오류**가 납니다. 키 문제가 아니라 해당 데이터셋 미신청이 원인인 경우가 흔합니다 — 스킬이 오류 메시지에 신청 링크를 표시해 줍니다.
- 신청 직후에는 반영까지 5~30분(드물게 1시간) 걸릴 수 있습니다.
- 일일 트래픽 한도(데이터셋별 보통 1,000~10,000회)가 있으니 동일 조건 반복 조회는 피하세요.
- 운영계정 전환 전 개발계정 한도로도 개인 사용에는 충분합니다.
