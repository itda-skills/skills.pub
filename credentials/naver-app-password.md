# 네이버 앱 비밀번호 — 발급 가이드

- 공식 사이트: <https://nid.naver.com> (네이버 내 정보·보안 설정)
- 발급 키: `NAVER_EMAIL` + `NAVER_APP_PASSWORD`
- Last Verified: 2026-06-10 (email·calendar 운영 GUIDE에서 통합 — 실화면 재검증 시 갱신)

## 1. 가입 조건

- 일반 네이버 계정이면 됩니다 (비용 없음).
- 단, **2단계 인증이 켜져 있어야** 앱 비밀번호를 발급할 수 있습니다.
- 보안상 계정 비밀번호를 그대로 쓰지 않고, 메일·캘린더 같은 외부 연결 전용의 **앱 비밀번호**를 따로 발급받는 방식입니다.

## 2. 발급 절차

1. 네이버 로그인 → **내 정보 → 보안 설정 → 2단계 인증**을 켭니다
2. 같은 보안 설정의 **애플리케이션 비밀번호 관리 → 발급**
3. 발급된 비밀번호를 **그 자리에서 복사**합니다 — 한 번만 표시됩니다
4. (메일 스킬 사용 시) 네이버 메일 → 설정(톱니바퀴) → **POP3/IMAP 설정**에서 IMAP을 **사용함**으로 켭니다

## 3. 키 ↔ 환경변수 매핑

| 항목 | 환경변수 |
|---|---|
| 네이버 로그인 아이디 (**전체 이메일** 형식) | `NAVER_EMAIL` |
| 발급된 앱 비밀번호 | `NAVER_APP_PASSWORD` |

```dotenv
NAVER_EMAIL=you@naver.com
NAVER_APP_PASSWORD=발급된_앱비밀번호
```

## 4. 한도·주의사항

- **하나의 앱 비밀번호를 메일·캘린더가 공용**합니다 — itda-email용으로 발급했다면 calendar 등에서 추가 발급 없이 재사용하면 됩니다.
- 발급 화면을 닫으면 다시 볼 수 없습니다. 분실 시 같은 메뉴에서 재발급하세요(기존 값은 무효화).
- "앱 비밀번호 오류"가 나면 흔한 원인 두 가지: ① 계정 본 비밀번호를 넣은 경우 ② 앱 비밀번호가 만료·삭제된 경우 — 재발급으로 해결됩니다.
- 아이디는 `you` 가 아니라 `you@naver.com` **전체 이메일**을 넣어야 합니다.

## 5. 이 키를 쓰는 스킬

- `itda-work/email` — 네이버 메일 읽기·발송 (IMAP 켜기 필요)
- `itda-work/calendar` — 네이버 캘린더 (CalDAV)
- `itda-work/plan-work` — 일정 기반 업무 계획 (calendar 경유)
