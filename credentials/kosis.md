# KOSIS 국가통계포털 Open API — 가입·키 발급 가이드

- 공식 사이트: <https://kosis.kr/openapi/>
- 발급 키: `KOSIS_API_KEY`
- Last Verified: 2026-06-10 (kosis 스킬 GUIDE에서 이전 — 실화면 재검증 시 갱신)

## 1. 가입 조건

- KOSIS 일반 회원가입으로 충분합니다 (비용 없음).
- 활용신청은 **자동 승인**이라 신청 즉시 사용할 수 있습니다.

## 2. 발급 절차

1. [kosis.kr/openapi](https://kosis.kr/openapi/) 접속 → 회원가입 후 로그인
2. 상단 메뉴 **Open API → 활용신청** → 사용 목적·서비스명 입력 후 신청 (**자동 승인, 즉시 이용 가능**)
3. **마이페이지 → 인증키 확인**에서 키 복사

## 3. 키 ↔ 환경변수 매핑

| 항목 | 환경변수 |
|---|---|
| KOSIS 인증키 | `KOSIS_API_KEY` |

```dotenv
KOSIS_API_KEY=발급받은_키
```

## 4. 한도·주의사항

- 인증키는 **Base64 형식이라 끝에 `=` 문자**가 붙는 경우가 있습니다 — 복사할 때 끝 문자가 잘리면 인증 실패하므로 전체를 한 번에 복사하세요.
- "활용신청이 필요합니다" 또는 인증 오류(코드 `10`/`11`/`42`)가 보이면 마이페이지에서 활용 상태와 **인증키 만료 여부**를 확인하세요 — 키가 만료되면 코드 `11` 오류가 납니다.
- 대용량 일괄(배치) 다운로드는 제공되지 않습니다 — 기간·항목을 나눠 조회합니다.

## 5. 이 키를 쓰는 스킬

- `itda-gov/kosis` — 국가통계 조회
- `itda-realty/realty-supply` — 주택 공급 통계
- `itda-work/market-scan` — 시장 규모·통계 보강 (없어도 웹 대체로 동작)
