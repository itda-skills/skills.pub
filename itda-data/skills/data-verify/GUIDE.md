title: "data-verify 상세 가이드"
---

엑셀·CSV에 정리한 숫자가 실제로 맞는지 검수합니다. 부분합과 총계가 어긋나거나, 집계값이 원장과 다르거나, 음수가 나오면 안 되는 칸에 음수가 있거나, 두 시트에서 같아야 할 값이 다를 때 — 기대값·실제값·차이를 근거로 "이 숫자 틀렸어요"를 짚어 줍니다.
**Claude에게 말로 부탁하면 됩니다** — "이 수치 맞는지 검수해줘", "합계 검산해줘"처럼요. 보고 전용이라 확인 없이 값을 바꾸지 않습니다.

---

## 빠른 시작

```
이 수치 맞는지 검수해줘
합계 검산해줘
원장이랑 대조해줘
```

파일을 보여 주면 합계·규칙·원장·시트 대조를 검사하고, 불일치를 표로 정리해 보고합니다.

---

## 활용 시나리오

### 합계·소계가 총계와 맞는지 (내부 정합)

합계·소계·총계 라벨이 붙은 행을 자동으로 찾아, 구성요소의 합과 표기된 값이 일치하는지 검산합니다. 별도 기준 없이 바로 됩니다.

```
이 매출표 소계랑 총계 맞는지 검산해줘
```

### 원장·원천과 대조 (외부 대조)

집계값이 원장 같은 정답셋과 일치하는지 대조합니다. 코드별 정답 금액을 함께 알려 주세요.

```
코드별 금액이 원장이랑 맞는지 대조해줘
```

### 값 규칙 위반 점검 (음수·범위·중복·합계 목표)

음수 불가 칸의 음수, 정해진 범위를 벗어난 값, 중복되면 안 되는 키, 특정 합계 목표(예: 비중 합 100) 위반을 찾습니다.

```
금액 칸에 음수 없는지, ID 중복 없는지 봐줘
비율이 0~100 범위인지, 비중 합이 100인지 검산해줘
```

### 시트 간 값 대조 (교차 참조)

두 시트(또는 파일)에서 같아야 할 값이 실제로 같은지 짝을 지어 대조합니다.

```
요약 시트 값이랑 상세 시트 합계가 같은지 대조해줘
```

## 출력 옵션

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| 보고서 테이블 | 종류·위치·기대값·실제값·차이·심각도·내용 열로 불일치를 한눈에 | 모든 검수 결과의 기본 형태 |
| 심각도 등급 | Critical(정합 깨짐·원장 불일치) · Warning(규칙 위반·범위) · Info(참고) | 무엇부터 볼지 우선순위 판단 |
| 요약 한 줄 | "이상 없음 / 주요 불일치 / 경미 불일치" + Critical·Warning·Info 개수 | 빠른 총평·보고서 첨부 |

## 실시간 지적 경로 (열린 엑셀)

Windows에서 엑셀을 열어 둔 채 이상치를 그 자리에서 짚어 달라고 하면, 파일 검수 대신 열린 워크북에 직접 표시할 수 있습니다.

```
열어둔 엑셀에서 이상치를 바로 하이라이트하고 코멘트로 짚어줘
```

- 이상치 셀을 **하이라이트(Critical 연빨강 / Warning 연노랑) + 코멘트**로 실시간 표시합니다.
- hyve가 가동 중이어야 하고, 설정 > MCP 탭에서 **office 프리셋**(`/mcp/office`)을 등록해야 하며 Windows + Microsoft Office 환경이 필요합니다.
- ⚠️ macOS·Linux이거나 Office가 없으면 이 경로는 지원되지 않아 파일 검수로 전환한다고 먼저 안내합니다. 두 경로는 같은 검수 규칙을 공유합니다.

---

## 팁

- **허용오차로 비교**: 부동소수점은 정확히 같은지가 아니라 허용오차(기본 0.01)로 판정합니다. "허용오차 0.1로 봐줘"처럼 느슨하거나 엄격하게 조정할 수 있습니다.
- **통화·백분율·천단위 자동 인식**: `$1,200`이나 `45%` 같은 표기도 숫자로 읽어 비교하므로 `45% + 55% = 100` 검산이 자연스럽게 됩니다.
- **합계 행은 자동 제외**: 규칙 검수(중복·합계 목표 등)에서 합계·소계 행은 이중 계산을 막기 위해 자동으로 건너뜁니다.
- **대조 기준은 함께 알려 주기**: 내부 정합은 자동이지만, 원장 대조·규칙·시트 짝은 기준(정답셋·규칙·짝 위치)을 함께 알려 주면 됩니다.

---

## 제한사항

- **보고 전용**: 확인 없이 값을 바꾸지 않습니다. 특정 발견을 수정해 달라고 명시적으로 요청하면 그 셀만 고칩니다.
- **범위 밖**: 수식 구조·오류 감사(`#REF!`·하드코드 등)와 수식 재계산은 `data-audit`가 담당합니다. 재무모델 무결성(BS 밸런스·DCF/LBO)은 범위 밖입니다.
- **실시간 지적 경로 조건**: 열린 엑셀 실시간 지적은 Windows + Office에서만 됩니다. 그 외 환경은 파일 검수로 전환됩니다.
- **인큐베이션 단계**: 이 스킬은 실험(experimental) 상태이며 기본 권장 대상은 아닙니다. 결과는 근거(기대값·실제값·차이)와 함께 검토해 확정하세요.
