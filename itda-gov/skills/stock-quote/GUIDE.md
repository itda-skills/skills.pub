# 주식 시세 조회 사용 가이드

금융위원회 공공데이터 시세 API로 KOSPI·KOSDAQ 주식의 현재가·과거 시세·종목 검색을 제공하는 스킬입니다.
**Claude에게 말로 부탁하면 됩니다** — "삼성전자 현재가 알려줘", "005930 시세 조회해줘", "삼성전자 최근 1주일 시세 조회해줘"처럼요.

> 이 스킬은 시세 **사실 조회**만 합니다. 매수·매도·목표가·비중 추천은 하지 않습니다(자본시장법 §6).

---

## 사전 준비

공공데이터포털 API 키가 필요합니다.

1. [https://www.data.go.kr/data/15094808/openapi.do](https://www.data.go.kr/data/15094808/openapi.do) 접속
2. "활용신청" 클릭 (자동승인)
3. 마이페이지 → 개발계정 → **Decoding 인증키** 복사
4. 발급받은 키를 **Claude Desktop 지침**에 추가합니다 — Claude Desktop → **설정 → 일반 → "Claude 지침"** 에 아래 한 줄을 붙여넣고 저장하세요.

```dotenv
KO_DATA_API_KEY=발급받은_Decoding키
```

> 자세한 가입·키 발급 절차(Decoding 키 주의사항 포함)는 [공공데이터포털 발급 가이드](https://skills.itda.work/credentials/data-go-kr/)를 참고하세요.

한 번 넣으면 이후 모든 대화에서 Claude가 자동으로 사용합니다 — `.env` 같은 파일을 직접 만들 필요가 없습니다.

> 개발자라면 작업 폴더 루트의 `.env`에 같은 한 줄을 넣어도 됩니다.

> 이 키는 itda-gov의 g2b·funding·realestate 스킬과 공유되지만, 금융위원회 주식시세정보 데이터셋(15094808)에는 **별도 활용신청**이 필요합니다.

---

## 자주 쓰는 요청

| 하고 싶은 것 | 이렇게 말하세요 |
|--------------|-----------------|
| 현재가 조회 | "삼성전자 현재가 알려줘" |
| 종목코드로 조회 | "005930 시세 조회해줘" |
| 과거 시세 (기간 지정) | "삼성전자 5월 1일부터 14일까지 주가 보여줘" |
| 과거 시세 (최근 N일) | "005930 최근 1주일 시세 조회해줘" |
| 종목 검색 | "카카오 들어가는 종목 찾아줘" |
| 종목명으로 검색 | "삼성 들어간 종목 목록 보여줘" |

종목은 **종목코드(6자리)** 또는 **종목명**으로 입력할 수 있습니다.

---

## 출력 형식

| 형식 | 설명 | 사용 시점 |
|------|------|-----------|
| JSON (기본) | 시세 수치 + 기준일자 + 디스클레이머 포함 구조화 JSON | Claude가 이어서 분석·요약할 때 |
| 테이블 | 사람이 읽기 좋은 표 | 수치를 눈으로 바로 확인할 때 |
| CSV | 스프레드시트로 옮기기 좋은 CSV | 엑셀·스프레드시트에 붙여넣을 때 |

출력 형식을 바꾸고 싶으면 "표로 보여줘", "CSV로 내보내줘"처럼 말하면 됩니다.

---

## 결과 내용

과거 시세 조회 결과에는 기준일자별로 시가·고가·저가·종가(OHLC)와 거래량이 포함됩니다. 종료일은 조회 범위에 포함됩니다.

| 항목 | 설명 |
|------|------|
| 현재가(종가) | 해당 기준일 기준 종가 |
| 시가·고가·저가 | 당일 시세 범위 |
| 거래량 | 해당일 거래량 |
| 기준일자 | 데이터 기준 날짜 (비실시간, 일 1회 갱신) |

> 모든 결과에 "정보 제공이며 투자자문이 아님, 투자판단·책임은 본인" 고정 디스클레이머가 붙습니다.

---

## 팁

- **종목명이 모호할 때**: 같은 검색어에 복수 종목이 일치하면 응답이 "모호한 종목"으로 반환됩니다. 정확한 6자리 종목코드 또는 정확한 종목명을 말하면 단일 결과로 좁혀집니다.
- **비실시간 데이터**: 이 API는 일 1회 갱신·비실시간(T+1)입니다. 결과의 기준일자를 확인하세요.
- **규제 디스클레이머 고정**: 모든 출력에 "정보 제공이며 투자자문이 아님, 투자판단·책임은 본인" 디스클레이머가 붙습니다. 매수·매도·목표가·비중 추천은 하지 않습니다.
- **보유종목 평가손익**은 자매 스킬 stock-portfolio를 사용하세요. 이 스킬은 단일 종목 조회·검색·과거 시세에 특화돼 있습니다.

## 안 될 때

| 증상 | 원인 / 해결 |
|------|-------------|
| "API 키가 설정되지 않았습니다" | **Claude 지침**(또는 `.env`)에 `KO_DATA_API_KEY=키` 누락. 위 "사전 준비" 확인 |
| HTTP 403 Forbidden | 금융위원회 주식시세정보(15094808) 활용신청 미완료. [data.go.kr](https://www.data.go.kr/data/15094808/openapi.do)에서 신청(자동승인) |
| "모호한 종목" 표시 | 종목명이 복수 일치. 6자리 종목코드로 다시 말하세요 |
| "종목 없음" 표시 | 종목코드·종목명 확인 후 재시도 |
