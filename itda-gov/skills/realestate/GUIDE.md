## 빠른 시작

국토교통부 부동산 실거래가를 자연어 한 줄로 조회할 수 있습니다.

```
강남구 아파트 2026년 1월 실거래가 알려줘
분당 전세 시세 확인해줘
래미안 단지 매매가 요약해줘
```

이 한 줄이면 스킬이 자동으로 지역 코드를 매칭하고, 조회 월을 추정하여 매매 또는 전월세 실거래 데이터를 반환합니다.

## 사전 준비

공공데이터포털 API 키가 필요합니다.

1. https://www.data.go.kr 회원가입
2. 사용하려는 데이터셋을 **각각 활용신청** (전부 자동승인, 신청 즉시 5~30분 내 사용 가능)
   - 아파트 매매: <https://www.data.go.kr/data/15126469/openapi.do>
   - 아파트 전월세: <https://www.data.go.kr/data/15126474/openapi.do>
   - 오피스텔 매매: <https://www.data.go.kr/data/15126464/openapi.do>
   - 오피스텔 전월세: <https://www.data.go.kr/data/15126475/openapi.do>
3. 마이페이지에서 **Decoding** 키(일반 인증키) 복사
4. `claude config set env.KO_DATA_API_KEY "발급받은_키"` 또는 `.env` 파일에 저장

> 동일 인증키 하나로 4개 모두 호출 가능하지만, **데이터셋마다 별도 활용신청**이 필요합니다. 신청하지 않은 서비스를 호출하면 오류 메시지에 해당 신청 링크가 자동으로 표시됩니다.
>
> `regions` 서브커맨드는 API 키 없이 동작합니다. 지역명을 미리 확인할 때 활용하세요.

## 활용 시나리오

### 매매 실거래 + 요약 통계

특정 지역·월의 아파트 매매 실거래를 평균·중위·최고가 통계와 함께 조회합니다.

```
강남구 아파트 2026년 1월 실거래가 평균이랑 중위가 알려줘
```

### 전월세 시세 조회

분당구 같은 시·구 단위로 전월세 실거래를 한 번에 가져옵니다.

```
성남시 분당구 2026년 1월 전월세 시세 확인해줘
```

### 지역 코드 미리 확인 (API 키 불필요)

지역명이 정확하지 않을 때 `regions` 명령으로 매칭 가능한 지역을 검색합니다.

```
국토부 지역코드에서 강남 들어간 지역 다 보여줘
```

## 출력 옵션

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| `--format json` | 구조화된 JSON 배열 (기본값) | 파이프라인·후속 가공·다른 스킬과 연계 |
| `--format table` | 정렬된 ASCII 표 | 터미널에서 바로 읽고 비교 |
| `--summary` | 평균·중위·최고가 통계 추가 | 시세 파악, 보고서 요약 |

## 팁

- **지역명은 `regions`로 먼저 확인**: 시·구 명칭이 살짝만 달라도 "지역 코드를 찾을 수 없습니다" 오류가 납니다. `regions` 명령으로 정확한 표기를 찾으세요.
- **`--summary`로 한눈에 파악**: 거래 100건을 직접 보지 말고 평균·중위·최고가 통계로 시세 흐름부터 잡으세요.
- **`--name`으로 단지 추적**: `--name "래미안"` 처럼 단지명 일부만 넣어도 해당 단지 거래만 필터링됩니다. 시계열로 한 단지를 추적할 때 유용합니다.
- **터미널 즉시 확인은 `--format table`**: JSON 기본값은 가공용입니다. 사람이 바로 읽을 때는 `--format table`을 추가하세요.

## 제한사항

- **아파트·오피스텔 실거래만 지원**: 단독·다세대·상가 등 다른 주택 유형은 조회되지 않습니다.
- **신고 지연으로 1~2개월 데이터 누락 가능**: 실거래 신고 의무 기한 때문에 조회 월 대비 최근 1~2개월 데이터는 일부 누락될 수 있습니다.
- **월 단위 조회만 가능**: `--year-month YYYYMM` 형식만 허용됩니다. 일별·주별 조회나 다중 월 일괄 조회는 지원하지 않습니다. 연간 조회가 필요하면 12회 반복 호출하세요.
- **공공데이터포털 트래픽 한도**: 일일 호출량 한도가 있으므로 자주 동일 조건을 반복 조회하지 마세요. 결과는 직접 캐싱해두는 것이 안전합니다.
- **실거래가만 제공**: 호가·시세·의도 가격은 포함되지 않습니다. 국토교통부에 신고된 실제 거래 가격만 반환됩니다.

## 오류 처리 빠른 참조

| 오류 | 해결 |
|------|------|
| `KO_DATA_API_KEY가 설정되지 않았습니다` | `claude config set env.KO_DATA_API_KEY "키"` |
| `지역 코드를 찾을 수 없습니다` | `regions` 명령으로 정확한 지역명 확인 |
| `HTTP Error 403: Forbidden — 게이트웨이 권한 거부` | 메시지에 표시된 신청 링크에서 활용신청 (자동승인). 신청 직후라면 5~30분 후 재시도 |
| `데이터가 없습니다` (resultCode=03) | 다른 년월로 재시도 (신고 지연 또는 거래 없음) |
| `활용 미승인` (resultCode=20) | 마이페이지 활용신청 승인 상태 확인 |
| `등록되지 않은 서비스키` (resultCode=30) | 마이페이지에서 Decoding 키 다시 복사해 환경변수 갱신 |
