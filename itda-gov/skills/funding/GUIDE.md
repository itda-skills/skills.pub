# funding 사용 가이드

K-Startup 공공데이터에서 정부 창업·중소기업 지원사업 공고를 검색합니다.
**Claude에게 자연어로 부탁하면 됩니다** — "AI 스타트업 지원사업 찾아줘", "지금 모집 중인 창업 공고 알려줘"처럼요. 터미널 명령을 직접 입력할 필요는 없습니다.

---

## 사전 준비 — API 키 발급

funding 스킬은 공공데이터포털 API 키가 필요합니다.

1. [창업진흥원 K-Startup 조회서비스 활용신청 페이지](https://www.data.go.kr/data/15125364/openapi.do)에서 회원가입 후 활용신청 (**자동승인**)
2. 승인 후 마이페이지 → **개발계정 → Decoding** 키(일반 인증키) 복사
3. 발급받은 키를 **Claude Desktop 지침**에 추가합니다 — Claude Desktop → **설정 → 일반 → "Claude 지침"** 에 아래 한 줄을 붙여넣고 저장하세요.

```dotenv
KO_DATA_API_KEY=발급받은_Decoding_키
```

한 번 넣으면 이후 모든 대화에서 Claude가 자동으로 사용합니다 — `.env` 같은 파일을 직접 만들 필요가 없습니다.

> 개발자라면 작업 폴더 루트의 `.env`에 같은 한 줄을 넣어도 됩니다.

> 자동승인이지만 게이트웨이 동기화에 5~30분(드물게 1시간) 걸립니다. 신청 직후 "인증 실패"가 뜨면 잠시 후 다시 요청해보세요.

> 같은 키를 `realestate`·`g2b` 스킬에서도 함께 사용할 수 있습니다. 단, 각 API는 별도로 활용신청해야 합니다.

---

## 자주 쓰는 요청

| 하고 싶은 것 | 이렇게 말하세요 |
|--------------|-----------------|
| 지금 신청 가능한 공고만 | "AI 스타트업 지원사업 중에서 현재 모집 중인 것만 찾아줘" |
| 키워드 검색 | "중소기업이 신청할 수 있는 사업화 지원사업 알려줘" |
| 분야 좁혀 검색 | "소프트웨어 쪽 사업화 지원사업만 추려줘" |
| 연도별 현황 | "2026년 청년창업 통합공고 현황 정리해줘" |
| 일반 질문형 | "제가 받을 수 있는 AI 관련 정부 지원사업이 있을까요?" |

## 활용 시나리오

### 현재 모집 중인 공고 찾기

가장 자주 쓰는 패턴입니다. 키워드와 함께 "모집 중인 것만"이라고 말하면 이미 마감된 공고를 걸러내고 지금 신청 가능한 공고만 보여 줍니다.

```
AI 스타트업 지원사업 중에서 현재 모집 중인 것만 찾아줘
```

```
청년창업으로 지금 모집 중인 공고 뭐가 있나요?
```

### 분야를 좁혀서 맞춤 검색

키워드만으로 결과가 너무 많을 때 분야(사업화·R&D·창업교육 등)를 함께 말하면 내 상황에 맞는 공고가 드러납니다.

```
소프트웨어 쪽 사업화 지원사업만 추려줘
```

### 연도별 통합공고 현황 훑어보기

특정 연도에 어떤 통합공고가 있었는지 사업명·지원 분야·지원 규모를 한눈에 보여 줍니다. 자금 조달 계획을 세우거나 제안서를 작성할 때 참고용으로 유용합니다.

```
2026년 청년창업 통합공고 현황 정리해줘
```

## 검색 결과에 포함되는 정보

기본 결과에는 다음이 포함됩니다: **공고명·사업명**, **접수 시작일~종료일**, **주관기관**, **지원 분야**, **상세 페이지 링크**, (데이터가 있는 경우) **지원 총액**.

---

## 팁

- **"모집 중인 것만"이라고 말해주세요.** 그냥 검색하면 이미 마감된 공고까지 섞여 나옵니다. 지금 신청 가능한 공고만 보려면 "현재 모집 중인 것만"이라는 말을 꼭 넣으세요.
- **분야를 함께 말하면 정밀도가 올라갑니다.** "사업화", "R&D", "창업교육" 같은 분야를 키워드와 함께 말하면 관계없는 공고가 크게 줄어듭니다.
- **원본 공고 링크를 꼭 확인하세요.** 스킬이 돌려주는 정보에 포함된 상세 페이지 링크로 원본 공고를 직접 확인하고 신청 조건을 꼼꼼히 검토하세요.

---

## 안 될 때

| 증상 | 원인 / 해결 |
|------|-------------|
| "API 키가 없다"는 안내 | **Claude 지침**(또는 `.env`)에 `KO_DATA_API_KEY`가 없거나 키가 비어 있음. 위 "사전 준비" 확인 |
| "인증 실패" 또는 HTTP 403 | 활용신청 직후 게이트웨이 동기화 중. 5~30분 후 다시 요청 |
| "검색 결과가 없습니다" | 해당 키워드 공고 없음. 다른 키워드나 더 넓은 표현으로 재검색 |

---

## 주의사항

- ⚠️ **K-Startup 범위만 커버합니다.** 중소벤처기업부·K-Startup 통합공고에 등록된 사업이 기준입니다. 자치단체·개별 기관 공고나 다른 부처의 공고는 빠질 수 있으니, 최종 검토는 원 기관 홈페이지도 함께 확인하세요.
- ⚠️ **공고 시점과 즉시 동기화되지 않습니다.** K-Startup 웹사이트의 공고와 몇 시간~하루 차이가 날 수 있습니다. 최종 신청 직전에는 결과에 포함된 상세 페이지 링크로 원본을 꼭 재확인하세요.
