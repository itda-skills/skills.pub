## 빠른 시작

정부 지원사업 공고를 자연스럽게 물어볼 수 있습니다. 아래처럼 질문형으로 말하면 스킬이 알아서 K-Startup 데이터를 뒤져 모집 중인 공고부터 추려 줍니다.

```
제가 받을 수 있는 AI 관련 정부 지원사업이 있을까요?
```

```
청년창업으로 지금 모집 중인 공고 뭐가 있나요?
```

```
중소기업이 신청할 수 있는 사업화 지원사업 알려주세요.
```

## 사전 준비

`KO_DATA_API_KEY` 환경변수 등록이 필요합니다.

1. **API 키 발급처**: [공공데이터포털](https://www.data.go.kr)
2. **발급 절차**:
   - 공공데이터포털 회원가입
   - "K-Startup 통합공고 지원사업 정보 서비스" 검색 → 활용신청 (승인까지 수 시간 소요)
   - 마이페이지 → 개발계정 → **Decoding** 키(일반 인증키) 복사
3. **환경변수 등록** (택 1):
   ```bash
   # 권장: Claude Cowork 설정에 등록
   claude config set env.KO_DATA_API_KEY "발급받은_키"
   ```
   또는 `CLAUDE.md`·`.env` 파일에 다음 줄을 추가합니다.
   ```
   KO_DATA_API_KEY=발급받은_키
   ```

> 같은 키를 `realestate`·`g2b` 스킬에서도 공용으로 사용할 수 있습니다. 단, 각 API는 별도로 활용신청해야 합니다.

## 활용 시나리오

### 키워드로 현재 모집 중인 공고만 찾기

가장 자주 쓰는 조합입니다. 키워드와 `--active`를 함께 쓰면 이미 종료된 공고를 걸러내고 지금 신청 가능한 것만 보여 줍니다.

```
AI 스타트업 지원사업 중에서 현재 모집 중인 것만 찾아줘
```

### 분야를 좁혀서 맞춤 검색

키워드가 너무 넓으면 결과가 많아져 노이즈가 생깁니다. 이럴 때 분야(사업화·R&D·창업교육 등)로 한 번 더 필터링하면 내 상황에 맞는 공고가 드러납니다.

```
소프트웨어 쪽 사업화 지원사업만 추려줘
```

### 연도별 통합공고 현황 훑어보기

`overview` 서브커맨드는 특정 연도에 어떤 통합공고가 있었는지 사업명·지원 분야·지원 규모를 한눈에 보여 줍니다. 자금 조달 계획을 세우거나 RFP를 쓸 때 참고용으로 유용합니다.

```
2026년 청년창업 통합공고 현황 정리해줘
```

## 출력 옵션

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| JSON (기본값) | 공고명·마감일·주관기관·지원 금액을 구조화된 JSON으로 반환 | 다른 도구와 연동하거나 LLM이 재가공할 때 |
| `--format table` | 공고명·신청 기간·주관기관 컬럼을 가진 사람 친화적 표 | 즉시 눈으로 훑어볼 때 |
| `search` 서브커맨드 | 개별 공고 단위 결과 (상세 페이지 URL 포함) | 신청 여부를 바로 판단하고 싶을 때 |
| `overview` 서브커맨드 | 연도별 사업 현황 요약 | 큰 그림과 예산 흐름 파악용 |

검색 결과에는 다음이 기본으로 포함됩니다: **공고명·사업명**, **접수 시작일~종료일**, **주관기관**, **지원 분야**, **상세 페이지 URL**, (데이터가 있는 경우) **지원 총액**.

## 팁

- **`--active`를 꼭 붙이세요.** 빼고 검색하면 이미 마감된 공고까지 섞여 나옵니다. 실제 신청 가능한 공고만 보려면 거의 항상 `--active`가 필요합니다.
- **`--format table`로 확인용 출력.** JSON은 파이프·LLM 재가공에는 좋지만 사람 눈으로 확인할 때는 표가 훨씬 빠릅니다.
- **키워드 + `--field` 조합으로 정밀도 상승.** 키워드만 넣으면 결과가 많아지니 `--field "사업화"` 같은 분야 필터를 추가하면 노이즈가 급감합니다.

## 주의사항

- ⚠️ **API 키는 별도 신청이 필요합니다.** 공공데이터포털에서 `K-Startup 통합공고 지원사업 정보 서비스`를 활용신청해야 키가 발급됩니다. 서비스에 따라 승인까지 시간이 걸릴 수 있습니다.
- ⚠️ **K-Startup 범위만 커버합니다.** 중소벤처기업부·K-Startup 통합공고에 등록된 사업이 기준입니다. 자치단체·법정동 개별 기관 공고나 다른 부처의 공고는 빠질 수 있으니, 최종 검토는 원 기관 홈페이지도 함께 확인하세요.
- ⚠️ **공고 시점과 API는 즉시 동기화되지 않습니다.** 실시간 업데이트가 아니므로 K-Startup 웹사이트의 공고와 몇 시간~하루 차이가 날 수 있습니다. 최종 신청 직전에는 결과에 포함된 `detl_pg_url`로 원본을 꼭 재확인하세요.
