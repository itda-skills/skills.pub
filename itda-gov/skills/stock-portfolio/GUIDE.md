# 보유종목 평가손익 계산 사용 가이드

금융위원회 공공데이터 시세 API로 보유 종목의 평가금액·평가손익·수익률을 순수 산술로 계산하는 스킬입니다.
**Claude에게 말로 부탁하면 됩니다** — "삼성전자 10주 평단가 28만 원 기준 손익 계산해줘", "삼성전자 10주 카카오 5주 포트폴리오 평가금액 알려줘"처럼요.

> 이 스킬은 **순수 산술 계산**만 합니다. 매수·매도·목표가·리밸런싱 추천은 하지 않습니다(자본시장법 §6).

---

## 사전 준비

공공데이터포털 API 키가 필요합니다.

1. [https://www.data.go.kr/data/15094808/openapi.do](https://www.data.go.kr/data/15094808/openapi.do) 접속
2. "활용신청" 클릭 (자동승인)
3. 마이페이지 → 개발계정 → **Decoding 인증키** 복사
4. 발급받은 키를 **Claude Desktop 지침**에 추가합니다 — Claude Desktop → **설정 → 일반 → "Claude 지침"** 에 아래 한 줄을 붙여넣고 저장하세요.

```dotenv
KO_DATA_API_KEY=발급받은_Decoding키
```

> 자세한 가입·키 발급 절차(Decoding 키 주의사항 포함)는 [공공데이터포털 발급 가이드](https://skills.itda.work/credentials/data-go-kr/)를 참고하세요.

한 번 넣으면 이후 모든 대화에서 Claude가 자동으로 사용합니다 — `.env` 같은 파일을 직접 만들 필요가 없습니다.

> 개발자라면 작업 폴더 루트의 `.env`에 같은 한 줄을 넣어도 됩니다.

> 이 키는 itda-gov의 g2b·funding·realestate 스킬과 공유되지만, 금융위원회 주식시세정보 데이터셋(15094808)에는 **별도 활용신청**이 필요합니다.

---

## 자주 쓰는 요청

| 하고 싶은 것 | 이렇게 말하세요 |
|--------------|-----------------|
| 단일 종목 평가손익 | "삼성전자 10주 평단가 28만 원 기준 손익 계산해줘" |
| 종목코드로 입력 | "005930 10주 평단가 28만 원 손익 계산해줘" |
| 복수 종목 포트폴리오 | "삼성전자 10주 카카오 5주 포트폴리오 평가금액 알려줘" |
| 표 형식으로 보기 | "내 보유종목 평가손익 표로 정리해줘" |

종목은 **종목코드(6자리)** 또는 **종목명**으로 입력할 수 있습니다. 종목명으로 입력했을 때 복수 후보가 있으면 해당 종목만 "모호한 종목"으로 표시됩니다 — 이때는 정확한 6자리 종목코드를 말하면 됩니다.

## 결과 내용

| 항목 | 설명 |
|------|------|
| 평가금액 | 보유 수량 × 현재가(종가) |
| 매입금액 | 보유 수량 × 평균매입단가 |
| 평가손익 | 평가금액 − 매입금액 |
| 수익률 | 평균매입단가 대비 손익 비율 (%) |
| 계산 수식 | 계산 과정을 수식으로 함께 제공 |
| 기준일자 | 데이터 기준 날짜 (비실시간, 일 1회 갱신) |

> 모든 결과에 "정보 제공이며 투자자문이 아님, 투자판단·책임은 본인" 고정 디스클레이머가 붙습니다.

---

## 팁

- **보유종목은 저장되지 않습니다**: 입력한 보유내역은 계산 후 출력만 하며 로컬 파일·DB에 기록되지 않습니다. 매 호출마다 보유내역을 다시 입력해야 합니다.
- **비실시간 데이터**: 현재가는 일 1회 갱신·비실시간(T+1)입니다. 결과의 기준일자를 확인하세요.
- **종목명이 모호할 때**: 같은 검색어에 복수 종목이 일치하면 해당 종목만 "모호한 종목"으로 표시됩니다. 정확한 6자리 종목코드를 입력하면 해결됩니다.
- **평균매입단가가 0이면**: 수익률은 계산 불가로 표시됩니다.
- **단일 종목 조회·과거 시세·종목 검색**은 자매 스킬 stock-quote를 사용하세요.

## 안 될 때

| 증상 | 원인 / 해결 |
|------|-------------|
| "API 키가 설정되지 않았습니다" | **Claude 지침**(또는 `.env`)에 `KO_DATA_API_KEY=키` 누락. 위 "사전 준비" 확인 |
| HTTP 403 Forbidden | 금융위원회 주식시세정보(15094808) 활용신청 미완료. [data.go.kr](https://www.data.go.kr/data/15094808/openapi.do)에서 신청(자동승인) |
| "모호한 종목" 표시 | 종목명이 복수 일치. 6자리 종목코드로 다시 말하세요 |
| "종목 없음" 표시 | 종목코드·종목명 확인 후 재시도 |
