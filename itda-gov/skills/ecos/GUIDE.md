## 빠른 시작

한국은행 ECOS(경제통계시스템)에서 거시경제 지표를 가져오는 가장 빠른 방법입니다.

```
최근 5년간 GDP 추이 정리해줘
기준금리와 환율 요약해줘
100대 경제지표 보여줘
CPI 추이 분기별로 가져와
```

스킬이 자동으로 통계표 코드를 찾고, 기간과 주기에 맞게 데이터를 조회해 JSON이나 표로 돌려줍니다.

## 사전 준비

`ECOS_API_KEY` 환경변수 등록이 필요합니다.

1. **API 키 발급처**: [한국은행 경제통계시스템 ECOS Open API](https://ecos.bok.or.kr/api/)
2. **발급 절차**:
   - ECOS 회원가입
   - **Open API** → **인증키 신청** → 사용 목적·서비스명 입력 후 제출
   - 한국은행 승인 대기 (영업일 기준 2~3일)
   - 마이페이지 → 인증키 확인
3. **환경변수 등록** (택 1):
   ```bash
   # 권장: Claude Cowork 설정에 등록
   claude config set env.ECOS_API_KEY "발급받은_키"
   ```
   또는 `CLAUDE.md`·`.env` 파일에 다음 줄을 추가합니다.
   ```
   ECOS_API_KEY=발급받은_키
   ```

> 보고서 마감이 임박했을 때 키 발급을 시작하면 시간을 못 맞춥니다. 미리 발급받아 두세요.

## 활용 시나리오

### 거시경제 지표 시계열 조회

보고서·사업계획서 본문에 인용할 GDP·CPI·기준금리 같은 지표를 기간을 지정해 가져옵니다.

```
최근 5년간 소비자물가지수를 연도별로 표로 정리해줘
```

### 100대 경제지표 요약

한국은행이 선정한 핵심 지표를 한 번에 조회해 요약본을 만듭니다.

```
한국은행 100대 경제지표 중 금리 관련 항목만 뽑아줘
```

### 경제 용어 각주 생성

본문에 등장한 전문용어를 공식 정의로 각주 처리합니다.

```
"GDP디플레이터" 용어 정의 가져와서 각주로 넣어줘
```

## 출력 옵션

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| JSON (기본) | `--format json`. stdout으로 구조화된 시계열 출력 | LLM·후속 스크립트 소비 |
| 텍스트 표 | `--format table`. markdown 표 형식 | 보고서·노션·README 붙여넣기 |
| 시계열 숫자 배열 | 날짜·기간과 값 쌍으로 배열 출력 | 차트·그래프 생성 전처리 |

## 주요 CLI 플래그

| 플래그 | 설명 |
|--------|------|
| `--stat` | 통계표 코드 (예: `901Y009`=CPI, `200Y001`=GDP). search·items에 필수 |
| `--start` / `--end` | 기간. 연도(`2020`) 또는 분기·월 코드(`20201`=2020년 1분기) |
| `--period` | `annual` / `quarter` / `month` / `day` — 기본 `annual` |
| `--format` | `json`(기본) / `table` |

## 팁

- **서브커맨드 순서**: 통계표 코드를 모르면 `tables` → `items` → `search` 순으로 탐색하세요. tables로 목록을 보고, items로 하위 항목 코드를 확인한 뒤 search로 실제 데이터를 조회합니다.
- **분기·월 코드 형식**: `--period quarter`일 때 `--start 20201 --end 20244`는 "2020년 1분기 ~ 2024년 4분기"를 뜻합니다. 월별은 `YYYYMM`(예: `202401`).
- **용어 각주 자동화**: `word` 서브커맨드로 공식 용어 정의를 뽑아 보고서의 전문용어 각주를 일괄 생성할 수 있습니다.
- **보고서 바로 붙여넣기**: `--format table`은 markdown 테이블로 나오므로 노션·Obsidian·README에 그대로 붙여넣기 가능합니다.

## 제한사항

- ⚠️ **통계표 코드 사전 확인 필수**: `--stat`이 틀리면 "통계표를 찾을 수 없습니다" 오류만 반환됩니다. 불확실하면 반드시 `tables`·`items`로 먼저 확인하세요.
- ⚠️ **요청 회수 제한**: ECOS API는 호출당 최대 100,000건이라 장기간 일별·월별 조회는 분할이 필요합니다.
- ⚠️ **실시간 데이터 아님**: ECOS는 확정 통계이므로 공표 지연(예: 분기 GDP는 약 2개월 뒤 공표)이 있습니다.
