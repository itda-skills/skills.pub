# StatisticMeta — 통계메타DB

출처: [API개발명세서_통계메타DB.xls](API개발명세서_통계메타DB.xls)
원본 발간: 한국은행, 2025-03-26
추출일: 2026-04-28

## 기본 정보

| 항목 | 내용 |
|------|------|
| 서비스명 | StatisticMeta |
| 베이스 URL | https://ecos.bok.or.kr/api/ |
| 호출 방식 | PATH 기반 |
| 출력 형식 | XML, JSON |

통계의 메타데이터(작성 연혁, 출처 등)를 조회. 데이터의 맥락 파악에 유용.

## 요청 인자

| 항목명(국문) | 필수 | 샘플 | 설명 |
|------------|------|------|------|
| 서비스명 | Y | StatisticMeta | API 서비스명 |
| 인증키 | Y | sample | 인증키 |
| 요청유형 | Y | xml | xml, json |
| 언어구분 | Y | kr | kr/en |
| 요청시작건수 | Y | 1 | 시작 번호 |
| 요청종료건수 | Y | 10 | 끝 번호 |
| **데이터명** | Y | 경제심리지수 | 검색할 데이터명 (자연어 키워드) |

## 출력값

| 항목명(국문) | 항목명(영문) | 크기 | 샘플 | 설명 |
|------------|-------------|------|------|------|
| 레벨 | LVL | 2 | 2 | 트리 레벨 (계층 구조) |
| 상위통계항목코드 | P_CONT_CODE | 8 | 1 | 부모통계항목코드 |
| 통계항목코드 | CONT_CODE | 8 | 0000000003 | 통계항목코드 |
| 통계항목명 | CONT_NAME | 200 | 최초작성연도 | 통계항목명 |
| 메타데이터 | META_DATA | 200 | 2012.6월 최초 작성(과거 시계열은 2003.1월부터 제공) | 메타데이터 |

## 테스트 URL

```
https://ecos.bok.or.kr/api/StatisticMeta/sample/xml/kr/1/10/경제심리지수
```
