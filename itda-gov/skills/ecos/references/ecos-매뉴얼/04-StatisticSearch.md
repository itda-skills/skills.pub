## 테스트 URL

```
https://ecos.bok.or.kr/api/StatisticSearch/sample/xml/kr/1/10/200Y001/A/2015/2021/10101/?/?/?
```

> 통계항목코드 2~4가 없을 때 `?`를 placeholder로 사용. 코드 `ecos_api.py:search_statistics()`는 빈 코드를 빈 세그먼트로 처리.

## 주기-날짜 형식 매트릭스

| 주기(CYCLE) | 의미 | 날짜 형식 | 예시 |
|------------|------|----------|------|
| A | 년 | YYYY | 2024 |
| S | 반년 | YYYYS{1,2} | 2024S1 |
| Q | 분기 | YYYYQ{1~4} | 2024Q1 |
| M | 월 | YYYYMM | 202401 |
| SM | 반월 | YYYYMMS{1,2} | 202401S1 |
| D | 일 | YYYYMMDD | 20240101 |
