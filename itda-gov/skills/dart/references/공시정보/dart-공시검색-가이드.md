## 에러 코드

| 코드 | 메시지 |
|------|--------|
| 000 | 정상 |
| 010 | 등록되지 않은 키 |
| 011 | 사용할 수 없는 키(일시적 중지) |
| 012 | 접근할 수 없는 IP |
| 013 | 조회된 데이터 없음 |
| 014 | 파일이 존재하지 않음 |
| 020 | 요청 제한 초과(20,000건 이상) |
| 021 | 조회 가능한 회사 개수 초과(최대 100건) |
| 100 | 필드의 부적절한 값 |
| 101 | 부적절한 접근 |
| 800 | 시스템 점검으로 인한 서비스 중지 |
| 900 | 정의되지 않은 오류 |
| 901 | 사용자 계정 개인정보 보유기간 만료 |
