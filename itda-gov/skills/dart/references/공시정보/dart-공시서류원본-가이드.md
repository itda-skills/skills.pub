# 공시서류원본파일 API 개발가이드

출처: https://opendart.fss.or.kr/guide/detail.do?apiGrpCd=DS001&apiId=2019003
추출일: 2026-04-28

## 기본 정보

| 항목 | 내용 |
|------|------|
| 메서드 | GET |
| 요청URL | https://opendart.fss.or.kr/api/document.xml |
| 인코딩 | UTF-8 |
| 출력포멧 | Zip FILE (binary) |

## 요청 인자

| 요청키 | 명칭 | 타입 | 필수여부 | 값설명 |
|--------|------|------|---------|--------|
| crtfc_key | API 인증키 | STRING(40) | Y | 발급받은 인증키(40자리) |
| rcept_no | 접수번호 | STRING(14) | Y | 접수번호 |

## 응답 결과

| 응답키 | 명칭 | 출력설명 |
|--------|------|---------|
| result | status | 에러 및 정보 코드 |
| | message | 에러 및 정보 메시지 |

## 에러 코드

| 코드 | 설명 |
|------|------|
| 000 | 정상 |
| 010 | 등록되지 않은 키 |
| 011 | 사용할 수 없는 키(일시 중지된 상태) |
| 012 | 접근할 수 없는 IP |
| 013 | 조회된 데이터 없음 |
| 014 | 파일이 존재하지 않음 |
| 020 | 요청 제한 초과(20,000건 이상) |
| 021 | 조회 가능한 회사 개수 초과(최대 100건) |
| 100 | 필드의 부적절한 값 |
| 101 | 부적절한 접근 |
| 800 | 시스템 점검으로 인한 서비스 중지 |
| 900 | 정의되지 않은 오류 |
| 901 | 사용자 계정 개인정보 보유기간 만료 |

## 응답 형식 주의

응답이 성공일 경우 ZIP 파일(binary)이 반환되며, 오류일 경우 XML로 result/status/message가 반환됨. 코드에서는 Content-Type 또는 응답 첫 바이트로 분기 처리해야 함.
