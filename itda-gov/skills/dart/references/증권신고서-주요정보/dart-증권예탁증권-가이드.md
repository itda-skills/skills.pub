# 증권예탁증권 API 개발가이드

추출일: 2026-04-28

## 기본 정보

| 항목 | 내용 |
|------|------|
| 메서드 | GET |
| JSON URL | https://opendart.fss.or.kr/api/stkdpRs.json |
| XML URL | https://opendart.fss.or.kr/api/stkdpRs.xml |
| 인코딩 | UTF-8 |

## 요청 인자

| 요청키 | 명칭 | 타입 | 필수 | 설명 |
|--------|------|------|------|------|
| crtfc_key | API 인증키 | STRING(40) | Y | 발급받은 인증키(40자리) |
| corp_code | 고유번호 | STRING(8) | Y | 공시대상회사의 고유번호(8자리) |
| bgn_de | 시작일 | STRING(8) | Y | 접수일자(YYYYMMDD), 2015년 이후 |
| end_de | 종료일 | STRING(8) | Y | 접수일자(YYYYMMDD), 2015년 이후 |

## 응답 결과

응답 그룹: 일반사항, 증권의종류, 인수인정보, 자금의사용목적, 매출인에관한사항.

주요 필드: 접수번호, 법인구분, 회사명, 증권종류, 수량, 액면가액, 모집금액, 인수인정보, 보유 정보 등.

## 에러 코드

| 코드 | 메시지 |
|------|--------|
| 000 | 정상 |
| 010 | 등록되지 않은 키 |
| 013 | 조회된 데이타가 없습니다 |
| 020 | 요청 제한 초과(20,000건 이상) |
| 100 | 필드의 부적절한 값 |
| 800 | 시스템 점검 중 |
| 900 | 정의되지 않은 오류 |
