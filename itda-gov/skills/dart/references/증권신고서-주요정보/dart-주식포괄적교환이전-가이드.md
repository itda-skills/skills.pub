## 요청 인자

| 요청키 | 명칭 | 타입 | 필수 | 설명 |
|--------|------|------|------|------|
| crtfc_key | API 인증키 | STRING(40) | Y | 40자리 인증키 |
| corp_code | 고유번호 | STRING(8) | Y | 8자리 고유번호 |
| bgn_de | 시작일 | STRING(8) | Y | 접수일자(YYYYMMDD), 2015년 이후 |
| end_de | 종료일 | STRING(8) | Y | 접수일자(YYYYMMDD), 2015년 이후 |

## 응답 결과

응답 그룹: 일반사항, 발행증권, 당사회사에관한사항.

주요 필드: 접수번호, 법인구분, 회사코드, 회사명, 계약일자, 주주총회일, 주식매수청구권 행사기간, 합병기일, 교환비율, 외부평가자, 발행증권 정보.

## 에러 코드

| 코드 | 메시지 |
|------|--------|
| 000 | 정상 |
| 010 | 등록되지 않은 키입니다 |
| 013 | 조회된 데이타가 없습니다 |
| 020 | 요청 제한 초과(20,000건 이상) |
| 021 | 조회 회사 개수 초과(최대 100건) |
| 100 | 필드의 부적절한 값 |
| 800 | 시스템 점검 중 |
