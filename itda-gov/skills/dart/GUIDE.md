## 빠른 시작

DART 전자공시 시스템에서 기업정보를 조회하는 가장 간단한 방법입니다.

```
삼성전자 매출과 직원수 조회해줘
```

이 한 줄이면 스킬이 DART에서 법인 검색 → 재무제표 → 직원현황을 차례로 조회해 정리해 줍니다.

## 사전 준비

`DART_API_KEY` 환경변수 등록이 필요합니다.

1. **API 키 발급처**: [금융감독원 DART](https://opendart.fss.or.kr)
2. **발급 절차**:
   - DART 회원가입
   - **오픈 API** → **인증키 신청/관리** → 사용 목적 입력 후 신청
   - 40자리 키 즉시 발급 (승인 대기 없음)
3. **환경변수 등록** (택 1):
   ```bash
   # 권장: Claude Cowork 설정에 등록
   claude config set env.DART_API_KEY "발급받은_키"
   ```
   또는 `CLAUDE.md`·`.env` 파일에 다음 줄을 추가합니다.
   ```
   DART_API_KEY=발급받은_키
   ```
   CLI 호출 시 `--api-key` 인자로 일회성 전달도 가능합니다.

## 활용 시나리오

### 경쟁사·고객사 종합 프로필

회사명 하나로 기업개황·재무제표·직원현황까지 한 번에 조회합니다. 제안서·영업자료 작성에 가장 많이 쓰는 패턴입니다.

```
카카오 기업 정보 정리해줘
```

### 정확한 법인명 검색

"삼성", "현대" 같은 축약형이 여러 계열사와 매칭될 때, 먼저 검색으로 공식 법인명과 기업코드를 확정합니다.

```
DART에서 "현대" 들어간 회사 찾아줘
```

### 연도별 재무제표 비교

경쟁사의 매출·영업이익 추이를 연도별로 비교해 시장 분석 보고서를 만듭니다.

```
카카오 2022년, 2023년, 2024년 매출 비교해줘
```

### 직원현황으로 회사 규모 파악

구분별 직원수, 평균근속연수, 연간급여를 조회해 고객사 규모 감을 잡습니다.

```
네이버 2024년 직원현황 알려줘
```

### 공시 목록으로 최신 공시 추적

특정 기간의 정기·수시 공시 이력을 확인하고 접수번호(rcept_no)를 확보합니다.

```
삼성전자 2024년 공시 목록 알려줘
```

### 사업보고서 본문 텍스트 추출

"사업의 내용" 같은 서술형 섹션을 텍스트로 바로 추출합니다. 제안서·경쟁사 분석에 바로 활용 가능합니다.

```
삼성전자 최신 사업보고서 사업의 내용 섹션 추출해줘
```

### 다기업 재무 비교 (신규)

여러 기업의 매출·영업이익을 한 번에 비교합니다. 경쟁사 분석, 시장 조사 보고서에 바로 사용할 수 있습니다.

```
삼성전자, LG전자, SK하이닉스 2024년 매출액과 영업이익 비교해줘
```

### 분기·반기 최신 데이터 (신규)

사업보고서(연 1회)보다 최신 데이터가 필요할 때 `--prefer latest` 옵션으로 분기·반기 보고서까지 자동 탐색합니다.

```
SK하이닉스 가장 최신 재무 데이터 조회해줘  (분기·반기 포함)
```

### CSV 내보내기 — 엑셀에서 바로 열기 (신규)

모든 커맨드에서 `--format csv`를 사용하면 엑셀에서 바로 열 수 있는 UTF-8 BOM CSV를 출력합니다.

```
삼성전자 재무제표를 엑셀 파일로 저장해줘
```

## 출력 옵션

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| `--format json` | 구조화된 JSON (기본) | LLM이 파싱해 후속 분석 |
| `--format table` | 사람이 읽기 좋은 표 | 사용자가 바로 확인 |
| `--format csv` | UTF-8 BOM CSV | 엑셀 저장, 스프레드시트 활용 |

> **참고:** `--format`은 서브커맨드 **앞에** 위치해야 합니다.
> ```bash
> # 올바른 사용법
> python3 scripts/collect_company.py --format table search --name "삼성전자"
> python3 scripts/collect_company.py --format csv finance --corp-code 00126380 --year 2024 > finance.csv
>
> # 잘못된 사용법 (오류 발생)
> python3 scripts/collect_company.py search --name "삼성전자" --format table  ❌
> ```
| `--name` | 회사명으로 검색 | 기업코드를 모를 때 진입점 |
| `--year` | 사업연도 지정 | 재무·직원 데이터 조회 필수 |
| `--corp-code` | 8자리 DART 코드 직접 전달 | search 단계를 건너뛸 때 |
| `--prefer` | `annual`(기본)=사업보고서, `latest`=분기·반기 포함 | 최신 분기 데이터 필요 시 |

## 팁

- **공식 법인명을 쓰세요**: "카카오" 대신 "주식회사 카카오", "네이버" 대신 "네이버주식회사"로 조회하면 검색 정확도가 높아집니다. 축약형으로 조회했을 때 동명 계열사가 섞이면 `search`로 먼저 확인하세요.
- **`profile` 서브커맨드 우선 사용**: `info`·`finance`·`employees`를 각각 실행하는 대신 `profile --name` 한 번으로 통합 조회합니다. API 호출 횟수를 아낄 수 있고 결과 구조도 한눈에 정리됩니다.
- **`--year` 없이도 재무·사업보고서 조회 가능**: `finance`·`business` 커맨드에서 `--year`나 `--rcept-no`를 생략하면 최근 95일 내 가장 최신 사업보고서를 자동으로 찾아 조회합니다. 연도를 모를 때 편리합니다.
- **분기 데이터가 필요하면 `--prefer latest`**: 사업보고서는 보통 3월 말에 공시됩니다. 3분기(11월) 데이터가 필요하면 `finance --prefer latest`로 조회하면 됩니다. stderr에 어떤 보고서가 채택됐는지 표시됩니다.
- **경쟁사 비교는 `compare`**: 여러 기업을 한 줄로 비교할 때는 `compare --names "회사1,회사2" --year 2024`를 사용하세요. `--format csv`로 엑셀 파일로 바로 내보낼 수 있습니다.
- ⚠️ **상장사·공시기업만 조회 가능**: DART는 금융감독원 전자공시 시스템입니다. 비상장·미공시 스타트업·개인사업자는 조회되지 않습니다.
- ⚠️ **DART_API_KEY가 없으면 실행 안 됨**: opendart.fss.or.kr에서 발급받은 40자리 키가 반드시 필요합니다. 환경변수 또는 `--api-key` 인자로 전달해야 합니다.
- ⚠️ **하루 조회 한도**: DART는 키당 일 20,000건 조회 제한이 있습니다. 한도 초과 시 자동 재시도하지만 대량 분석은 배치 간격을 두세요.
- ⚠️ **권한 오류 메시지**: "활용신청 URL: ..." 표시가 보이면 [https://opendart.fss.or.kr](https://opendart.fss.or.kr)에서 키 활용 상태를 확인하세요. 즉시 발급된 키여도 일시 중지(011)나 미반영 상태일 수 있습니다.
