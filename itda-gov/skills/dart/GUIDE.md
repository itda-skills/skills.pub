# dart 사용 가이드

금융감독원 DART 전자공시시스템에서 기업 정보를 조회합니다.
**Claude에게 자연어로 부탁하면 됩니다** — "삼성전자 재무제표 알려줘", "경쟁사 직원수 비교해줘"처럼요. 터미널 명령을 직접 입력할 필요는 없습니다.

---

## 사전 준비 — API 키 발급

DART 스킬은 금융감독원 DART API 키가 필요합니다.

1. [금융감독원 DART](https://opendart.fss.or.kr)에서 회원가입
2. **오픈 API → 인증키 신청/관리** → 사용 목적 입력 후 신청 → **40자리 키 즉시 발급** (별도 승인 대기 없음)
3. 발급받은 키를 **Claude Desktop 지침**에 추가합니다 — Claude Desktop → **설정 → 일반 → "Claude 지침"** 에 아래 한 줄을 붙여넣고 저장하세요.

```dotenv
DART_API_KEY=발급받은_40자리_키
```

한 번 넣으면 이후 모든 대화에서 Claude가 자동으로 사용합니다 — `.env` 같은 파일을 직접 만들 필요가 없습니다.

> 개발자라면 작업 폴더 루트의 `.env`에 같은 한 줄을 넣어도 됩니다.

### 첫 호출이 실패할 때

- 키가 40자리 영숫자인지, 앞뒤에 공백은 없는지 확인하세요.
- 발급 직후에는 수 분간 일시적으로 미반영될 수 있습니다. 잠시 후 다시 요청하세요.
- "활용신청 URL: ..." 안내가 보이면 [https://opendart.fss.or.kr](https://opendart.fss.or.kr)에서 키 활용 상태를 확인하세요.

## 활용 시나리오

### 경쟁사·고객사 종합 프로필

회사명 하나로 기업개황·재무제표·직원현황까지 한 번에 조회합니다.

```
카카오 기업 정보 정리해줘
```

### 연도별 재무 추이 비교

경쟁사의 매출·영업이익 추이를 연도별로 비교해 시장 분석 보고서를 만듭니다.

```
카카오 2022년, 2023년, 2024년 매출 비교해줘
```

### 다기업 재무 비교

여러 기업의 주요 계정을 한 번에 비교하고 원하는 경우 영업이익률·순이익률까지 함께 확인합니다.

```
삼성전자, LG전자, SK하이닉스 2024년 매출액과 영업이익 비교해줘
```

### 분기·최신 데이터

사업보고서(연 1회)보다 최신 데이터가 필요할 때 분기·반기 보고서까지 포함해 자동 탐색할 수 있습니다.

```
SK하이닉스 가장 최신 재무 데이터 조회해줘
```

### 사업보고서 본문 추출

"사업의 내용" 같은 서술형 섹션을 텍스트로 바로 추출합니다.

```
삼성전자 최신 사업보고서 사업의 내용 섹션 추출해줘
```

### 공시 목록 추적

특정 기간의 공시 이력을 확인합니다.

```
삼성전자 2024년 공시 목록 알려줘
```

### 결과를 엑셀로 저장

조회 결과를 엑셀에서 바로 열 수 있는 CSV 파일로 내보냅니다.

```
삼성전자 2024년 재무제표를 CSV 파일로 저장해줘
삼성전자, LG전자 비교 결과를 엑셀용 파일로 만들어줘
```

## 출력 형식 지정

출력 형식을 자연어로 함께 말하면 그에 맞게 정리해 줍니다.

| 원하는 형식 | 이렇게 말하세요 |
|-------------|-----------------|
| 사람이 읽기 좋은 표 | "표로 보여줘" |
| 엑셀 파일 | "CSV 파일로 저장해줘" |
| 전기 비교 포함 | "전기 수치와 함께 보여줘" |
| 증감률도 함께 | "전기 대비 증감률도 같이 보여줘" |

## 팁

- **공식 법인명을 쓰세요**: "카카오" 대신 "주식회사 카카오", "네이버" 대신 "네이버주식회사"처럼 전체 법인명으로 조회하면 정확도가 높아집니다. 축약형으로 조회했을 때 동명 계열사가 섞이면 "DART에서 '카카오' 들어간 회사 찾아줘"로 먼저 목록을 확인하세요.
- **종합 조회를 우선 요청하세요**: 기업개황·재무·직원 정보를 따로 요청하는 대신 "○○ 기업 정보 정리해줘"처럼 통합 조회로 시작하면 한 번에 볼 수 있습니다.
- **연도를 몰라도 됩니다**: 연도를 따로 말하지 않으면 최근 사업보고서를 자동으로 찾아 조회합니다.
- **분기 데이터가 필요하면**: 사업보고서는 보통 3월 말에 공시됩니다. 더 최신 데이터가 필요하면 "가장 최신 분기 데이터로 알려줘"라고 말하세요.
- **여러 기업 비교 시 계정명도 말해주세요**: "삼성전자, LG전자, SK하이닉스 2024년 매출액, 영업이익, 자산총계 비교해줘"처럼 원하는 계정을 함께 말하면 더 정확하게 비교해 줍니다.

## 안 될 때

| 증상 | 원인 / 해결 |
|------|-------------|
| "API 키가 없다"는 안내 | **Claude 지침**(또는 `.env`)에 `DART_API_KEY`가 없거나 비어 있음. 위 "사전 준비" 확인 |
| "기업을 찾을 수 없습니다" | 회사명 불일치. 공식 법인명 전체로 재검색하거나 "DART에서 ○○ 검색해줘"로 목록 확인 |
| "재무 데이터가 없습니다" | 해당 연도 미공시. 이전 연도로 다시 요청 |
| 권한 오류 (`010`/`011`/`012`) | [https://opendart.fss.or.kr](https://opendart.fss.or.kr)에서 키 활용 상태 확인 |
| 요청 한도 초과 (`020`) | 하루 20,000건 한도 초과. 다음 날 다시 시도 |
| 시스템 점검 (`800`) | 잠시 후 재시도 |

---

## 주의사항

- ⚠️ **상장사·공시 기업만 조회 가능합니다.** DART는 금융감독원 전자공시 시스템입니다. 비상장·미공시 스타트업·개인사업자는 조회되지 않습니다.
- ⚠️ **API 키가 없으면 조회할 수 없습니다.** [opendart.fss.or.kr](https://opendart.fss.or.kr)에서 발급받은 40자리 키를 **Claude 지침**에 반드시 추가해야 합니다(위 "사전 준비").
- ⚠️ **하루 조회 한도가 있습니다.** 키당 일 20,000건 한도입니다. 대량 분석 시 여러 세션에 나눠 조회하세요.
