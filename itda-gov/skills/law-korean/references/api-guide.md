# 법제처 Open API (DRF) 가이드

법제처 데이터 개방 서비스(DRF)의 두 핵심 엔드포인트 상세 문서입니다.

## 공통 파라미터

| 파라미터 | 설명 | 예시 |
|----------|------|------|
| `OC` | 사용자 ID (이메일 또는 test) | `test` |
| `target` | 검색 대상 | `law` (법령) |
| `type` | 응답 형식 | `XML` |

---

## 1. 법령 검색 API (lawSearch)

법령명 또는 본문 키워드로 법령 목록을 검색합니다.

### 엔드포인트

```
GET http://www.law.go.kr/DRF/lawSearch.do
```

### 요청 파라미터

| 파라미터 | 필수 | 설명 | 기본값 |
|----------|------|------|--------|
| `OC` | 필수 | 사용자 ID | — |
| `target` | 필수 | `law` 고정 | — |
| `type` | 필수 | `XML` | — |
| `query` | 필수 | 검색어 | — |
| `search` | 선택 | `1`: 법령명 검색, `2`: 본문 검색 | `1` |
| `display` | 선택 | 결과 수 (1~100) | `20` |
| `page` | 선택 | 페이지 번호 | `1` |

### 요청 예시

```
# 법령명으로 검색
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=law&type=XML&query=근로기준법&search=1&display=10&page=1

# 본문으로 검색
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=law&type=XML&query=연차유급휴가&search=2&display=10&page=1
```

### 응답 XML 구조

```xml
<?xml version="1.0" encoding="UTF-8"?>
<LawSearch>
  <law>
    <법령명한글>근로기준법</법령명한글>
    <법령명영어></법령명영어>
    <법령구분명>법률</법령구분명>
    <소관부처명>고용노동부</소관부처명>
    <시행일자>20240101</시행일자>
    <공포일자>20231027</공포일자>
    <공포번호>19765</공포번호>
    <법령ID>9682</법령ID>
    <법령일련번호>4401</법령일련번호>
  </law>
  <law>
    <!-- 추가 결과 -->
  </law>
</LawSearch>
```

### 응답 필드 설명

| 필드명 | 설명 | 예시 |
|--------|------|------|
| `법령명한글` | 법령명 (한글) | 근로기준법 |
| `법령구분명` | 법령 종류 | 법률, 대통령령, 부령 |
| `소관부처명` | 담당 부처 | 고용노동부 |
| `시행일자` | 시행 날짜 (YYYYMMDD) | 20240101 |
| `법령ID` | 법령 고유 ID | 9682 |

## 2. 법령 본문 조회 API (lawService)

법령의 전체 조문 또는 특정 조문 원문을 조회합니다.

### 엔드포인트

```
GET http://www.law.go.kr/DRF/lawService.do
```

### 요청 파라미터

| 파라미터 | 필수 | 설명 | 예시 |
|----------|------|------|------|
| `OC` | 필수 | 사용자 ID | `test` |
| `target` | 필수 | `law` 고정 | `law` |
| `type` | 필수 | `XML` | `XML` |
| `ID` | 필수 | 법령 ID | `9682` |
| `JO` | 선택 | 조문 번호 (6자리) | `000060` |

### JO 파라미터 인코딩 규칙

| 조문 형태 | JO 값 | 계산 방법 |
|-----------|-------|-----------|
| 제60조 (일반) | `000060` | `str(60).zfill(6)` |
| 제76조의2 (가지조문) | `007602` | `str(76).zfill(4) + str(2).zfill(2)` |
| 제14조의3 (가지조문) | `001403` | `str(14).zfill(4) + str(3).zfill(2)` |
| 제1조 (일반) | `000001` | `str(1).zfill(6)` |

### 요청 예시

```
# 법령 전체 조문 조회
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&type=XML&ID=9682

# 특정 조문(제60조) 조회
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&type=XML&ID=9682&JO=000060

# 가지조문(제76조의2) 조회
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&type=XML&ID=9682&JO=007602
```

### 응답 XML 구조 (전체 조문)

```xml
<?xml version="1.0" encoding="UTF-8"?>
<법령>
  <기본정보>
    <법령명_한글>근로기준법</법령명_한글>
    <법령명_영어>Labor Standards Act</법령명_영어>
    <법령구분>법률</법령구분>
    <소관부처>고용노동부</소관부처>
    <시행일자>20240101</시행일자>
  </기본정보>
  <조문>
    <조문단위>
      <조문번호>60</조문번호>
      <조문제목>연차유급휴가</조문제목>
      <조문내용>① 사용자는 1년간 80퍼센트 이상 출근한 근로자에게 15일의 유급휴가를 주어야 한다.
② 사용자는 계속하여 근로한 기간이 1년 미만인 근로자 또는...</조문내용>
    </조문단위>
    <조문단위>
      <조문번호>61</조문번호>
      <조문제목>연차유급휴가의 사용 촉진</조문제목>
      <조문내용>사용자가 제60조에 따른 유급휴가의 사용을 촉진하기 위하여...</조문내용>
    </조문단위>
  </조문>
</법령>
```

### 응답 필드 설명

| 필드명 | 경로 | 설명 |
|--------|------|------|
| `법령명_한글` | `기본정보/법령명_한글` | 법령 이름 |
| `조문번호` | `조문/조문단위/조문번호` | 조문 번호 (문자열) |
| `조문제목` | `조문/조문단위/조문제목` | 조문 제목 (없을 수 있음) |
| `조문내용` | `조문/조문단위/조문내용` | 조문 전문 텍스트 |

## 3. OC 파라미터 안내

OC는 사용자를 식별하는 파라미터입니다.

- **OC 등록 필수**: 법제처 Open API 포털(https://open.law.go.kr/LSO/openApi/openApiInfo.do)에서 회원가입 후 OC를 발급받아야 합니다.
- **등록 후 발급받은 OC**를 환경변수(`LAW_API_OC`) 또는 `.env` 파일에 설정하여 사용합니다.

```bash
# 환경변수 설정 예시
export LAW_API_OC="your@email.com"
```

---

## 4. 에러 처리

| HTTP 상태 / XML | 원인 | 대응 |
|----------------|------|------|
| URLError / timeout | 네트워크 오류 | 10초 타임아웃, 사용자에게 오류 메시지 |
| 빈 `<LawSearch>` | 검색 결과 없음 | 다른 키워드 안내 |
| XML ParseError | 응답 형식 변경 | LawAPIError 발생, 오류 로그 |
