## 빠른 시작

법제처 Open API로 한국 법령·판례·행정규칙·자치법규를 조회해 조문 원문이나 판례를 그대로 인용할 수 있게 만들어 주는 스킬입니다.

```
근로기준법 제60조 보여줘
```

```
부당해고 대법원 판례 마크다운으로 찾아줘
```

```
화관법 조문 목록 알려줘
```

이렇게 요청하면 스킬이 법령명/판례/행정규칙을 식별해 `search_*`와 `get_*` 스크립트를 연결 실행하고, 조문 원문 마크다운과 검색 결과 테이블을 반환합니다.

## 활용 시나리오

### 민원 답변 — 조문 원문 인용

민원 질의에 법령 원문을 정확히 인용해 답변 초안을 만들 때 사용합니다.

```
국민건강보험법에서 본인부담금 관련 조문 뽑아서 마크다운으로 정리해줘
```

`search_law`로 법령을 찾고, `get_law --article`로 해당 조문만 잘라 마크다운을 반환합니다. 조문 번호를 수작업으로 외울 필요 없이 키워드만으로도 정확한 조문을 가져옵니다.

### 감사 대응 — 근거 법령 빠른 확인

감사 지적 사항의 법적 근거를 단시간에 파악할 때 사용합니다.

```
산안법 조문 목록 테이블로 보여줘
```

약어(`산안법` → `산업안전보건법`)가 자동 인식되며, `--toc` 옵션으로 조문 제목만 훑어보고 관련 조문만 추려서 추가 조회할 수 있습니다.

### 판례·행정규칙 리서치

부당해고 판례나 특정 부처 고시를 필터링해 찾을 때 사용합니다.

```
2023년 이후 대법원 손해배상 판례 찾아줘
```

`search_prec`에 `--court 대법원`, `--date-from 20230101`을 조합하면 법원 범위와 선고일자를 좁혀 판례를 추출합니다. 행정규칙은 `search_admrul --kind 고시`로 훈령/예규/고시 종류별 필터가 가능합니다.

## 핵심 CLI 옵션

| 옵션 | 설명 | 사용 시점 |
|------|------|-----------|
| `--query` | 법령명·키워드 검색 (약어 자동 인식) | 모든 `search_*` 스크립트의 출발점 |
| `--article` | 조문 번호 지정 (`60`, `76조의2` 형식) | 특정 조문만 원문으로 뽑을 때 |
| `--toc` | 조문 제목 목록만 출력 | 감사 대응·조문 식별에 빠른 훑어보기 |
| `--format` | `table` / `json` / `md` 선택 | 인용은 `md`, 파이프라인은 `json`, 터미널은 `table` |

## 팁

- **약어 자동 인식**: `화관법`, `산안법`, `개인정보법` 같은 70여 개 약어가 정식 법령명으로 자동 치환됩니다. 정확한 명칭을 외우지 않아도 됩니다.
- **가지조문 표기법**: `76조의2`, `14조의3` 같은 가지조문은 `--article "76조의2"` 형식 그대로 넣으면 됩니다. 내부적으로 JO 파라미터 6자리(`007602`)로 변환됩니다.
- **MST 흐름**: 법령용어 연계 조회는 `search_lstrm`으로 MST 확인 → `get_lstrm_rlt` 또는 `get_lstrm_rlt_jo` 순서로 진행합니다. 역방향(조문 → 용어)은 `get_jo_rlt_lstrm --id {법령ID} --jo {JO}`를 사용합니다.

## 제한사항

- ⚠️ **OC 발급 필수**: 법제처 Open API는 OC(사용자 ID) 없이 호출이 불가합니다. 발급 전에는 모든 조회가 실패합니다. 법제처 Open API 포털(https://open.law.go.kr/LSO/openApi/openApiInfo.do)에서 신청해 CLAUDE.md, settings.json, `.env`, `--oc` 인자 중 한 곳에 등록해야 합니다.
- ⚠️ **판례 원문 30,000자 트런케이션**: `get_prec`는 토큰 절약을 위해 판례내용을 30,000자에서 잘라냅니다. 전문이 필요하면 법제처 웹사이트에서 직접 확인하거나, `--summary-only`로 판시사항+판결요지만 받아 핵심만 인용하세요.
- ⚠️ **API Rate limit · 네트워크 의존**: 법제처 API는 호출 빈도 제한이 있고 서버 장애 시 조회가 실패합니다. 기본 파일 캐시(TTL, 100개 항목 제한)가 동작하지만, 최신 개정본이 필요하면 `--no-cache`로 우회하세요.

## 지원하지 않는 케이스

- **외국법·국제조약**: 이 스킬은 한국 법제처 API 전용입니다. 미국·EU·일본 법령이나 국제조약은 조회 불가입니다.
- **법률 상담·해석 제공**: 법령 원문과 판례를 조회하는 도구이며 법률 자문이나 해석을 제공하지 않습니다. 변호사 상담을 대체할 수 없습니다.
