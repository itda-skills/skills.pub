## 1.3 인증키 발급 및 서비스 신청

### 1.3.1 활용신청

활용신청 양식을 활용용도와 사용목적 등을 작성한 뒤 등록버튼을 클릭한다. [약관보기]를 클릭하면 통계정보 활용약관 팝업이 호출된다.

**활용신청 양식 항목**:

| 항목 | 입력 내용 |
|------|----------|
| 신청자명 | (필수) 김홍영 (예시) |
| 전화번호 | (필수) — |
| 서비스 유형 | REST |
| 심의여부 | **자동승인** |
| 활용용도 | 별개발 / 연계발 / 기타 (라디오 선택) |
| 활용기간 | 인증키 만료일까지 활용 가능 |
| 상업적활용여부 | 예 / 아니오 (라디오) |
| 사업목적 | (필수) 사용목적을 반드시 작성. 활용신청서 작성 시에 활용신청 승인이 어려울 수 있습니다. (2,000자 이내) |
| 첨부파일 | 첨부파일 용량은 10MB이하로 한 개씩 첨부할 수 있습니다. [xls, doc, hwp, txt, gif, jpg, jpeg, png, bmp 파일만 가능합니다.] |
| 약관동의 | 통계정보 활용약관에 동의합니다. (체크박스) |

### 1.3.2 신청현황

활용신청이 완료되면 신청현황 단계로 넘어가며, 기본정보에서는 사용자 인증키가 발급된 것을 확인이 가능하고, 서비스정보, 활용정보를 확인할 수 있다. **신청정보 수정**을 누르면 신청정보 수정 단계로 넘어간다.

**기본정보 표시 항목**:

| 항목 | 값 (예시) |
|------|----------|
| 서비스유형 | REST |
| 활용기간 | 인증키 만료일까지 활용 가능 |
| 심의여부 | 자동승인 |
| 상업적활용유무 | 비허용 |
| 사용자 인증키 | NJYIY2lSWT7XNTM0NDg0... (마스킹) |

**활용정보 표시 항목**:

| 항목 | 값 (예시) |
|------|----------|
| 활용용도 | 기타 (테스트용) |
| 활용목적 | 테스트용 |
| 첨부파일 | 첨부파일이 없습니다. |

신청정보 수정 버튼으로 수정 가능.

## 1.4 에러메시지

KOSIS OpenAPI 에러 메시지는 아래와 같다.

### 1.4.1 오류메시지 형식

오류는 XML 형식으로 반환된다.

```xml
<?xml version="1.0" encoding="UTF-8" ?>
<error>
  <err>50</err>
  <errMsg>서버오류가 발생하였습니다.</errMsg>
</error>
```

### 1.4.2 오류메시지 유형

| 오류코드 | 오류메시지 | 조치방법 |
|---------|----------|---------|
| 10 | 인증키 누락 | 인증키 확인 |
| 11 | 인증키 기간만료 | 인증키 기간 연장 |
| 20 | 필수요청변수 누락 | 필수요청변수 확인 |
| 21 | 잘못된 요청변수 | 요청변수 확인 |
| 30 | 조회결과 없음 | 조회조건 확인 |
| 31 | 조회결과 초과 | 호출건수 조정 |
| 40 | 호출가능건수 제한 | 관리자에게 문의 |
| 41 | 호출가능ROW수 제한 | 관리자에게 문의 |
| 42 | 사용자별 이용 제한 | 관리자에게 문의 |
| 50 | 서버오류 | 관리자에게 문의 |

### 분류 의미

- **인증키 관련 (10, 11)**: 키가 누락되었거나 만료. KOSIS 신청현황에서 키 확인/갱신 필요.
- **요청 검증 (20, 21)**: 클라이언트의 요청 파라미터 자체 문제. 코드 수정 필요.
- **데이터 결과 (30, 31)**: 정상 호출이지만 결과가 0건 또는 초과. 조회 조건 조정.
- **호출 제한 (40, 41, 42)**: 일/시간/사용자별 한도 초과. KOSIS 관리자 문의 또는 호출 빈도 조정.
- **서버 오류 (50)**: KOSIS 측 서버 이슈. 잠시 후 재시도.
