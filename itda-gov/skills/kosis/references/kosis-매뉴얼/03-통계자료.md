# 2.2 통계자료 (statisticsData.do / statisticsParameterData.do)

출처: 통계청 발간 "KOSIS 공유서비스(OpenAPI) 개발가이드 v1.0" 2.2 (페이지 33~101)
원본 PDF: [openApi_manual_v1.0.pdf](openApi_manual_v1.0.pdf)
추출일: 2026-04-28 (1차 발췌, 페이지 33~50 부분)

## 2.2.1 특징

- 통계표의 수치자료를 **시계열**(단일계열, 여러시점) 또는 **횡단면**(다중계열, 단일시점)으로 제공
- 통계표의 수록정보, 분류/항목, 출처, 단위 등 **메타정보 제공**
- 자료 제공형태: **SDMX (DSD, Generic, StructureSpecific), JSON, XML**

### 두 가지 호출 방식

#### URL 선택 방식 (URL 생성)

별도 자료등록 없이 바로 필요한 통계표/항목/분류를 선택하고, URL에 테이블ID/항목/분류 파라미터를 제공한다. 이 값을 이용자가 동적으로 변경 가능. 미리 자료를 등록하는 과정이 불필요하다. (등록한 자료의 통계표 메타정보가 변경될 경우 확인 불가능.)

#### URL 자료등록 방식 (자료등록 → URL 생성)

생성내역의 통계표 및 URL이 저장되어 등록한 자료의 통계표 메타정보가 변경될 경우 이용자가 확인 가능. 다만 테이블/항목/분류 등 정보값을 이용자가 동적으로 변경할 수 없다.

### URL 길이 단축 키워드

URL 길이를 줄이기 위해 항목/분류 파라미터에 다음 키워드 적용 가능:

| 키워드 | 의미 | 예시 |
|--------|------|------|
| `all` | 항목과 분류별 전체선택 | `itmId=all`, `objL1=all` |
| `*` | 하위레벨 전체 선택 | `objL1=11*` (행정구역별에서 서울(11)의 하위레벨 전체) |
| `+` | 항목/분류값 추가 | `objL1=11+21` (서울+부산) |

> **시스템 부하를 줄이기 위해 통계자료 요청은 4만셀 이하로 제한**

## 2.2.2 서비스 활용

### 2.2.2.1 URL 생성 (URL 선택 방식)

개발가이드 > 통계자료 > URL생성

- 작성기관, 통계조사명, 통계표명 등을 입력하여 사용하고자 하는 자료를 조회한 뒤, 조회결과에서 생성할 통계표를 선택하면 하단에 URL생성 조건 설정이 출력된다.
- 통계표조회 버튼을 클릭하면 해당 자료의 통계표를 볼 수 있다.
- URL생성 조건 설정에서 분류/항목선택, 출력형태 설정, 조회기간 설정을 입력 후 URL보기 버튼을 클릭하면 생성된 URL이 조회되고, URL복사 버튼을 누르면 생성된 URL이 클립보드에 복사된다.

URL생성 화면 항목:

- **통계표 선택**: 작성기관, 통계조사명, 통계표명, 통계표ID, 분류, 주기, 분류값/항목
- **URL생성 조건 설정**:
  - 통계표명, 통계표ID
  - 분류/항목선택 (분류 + 항목 선택)
  - 출력형태 설정 (데이터 포멧: JSON / SDMX, 사용자 인증키 입력)
  - 조회기간 설정 (기간설정 / 최근시점기준)
  - 최근시점부터 수기 N년 / 최근 N개시점 / 수록시점간격 (옵션 설명: 최근시점은 신청한 시점이 있거나 정상으로 인쇄 시점인 시 기준입니다 산정을 합니다. 즉, 2013, 2017, 2025 등 3개시점 간격 추출되지 않습니다.)
  - 응답필드 설정 (선택사항):
    - 기관코드, 통계표ID, 통계표명, 분류값 ID, 분류값명, 분류영문명, 분류값영문명, 분류영문명, 분류영문명, 항목 ID, 항목명, 항목영문명, 단위 ID, 단위명, 단위영문명, 수록주기, 수록시점, 자료생성일

### 2.2.2.2 자료등록 URL 생성

#### 2.2.2.2.1 자료 등록

개발가이드 > 통계자료 > URL생성 > 자료등록

- '작성기관', '통계조사명', '통계표명' 등을 입력하여 사용하고자 하는 자료를 조회한 뒤 조회결과에서 등록할 자료의 사용여부 항목을 선택하고 통계표 등록 버튼을 누른다.
- 통계표조회 버튼을 클릭하면 해당 자료의 통계표를 볼 수 있다.

#### 2.2.2.2.2 등록된 자료

마이페이지 > 등록한 자료 > 통계자료 > 등록된 자료

- 자료등록의 통계표 등록을 마치면, 이용자가 등록한 자료들의 목록이 나타나며, 등록된 자료 중 URL생성을 원하는 자료의 URL생성 버튼을 누른다.
- 통계표조회 버튼을 클릭하면 해당 자료의 통계표를 볼 수 있다.
- 등록된 자료 삭제: 마이페이지 > 등록한 자료 > 통계자료 > 등록된 자료 > 삭제. 버튼을 클릭한 후 하단에 조회되는 사용자 생성 URL 목록을 삭제할 수 있다.

#### 2.2.2.2.3 URL생성

마이페이지 > 등록한 자료 > 통계자료 > 등록된 자료 > 생성

- URL생성 단계에서는 URL생성 조건 설정의 '활용 자료명', '분류/항목선택'을 입력 후 URL생성 버튼을 누르면 URL생성 상세조건 화면으로 이동 후 URL이 하단에 생성된다.
- URL생성 상세조건 화면에서 상세설정 후 URL보기, 결과값보기 버튼을 클릭하여 페이지 하단에서 결과를 확인할 수 있고, URL복사 버튼을 누르면 생성된 URL이 클립보드에 복사된다.

## 2.2.3 활용방법

### 2.2.3.1 JSON

**호출 URL**: `http://kosis.kr/openapi/statisticsData.do`

> 코드(`kosis_api.py`) 사용 endpoint는 `https://kosis.kr/openapi/Param/statisticsParameterData.do`. 이는 통계표선택 방식의 별도 진입점으로, 매뉴얼의 statisticsData.do와 동일 응답 스키마를 사용한다.

#### 입력 변수

##### 자료등록 방법 (userStatsId 사용)

| 항목명(영문) | 변수타입 | 항목설명 | 비고 |
|------------|---------|---------|------|
| apiKey | String | 발급된 인증키 | 필수 |
| userStatsId | String | 사용자 등록 통계표 | 필수 |
| prdSe | String | 수록주기 | 필수 |
| startPrdDe (시점기준) | String | 시작수록시점 | 선택 (시점기준 또는 최신자료기준 택1) |
| endPrdDe (시점기준) | String | 종료수록시점 | 선택 |
| newEstPrdCnt (최신자료기준) | String | 최근수록시점 개수 | 선택 |
| prdInterval (최신자료기준) | String | 수록시점 간격 | 선택 |
| format | String | 결과 유형(json, sdmx) | 필수 |
| content | String | 헤더 유형(html, json) | 선택 |

##### 통계표선택 방법 (orgId + tblId + objL + itmId 사용)

| 항목명(영문) | 변수타입 | 항목설명 | 비고 |
|------------|---------|---------|------|
| apiKey | String | 발급된 인증키 | 필수 |
| orgId | String | 기관 ID | 필수 |
| tblId | String | 통계표 ID | 필수 |
| objL1 | String | 분류1(첫번째 분류코드) | 필수 |
| objL2 ~ objL8 | String | 분류2(두번째 분류코드) ~ 분류8(여덟번째 분류코드) | 선택 |
| itmId | String | 항목 | 필수 |
| prdSe | String | 수록주기 | 필수 |
| startPrdDe (시점기준) | String | 시작수록시점 | 선택 (시점기준 또는 최신자료기준 택1) |
| endPrdDe (시점기준) | String | 종료수록시점 | 선택 |
| newEstPrdCnt (최신자료기준) | String | 최근수록시점 개수 | 선택 |
| prdInterval (최신자료기준) | String | 수록시점 간격 | 선택 |
| format | String | 결과 유형(json, sdmx) | 필수 |

호출 시 method=getList 쿼리 파라미터 추가 필요 (예제 참조).

#### 출력 변수 (JSON, 통계표선택 방식)

| 항목명(영문) | 항목설명 | 형식 | 비고 |
|------------|---------|------|------|
| ORG_ID | 기관코드 | VARCHAR2(40) | |
| TBL_ID | 통계표ID | VARCHAR2(40) | |
| TBL_NM | 통계표명 | VARCHAR2(300) | |
| C1 ~ C8 | 분류값 ID1 ~ 분류값 ID8 | VARCHAR2(40) | 2~8 분류값은 없을 경우 생략 |
| C1_OBJ_NM ~ C8_OBJ_NM | 분류명1 ~ 분류명8 | VARCHAR2(3000) | 2~8 분류값은 없을 경우 생략 |
| C1_OBJ_NM_ENG ~ C8_OBJ_NM_ENG | 분류 영문명1 ~ 분류 영문명8 | VARCHAR2(3000) | |
| C1_NM ~ C8_NM | 분류값 명1 ~ 분류값 명8 | VARCHAR2(3000) | |
| C1_NM_ENG ~ C8_NM_ENG | 분류값 영문명1 ~ 분류값 영문명8 | VARCHAR2(3000) | |
| ITM_ID | 항목 ID | VARCHAR2(40) | |
| ITM_NM | 항목명 | VARCHAR2(3000) | |
| ITM_NM_ENG | 항목영문명 | VARCHAR2(3000) | |
| UNIT_ID | 단위ID | VARCHAR2(40) | |
| UNIT_NM | 단위명 | VARCHAR2(1000) | |
| UNIT_NM_ENG | 단위영문명 | VARCHAR2(1000) | |
| PRD_SE | 수록주기 | VARCHAR2(20) | |
| PRD_DE | 수록시점 | VARCHAR2(8) | |
| DT | 수치값 | VARCHAR2(100) | |
| LST_CHN_DE | 최종수정일 | VARCHAR2(8) | |

### 예제 호출 패턴 (페이지 41~50)

#### 자료등록 방식 (userStatsId)

```
URL = "https://kosis.kr/openapi/statisticsData.do"
parameters = {
  "method": "getList",
  "apiKey": <KEY>,
  "format": "json",
  "jsonVD": "Y",
  "userStatsId": "openapisample/101/DT_1IN1502/2/1/20191106094026_1",  # 시계열 조회 키값
  "prdSe": "Y",
  "newEstPrdCnt": "3",   # 최근수록시점 개수
  "prdInterval": "1",    # 수록시점 간격
}
```

#### 통계표선택 방식 (orgId + tblId)

```
URL = "https://kosis.kr/openapi/Param/statisticsParameterData.do"  # 별도 진입점 (코드 사용)
parameters = {
  "method": "getList",
  "apiKey": <KEY>,
  "format": "json",
  "jsonVD": "Y",
  "orgId": "101",
  "tblId": "DT_1B040A3",
  "objL1": "ALL",   # 행정구역(시군구별) 전체
  "itmId": "ALL",
  "prdSe": "M",
  "startPrdDe": "202308",
  "endPrdDe": "202309",
}
```

### 출력 예시 (페이지 38, 인구수 통계표 DT_1B040A3)

```
< 데이터 보기 >
행정구역(시군구)별 종전구수 (2023.09)
전국                51370001
```

생성된 URL 형식:
```
https://kosis.kr/openapi/Param/statisticsParameterData.do?method=getList&apiKey=<KEY>
&itmId=T001+...&objL1=ALL&objL2=...&objL3=...&objL4=...&objL5=...&objL6=...&objL7=...&objL8=
&format=json&jsonVD=Y&prdSe=M&newEstPrdCnt=3&orgId=101&tblId=DT_1B040A3
```
