# 2.1 통계목록 (statisticsList.do)

출처: 통계청 발간 "KOSIS 공유서비스(OpenAPI) 개발가이드 v1.0" 2.1 (페이지 17~32)
원본 PDF: [openApi_manual_v1.0.pdf](openApi_manual_v1.0.pdf)
추출일: 2026-04-28

## 2.1.1 특징

- 통계표의 목록구성 정보 제공을 위한 OpenAPI
- 통계표 서비스 목록으로서 레벨 형태로 구성
- 자료 제공 형태: SDMX, JSON

## 2.1.2 서비스 활용

### 2.1.2.1 URL 생성

개발가이드 > 통계목록 > URL생성

- 목록구분, 목록조회, 결과유형을 입력한 뒤 'URL복사', '결과값보기' 중 원하는 서비스에 해당하는 버튼을 누르면 결과값을 제공받을 수 있다.

**URL반환 조건 설정** (예시):

| 항목 | 값 |
|------|---|
| 목록 구분 | 국내통계 주제별 |
| 목록 조회 | 목록ID = A, 목록명 = 인구 |
| 결과유형 | SDMX |

생성된 URL 예시:
```
https://kosis.kr/openapi/statisticsList.do?method=getList&apiKey=<KEY>&vwCd=MT_ZTITLE&parentListId=A&format=sdmx&jsonVD=Y&version=v2_1
```

## 2.1.3 활용방법

### 2.1.3.1 JSON

- 호출 URL: `http://kosis.kr/openapi/statisticsList.do`

#### 입력 변수

| 항목명(영문) | 변수타입 | 항목설명 | 비고 |
|------------|---------|---------|------|
| apiKey | String | 발급된 인증키 | 필수 |
| vwCd | String | 서비스뷰 코드 (아래 9가지 중 하나) | 필수 |
| parentListId | String | 시작목록 ID | 필수 |
| format | String | 결과 유형(json, sdmx) | 필수 |
| content | String | 헤더 유형(html, json) | 선택 |

**vwCd 서비스뷰 코드 (9종)**:

| 코드 | 의미 |
|------|------|
| MT_ZTITLE | 주제별 통계 |
| MT_OTITLE | 기관별 통계 |
| MT_CHOSUN_TITLE | 광복이전통계(1908~1943) |
| MT_HANKUK_TITLE | 대한민국통계연감 |
| MT_STOP_TITLE | 작성중지통계 |
| MT_ATITLE01 | 지역통계 (주제별) |
| MT_ATITLE02 | 지역통계 (기관별) |
| MT_GTITLE01 | e-지방지표(주제별) |
| MT_ETITLE | 영문KOSIS |

또한 실제 호출 시 `method=getList` 쿼리 파라미터가 추가로 필요하다 (예제 소스 참조).

#### 출력 변수 (JSON)

| 항목명(영문) | 항목설명 | 형식 |
|------------|---------|------|
| VW_CD | 서비스뷰ID | VARCHAR2(40) |
| VW_NM | 서비스뷰명 | VARCHAR2(300) |
| LIST_ID | 목록ID | VARCHAR2(40) |
| LIST_NM | 목록명 | VARCHAR2(300) |
| ORG_ID | 기관코드 | VARCHAR2(40) |
| TBL_ID | 통계표ID | VARCHAR2(40) |
| TBL_NM | 통계표명 | VARCHAR2(300) |
| STAT_ID | 통계조사ID | VARCHAR2(40) |
| SEND_DE | 최종갱신일 | VARCHAR2(8) |
| REC_TBL_SE | 추천 통계표 여부 | VARCHAR2(10) |

### 2.1.3.2 SDMX(Category)

- 호출 URL: `http://kosis.kr/openapi/statisticsList.do`

#### 입력 변수

| 항목명(영문) | 변수타입 | 항목설명 | 비고 |
|------------|---------|---------|------|
| apiKey | String | 발급된 인증키 | 필수 |
| vwCd | String | 서비스뷰 코드 (9종 동일) | 필수 |
| parentListId | String | 시작목록 ID | 필수 |
| format | String | 결과 유형(json, sdmx) | 필수 |
| version | String | 결과값 구분 | 생략시 구버전으로 데이터 출력 |

#### 출력 변수 (SDMX)

SDMX 표준 트리 구조:

```
- Header
    - Id              (서비스뷰ID)
    - Name            (서비스뷰명)
    - Prepared        (전송시간)
    - Sender
        - Id          (전송기관)
        - Name        (전송기관명)
- Structures
    - CategorySchemes
        - CategoryScheme
            - Id      (상위목록 ID)
            - Name    (상위목록 명)
            - Category
                - Id              (목록ID)
                - Name            (목록명칭)
                - Description     (기관코드_통계표ID)
                - StatId          (통계조사ID)
                - SendDe          (최종갱신일)
                - RecTblSe        (추천 통계표 여부)
```

## 예제 소스 (페이지 20~32)

각 언어별 통계목록 API 호출 예제:

- **JSP** (페이지 20~22): cross domain 우회를 위한 jsp 파라미터 패턴 + Ajax + dojo
- **JSP devGuidePop.jsp** (페이지 23): URL 패스스루 패턴
- **R** (페이지 23~24, 29~30): `library(httr) library(jsonlite)` 또는 `library(XML)` 사용. `Sys.getenv('KOSIS_TOKEN')`로 키 관리.
- **Python** (페이지 24~25, 31~32): `urllib.request.urlopen` 또는 `requests` + `BeautifulSoup`. PyQt5 GUI 예제.

호출 패턴 핵심:

```
URL = "https://kosis.kr/openapi/statisticsList.do"
parameters = {
  "method": "getList",
  "apiKey": <KEY>,
  "vwCd": "MT_ZTITLE",
  "parentListId": <ID>,
  "format": "json",
  "jsonVD": "Y",
}
```

(SDMX의 경우 추가 `version=v2_1`, `format=sdmx` 사용)
