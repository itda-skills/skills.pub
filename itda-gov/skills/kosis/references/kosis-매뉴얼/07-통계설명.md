# 2.4 통계설명 (statisticsExplData.do)

출처: 통계청 발간 "KOSIS 공유서비스(OpenAPI) 개발가이드 v1.0" 2.4 (페이지 110~136)
원본 PDF: [openApi_manual_v1.0.pdf](openApi_manual_v1.0.pdf)
추출일: 2026-04-28

## 2.4.1 특징

- 통계조사에 대한 설명자료 제공을 위한 OpenAPI
- **자료 제공형태**: XML, JSON

## 2.4.2 서비스 활용

### 2.4.2.1 URL생성

개발가이드 > 통계설명 > URL생성

URL생성 단계에서는 URL생성 조건 설정의 '통계조사', '설명항목', '설명자료'를 입력 후 URL복사, 결과값보기 버튼을 클릭하여 페이지 하단에서 결과를 확인할 수 있다.

URL 생성 조건 설정:
- **통계조사**: 작성기관 검색 후 선택 (예: 인구총조사)
- **설명항목** (체크박스 다중선택):
  - 전체 / 조사명 / 통계종류
  - 계속여부 / 법적근거 / 공표주기
  - 조사주기 / 조사체계 / 공표범위
  - 공표주기 / 연락처 / 통계활용분야 실태
  - 조사대상 범위 / 조사대상 지역 / 조사단위 및 조사대상규모
  - 적용분류 / 조사항목 / 공표항목 등 URL
  - 조사대상기간 및 조사기준시점 / 조사연혁 / 공표방법 및 URL
  - 주요 용어해설 / 자료 수집방법 / 조사연혁
  - 승인번호 / 승인일자 / 통계포함표 등
- **설명자료**: xml | json

#### 호출 예제 (Python)

```python
url = (
    "http://kosis.kr/openapi/statisticsExplData.do"
    "?method=getList&apiKey=<KEY>"
    "&format=json&jsonVD=Y"
    "&statId=1962009"
    "&metaItm=All"
)
import json, urllib.request
with urllib.request.urlopen(url) as r:
    data = json.loads(r.read().decode('utf-8'))
```

### 2.4.3.2 XML

- **호출 URL**: `http://kosis.kr/openapi/statisticsExplData.do?method=getList`

> 매뉴얼 페이지 123 표지에는 `http://kosis.kr/openApi/StatsExplain.domethod=getList`로 표기되었으나 이는 오타로 보임. 본문 예제(페이지 119, 131)는 정확한 URL `statisticsExplData.do`로 사용. 실제 호출은 정확한 URL 사용.

#### 입력 변수

| 항목명(영문) | 변수타입 | 항목설명 | 비고 |
|------------|---------|---------|------|
| apiKey | String | 발급된 인증키 | 필수 |
| statId (또는 orgId+tblId) | String | 통계조사 ID | 필수 |
| metaItm | String | 요청 항목 (28종, JSON 동일) | 필수 |
| format | String | 결과유형(JSON, SDMX) | 필수 |

#### 출력 변수 (XML)

JSON과 동일한 28개 필드, XML 노드로 응답:

```xml
<response>
  <Header>...</Header>
  <Structures>
    <statsNm>가계동향조사</statsNm>
    <statsKind>지정통계 / 조사통계</statsKind>
    <statsContinue>계속통계</statsContinue>
    <basisLaw>승인번호:10106, 승인일자:1962.12</basisLaw>
    <writingPurps>가구의 생활수준실태와 ...</writingPurps>
    ...
  </Structures>
</response>
```

각 필드 형식: VARCHAR2(4000) (writingTel은 VARCHAR2(8000)).
