## 1. KOSIS 공유서비스(OpenAPI) 개요

KOSIS 공유서비스는 외부(기관, 개인)에서 국가통계포털(KOSIS)의 통계정보를 서비스 또는 콘텐츠 개발에 활용할 수 있도록 인터페이스(API)를 제공하는 서비스이다.

### 서비스 대상 4종

| 서비스 대상 | 내용 | 제공 형태 |
|------------|------|-----------|
| 1) 통계목록 | KOSIS에서 서비스되고 있는 통계목록에 대한 정보 및 관련 통계표 정보 (국내통계 주제별, 국내통계 기관별, 광복이전통계(1908~1943), 대한민국통계연감, 작성중지통계, 지역통계(주제별), 지역통계(기관별), e-지방지표(주제별), 영문 KOSIS) | JSON, SDMX |
| 2) 통계자료 | KOSIS에서 공개되어 서비스되고 있는 약 7만여 개 통계표에 대한 수치 데이터 및 메타정보(분류, 항목, 단위 등) | JSON, SDMX |
| 3) 대용량 통계자료 | 상기 2) 통계자료와 동일한 정보를 제공하나 대량의 자료를 한 번에 제공 | XLS, SDMX |
| 4) 통계설명 | 통계조사에 대한 상세 설명자료 (조사명, 통계종류, 계속여부, 법적근거, 조사목적, 조사주기, 조사체계, 공표범위, 공표주기, 연락처) | JSON, XML |

**주의**: 통계자료와 대용량 통계자료는 국제통계 등 라이선스 제약에 해당되는 자료는 서비스 제외.

### 인증키 발급 정책

- KOSIS 공유서비스를 활용하려면 국가통계포털(KOSIS) 회원으로 가입되어야 함. 기존 KOSIS 회원은 그대로 사용 가능.
- 인증키는 회원당 **1개** 발급되며, 1개 인증키로 모든 서비스 이용 가능.
- 활용신청을 하면 모든 서비스 이용 가능.
- **활용신청 승인은 신청 후 자동 승인**되어 바로 이용 가능.

### 이용 흐름

```
로그인 → 활용신청 → 자동승인 → 신청현황 → 자료등록 → 서비스활용
```

자료등록은 통계자료, 대용량 통계자료에서 자료를 등록한 뒤 통계표를 선택하여 다시 자료등록하는 단계가 있고, 이후 URL을 생성해 호출한다.

## 1.1 제공 콘텐츠 (7종)

### 1.1.1 통계목록

통계표의 목록구성 정보 제공을 위한 OpenAPI. 통계목록 단위로 호출하고 서비스뷰(주제별, 기관별 등 9가지)별로 상위목록의 정보와 연결된 통계표명을 제공한다. 활용신청을 하면 자동으로 인증키가 발급되며 URL을 생성하여 자료를 활용할 수 있다.

서비스뷰 코드 중 **9가지** (예시):

| 서비스뷰 | 목록ID | 목록명 |
|---------|--------|--------|
| MT_ZTITLE | A1 | 인구총조사 |
| MT_ZTITLE | A2 | 인구동향조사 |
| MT_ZTITLE | A3 | 인구이동 |
| MT_ZTITLE | A4 | 추계인구·가구 |
| MT_ZTITLE | A5 | 생명표 |
| MT_ZTITLE | A6 | 주민등록인구통계 |
| MT_ZTITLE | A7 | 인구일도 |
| MT_ZTITLE | A8 | 체류외국인통계 |
| MT_ZTITLE | 110_11025 | 지방자치단체외국인주민현황 |

### 1.1.2 통계자료

통계표의 수치자료 및 메타정보(수록정보, 출처, 단위 등) 제공을 위한 OpenAPI. 메타정보는 수치자료와 같이 호출할 수도 있다. 활용신청을 하면 자동으로 인증키가 발급되며 통계자료를 등록하여 활용해야 한다.

자료 형태:
- **시계열 자료**: 한 항목을 여러 연도에 걸쳐 조회 (예: 주민등록인구 2010~2012)
- **횡단면 자료**: 한 시점에 여러 행정구역의 값을 조회 (예: 시도별 주민등록인구)

### 1.1.3 대용량 통계자료

통계표의 수치자료 및 메타정보 제공을 위한 OpenAPI로, 통계표 전체, 분류 전체(일부), 항목 전체(일부)를 선택적으로 요청한다. 데이터량이 많은 특성 상 자료제공 형태가 SDMX 외 **XLS**가 추가되며, 활용신청을 하면 자동으로 인증키가 발급되어 대용량 통계자료를 등록하여 활용해야 한다.

### 1.1.4 통계설명

통계조사에 대한 상세 설명자료 제공을 위한 OpenAPI. 통계표 또는 통계조사 단위로 호출하면 통계조사에 대한 설명자료 정보가 API로 제공된다.

활용신청을 하면 자동으로 인증키가 발급되며 URL을 생성하여 자료를 활용할 수 있다.

서비스 항목:

| 항목 | 설명 (예시: 가계동향조사) |
|------|-------|
| 조사명 | 가계동향조사 |
| 통계종류 | 지정통계 / 조사통계 |
| 계속여부 | 계속통계 |
| 법적근거 | 승인번호: 10106, 승인일자: 1962.12 |
| 조사목적 | 가구의 생활수준실태와 그 변동사항을 파악하기 위해서 가계의 수입과 지출을 조사 (이하 생략) |
| 조사주기 | 분기 |
| 조사체계 | 표본가구 → 조사원 → 지방 통계사무소(품목코드부여, 온라인입력) → 통계청 |
| 공표단위 | 전국 |
| 공표주기 | 분기 |
| 연락처 | 통계청 사회통계국 복지통계과 (전화: 042-481-2200) |

### 1.1.5 메타자료

통계자료에 대한 메타자료 제공을 위한 OpenAPI. 통계자료에 대한 통계표 명칭, 기관명칭, 수록정보, 분류/항목, 주석, 단위, 출처, 가중치, 자료갱신일에 대한 정보가 API로 제공된다.

서비스 항목:
- 통계표 명칭 (국문명/영문명)
- 기관 명칭 (국문명/영문명)
- 수록정보 (수록주기, 시작/종료 수록시점)
- 분류/항목 (분류 ID, 분류 국문명/영문명, 자료코드 ID, 자료코드 국문명/영문명, 상위 자료코드)
- 주석
- 단위 (국문명/영문명)
- 출처 (조사명, 통계표 담당부서, 담당부서 전화번호)
- 가중치
- 자료갱신일

### 1.1.6 KOSIS통합검색

통계표의 목록구성 정보 제공을 위한 OpenAPI. 통계목록 단위로 호출하고 서비스뷰(주제별, 기관별 등 13가지)별로 상위목록의 정보와 연결된 통계표명을 제공한다.

> 주의: 본 매뉴얼에서 1.1.1 통계목록은 9가지 서비스뷰, 1.1.6 KOSIS통합검색은 13가지 서비스뷰로 명시. 두 API는 별도이며 검색용 API가 더 많은 카테고리를 지원.

### 1.1.7 통계주요지표

통계표의 목록구성 정보 제공을 위한 OpenAPI. 통계목록 단위로 호출하고 서비스뷰(주제별, 기관별 등 13가지)별로 상위목록의 정보와 연결된 통계표명을 제공한다.

서비스 내용 (8개 조회 유형):

1. 지표고유번호별/지표명별 설명자료조회 — 지표ID, 지표명, 설명자료 제목, 개념, 산정방법, 출처정보
2. 지표명별 목록조회 — 지표ID, 지표명, 단위, 지역구분명, 수록주기명, 수록시작/종료시점, 수록시점개수, 종료시점+주기명
3. 고유번호별 목록조회
4. 고유번호별 지표 상세조회 — 지표ID, 지표명, 수록주기, 시점, 항목, 통계수치
5. 수록주기별 목록조회
6. 지표명별 상세조회

## 1.2 회원가입

OpenAPI를 활용하기 위해서는 KOSIS 회원이어야 한다. KOSIS 회원가입 방법 및 절차:

- 페이지 상단의 회원가입 버튼을 클릭하면 KOSIS 회원가입페이지로 이동하며 제공된 양식에 따라 회원가입을 진행할 수 있다.
- KOSIS 공유서비스는 통합 회원 계정으로 동작 (소개 및 이용방법, 활용신청, 개발가이드, 커뮤니티, 이용현황 메뉴 제공).
