# 2.7 통계주요지표 (지표 Open API)

출처: 통계청 발간 "KOSIS 공유서비스(OpenAPI) 개발가이드 v1.0" 2.7 (페이지 149~157)
원본 PDF: [openApi_manual_v1.0.pdf](openApi_manual_v1.0.pdf)
추출일: 2026-04-28

## 2.7.1 특징

- 지표 Open API 서비스를 이용하기 위한 JSON, XML 기반의 데이터형식, 요청변수, 반환되는 코드값을 제공
- 자료 제공형태: XML, JSON

## 2.7.2 서비스 활용 (8개 조회 유형)

각 조회 유형별로 별도 endpoint와 servicedetail 경로를 사용한다.

### 2.7.2.1 지표 고유번호별/지표명별 설명자료조회

- **호출 URL**: `http://kosis.kr/openapi/pkNumberService.do?method=getList&service=1&serviceDetail=pkAll`

| 입력 항목 | 변수타입 | 설명 | 비고 |
|----------|---------|------|------|
| apiKey | String | 인증키 | 필수 |
| jipyoId | String | 지표 ID | 필수 |
| pageNo | String | 페이지 번호 | 선택 |
| numOfRows | String | 데이터 출력 개수 (numOfRows=20, pageNo=1: 1~20번 결과 리턴; pageNo=2: 21~40번; 미지정 시 자동으로 pageNo=1, numOfRows=10) | 선택 |
| format | String | json/xml | 필수 |
| content | String | html/json (헤더) | 선택 |

| 출력 항목 | 설명 | 형식 |
|----------|------|------|
| statJipyoId | 지표ID | NUMBER(20) |
| statJipyoNm | 지표명 | VARCHAR2(300) |
| jipyoExplan | 설명자료 제목 | VARCHAR2(100) |
| jipyoExplan1 | 개념 | CLOB |

### 2.7.2.2 지표명별 설명자료조회

- **호출 URL**: `http://kosis.kr/openapi/indExpService.do?method=getList&service=2&serviceDetail=indAll`

입력: apiKey + **jipyoNm** (지표명, 필수) + pageNo, numOfRows, format, content.
출력: statJipyoId, statJipyoNm, jipyoExplan, jipyoExplan1 (위와 동일).

### 2.7.2.3 목록별 지표조회

- **호출 URL**: `http://kosis.kr/openapi/indiListService.do?method=getList&service=3`

| 입력 항목 | 설명 | 비고 |
|----------|------|------|
| apiKey | 인증키 | 필수 |
| listId | 목록ID | 필수 |
| pageNo | 페이지 번호 | 선택 |
| numOfRows | 출력 개수 | 선택 |
| format | json/xml | 필수 |
| content | html/json | 선택 |

| 출력 항목 | 설명 | 형식 |
|----------|------|------|
| listId | 세부목록ID | VARCHAR2(40) |
| listNm | 세부부문명 | VARCHAR2(300) |
| statJipyoId | 지표ID | NUMBER(20) |
| statJipyoNm | 지표명 | VARCHAR2(300) |
| unit | 단위 | VARCHAR2(200) |
| areaTypeName | 지역구분명 | VARCHAR2(10) |
| prdSeName | 수록주기명 | VARCHAR2(10) |
| strtPrdDe | 수록시작시점 | VARCHAR2(8) |
| endPrdDe | 수록종료시점 | VARCHAR2(8) |
| rn | 수록시점개수 | NUMBER(5) |
| listSn1 | 목록순서1 | NUMBER(22) |
| listSn2 | 목록순서2 | NUMBER(22) |
| prdDe | 시점 | VARCHAR2(100) |
| repJipyoId | 대표지표ID | NUMBER(20) |
| repJipyoNm | 대표지표명 | VARCHAR2(300) |
| repJipyoUrl | 대표지표URL | VARCHAR2(3000) |
| explainUrl | 지표설명URL | VARCHAR2(3000) |

### 2.7.2.4 지표명별 목록조회

- **호출 URL**: `http://kosis.kr/openapi/indListSearchRequest.do?method=getList&service=4&serviceDetail=indList`

입력: apiKey + jipyoNm (지표명, 필수) + pageNo, numOfRows, format, content.

| 출력 항목 | 설명 | 형식 |
|----------|------|------|
| statJipyoId | 지표ID | NUMBER(20) |
| statJipyoNm | 지표명 | VARCHAR2(300) |
| unit | 단위 | VARCHAR2(200) |
| areaTypeName | 지역구분명 | VARCHAR2(10) |
| prdSeName | 수록주기명 | VARCHAR2(10) |
| strtPrdDe | 수록시작시점 | VARCHAR2(8) |
| endPrdDe | 수록종료시점 | VARCHAR2(8) |
| rn | 수록시점개수 | NUMBER(5) |
| prdDe | 종료시점+주기명 | VARCHAR2(100) |

### 2.7.2.5 고유번호별 목록조회

- **호출 URL**: `http://kosis.kr/openapi/indListSearchRequest.do?method=getList&service=4&serviceDetail=indList`

> 매뉴얼 표기상 2.7.2.4와 동일 URL이지만 입력 변수에 jipyoId(지표ID)를 사용. 매뉴얼 페이지 154가 동일 URL을 표기한 것은 오기 가능성. 실제 호출은 jipyoId로 차별화.

입력: apiKey + jipyoId (지표ID, 필수) + pageNo, numOfRows, format, content.
출력: 위 2.7.2.4와 동일.

### 2.7.2.6 고유번호별 지표 상세조회

- **호출 URL**: `http://kosis.kr/openapi/indIdDetailSearchRequest.do?method=getList&service=4&serviceDetail=indIdDetail`

| 입력 항목 | 설명 | 비고 |
|----------|------|------|
| apiKey | 인증키 | 필수 |
| jipyoId | 지표ID | 필수 |
| 시점기준 startPrdDe / endPrdDe | 조회 시작/종료 시점 | 선택 |
| 최신자료기준 rn / srvRn | 조회 기준 시점 / 조회 시점 개수 | 선택 |
| pageNo | 페이지 번호 | 선택 |
| numOfRows | 출력 개수 | 선택 |
| format | json/xml | 필수 |
| content | html/json | 선택 |

| 출력 항목 | 설명 | 형식 |
|----------|------|------|
| statJipyoId | 지표ID | NUMBER(20) |
| statJipyoNm | 지표명 | VARCHAR2(300) |
| prdSe | 수록주기 | VARCHAR2(20) |
| prdDe | 시점 | VARCHAR2(8) |
| itmNm | 항목 | VARCHAR2(300) |
| val | 통계수치 | NUMBER(25,10) |

### 2.7.2.7 수록주기별 목록조회

- **호출 URL**: `http://kosis.kr/openapi/prListSearchRequest.do?method=getList&service=4&serviceDetail=prList`

| 입력 항목 | 설명 | 비고 |
|----------|------|------|

| prdSe | 수록주기 | 필수 |
| pageNo | 페이지 번호 | 선택 |
| numOfRows | 출력 개수 | 선택 |
| format | json/xml | 필수 |
| content | html/json | 선택 |

출력: statJipyoId, statJipyoNm, unit, areaTypeName, prdSeName, strtPrdDe, endPrdDe, rn, prdDe (2.7.2.4 동일).

### 2.7.2.8 지표명별 상세조회

- **호출 URL**: `http://kosis.kr/openapi/indDetailSearchRequest.do?method=getList&service=4&serviceDetail=indDetail`

| 입력 항목 | 설명 | 비고 |
|----------|------|------|

| jipyoNm | 지표명 | 필수 |

| 최신자료기준 rn / srvRn | 조회 기준 시점 / 조회 시점 개수 | 선택 |
| pageNo | 페이지 번호 | 선택 |
| numOfRows | 출력 개수 | 선택 |
| format | json/xml | 필수 |
| content | html/json | 선택 |

출력: statJipyoId, statJipyoNm, prdSe, prdDe, itmNm, val (2.7.2.6 동일).

| 조회 유형 | endpoint | service / serviceDetail |

## 코드 사용 관점

`kosis_api.py`에는 통계주요지표 endpoint 구현이 없음. 향후 지표 기반 통계 조회 기능 추가 시 활용 가능.
